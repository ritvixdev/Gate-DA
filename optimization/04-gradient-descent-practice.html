Q1. The gradient descent update is…

Q2. GD steps in the direction of…

Q3. $f(x)=x^2$, $\eta=0.1$, $x_0=5$. Find $x_1$.
It must move; $f'(5)=10\neq0$.
That's ascent; descent decreases $x$ here.
✅ Correct — $x_1=5-0.1(2\cdot5)=4$.
One step lands at 4, not 0.
Q4. If the learning rate is far too large, GD…
Too big breaks convergence.
It moves a lot — too much.
✅ Correct — large steps overshoot the minimum and may diverge.
A bad rate can prevent convergence entirely.
Q5. On a non-convex function, GD may get stuck in a…
That's the goal, not a trap.
✅ Correct — it can settle into a local minimum; the start matters.
Constraints aren't relevant to unconstrained GD.
GD moves away from maxima.
Q6. GD stops moving when…
The function value isn't the stopping rule.
The rate doesn't decide stopping.
✅ Correct — at a minimum the gradient vanishes, so the update is zero.
The minimizer need not be at 0.
Q7. For a convex $f$ with a good learning rate, GD converges to…
Convex functions have no saddles.
✅ Correct — convexity ⇒ GD reaches the global min.
It descends, away from maxima.
It converges deterministically to the min.
Q8. The learning rate $\eta$ controls…
Direction is set by $-\nabla f$.
$f$ is fixed; $\eta$ doesn't change it.
✅ Correct — $\eta$ scales how far each step moves.
Unrelated to the dimension.
