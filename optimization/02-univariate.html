Univariate Optimization

One variable, one rule: the bottom of a smooth valley is flat. Set the slope to zero, then check whether it curves up (min) or down (max).

2 Why this topic exists

The simplest optimization is a function of one variable. Calculus gives an exact recipe: a minimum or maximum can only happen where the slope is zero. This is the foundation the multivariate and gradient-descent methods generalize.

3 Game / visual imagination
🎮 Imagine it

Walk along the curve. At the very bottom of a valley the ground is level — slope $f'(x)=0$. To tell a valley (min) from a hilltop (max), feel the curvature: curving up like a smile ($f''\gt0$) is a min; curving down like a frown ($f''\lt0$) is a max.

4 Basic definitions
  • Derivative $f'(x)$: the slope / instantaneous rate of change of $f$.
  • Stationary (critical) point: a point where $f'(x)=0$ — a flat slope.
  • Second derivative $f''(x)$: the curvature (how the slope itself changes).
  • Local minimum: $f'(x)=0$ and $f''(x)\gt0$ (curves up). Local maximum: $f'(x)=0$ and $f''(x)\lt0$.
  • Inflection / inconclusive: $f''(x)=0$ — the simple test fails; check higher derivatives or the sign of $f'$ around the point.
  • Limit $\lim_{x\to a}f(x)$: the value $f$ approaches near $a$ (exists iff the left and right limits agree). Continuous at $a$: $\lim_{x\to a}f(x)=f(a)$.
  • Differentiable $\Rightarrow$ continuous (not the reverse — $|x|$ at $0$ is continuous but has a corner).
📖 Study in Depth — functions, limits, continuity & optimizing f(x) of one variable, from zero
1. Function — an input→output rule

A function maps each valid input to exactly one output.

f(x) = x²   →   f(2)=4, f(−3)=9   (an input-output machine)

Optimization lives on top of functions: we hunt for the input that makes the output best.

2. Function as a map (the height idea)

Read a 1-variable function as a landscape:

x = your position · f(x) = the height there
f(x)=x² is a U-shaped bowl; lowest point at x=0

Minimum = valley, maximum = peak — the whole point of optimization.

3. Functions in ML — model & loss

A model is a function (price = f(area), e.g. ŷ = 5000x + 200000). The key one to optimize is the loss function:

Loss(w) = error of the model when its weight is w
goal: minimize Loss(w)  → find the best w
4. Domain & range 
domain = allowed inputs · range = possible outputs
f(x)=1/x : domain = all reals except 0 (1/0 undefined)
f(x)=x²  : range  = f(x) ≥ 0 (never negative)
5. Function types you meet in optimization
  • Linear f=mx+c — straight line
  • Quadratic f=ax²+bx+c — parabola (the bread-and-butter of optimization)
  • Exponential eˣ & log log x — appear in sigmoid & cross-entropy
  • Sigmoid σ(x)=1/(1+e⁻ˣ) — logistic regression
6. What a limit is

A limit asks what value f(x) approaches as x gets close to a point (x need not equal it):

lim(x→3) (x+2) = 5

"approaches a" ≠ "equals a" — this distinction is the whole idea.

7. Left- & right-hand limits 
x→a⁻ : approach from below (1.9, 1.99, …)
x→a⁺ : approach from above (2.1, 2.01, …)
limit EXISTS only if  LHL = RHL
f(x)=1 for x<0, 2 for x>0 : at 0, LHL=1 ≠ RHL=2 → limit does NOT exist (a jump)
8. Why limits matter — the derivative

The derivative — the engine of optimization — is defined as a limit:

f′(x) = lim(h→0) [ f(x+h) − f(x) ] / h     (slope as the step shrinks to 0)
Limits → Derivatives → Optimization → Gradient Descent
9. Standard limits & small-x approximations 
lim(x→a) c = c · lim(x→a) x = a · lim(x→0) sin x / x = 1 · lim(x→0)(1−cos x)/x = 0

Near x = 0 (handy for limits & later for Taylor/optimization):

sin x ≈ x · cos x ≈ 1 − x²/2 · eˣ ≈ 1 + x · log(1+x) ≈ x
10. Continuity — the smooth-road test

f is continuous at x=a (no jump, hole or break — drawable without lifting the pen) when all three hold:

1. f(a) exists
2. lim(x→a) f(x) exists
3. lim(x→a) f(x) = f(a)
f(x)=x²+1 at x=2: f(2)=5 and lim=5 → continuous ✓
11. Two kinds of discontinuity 
jump:      f=1 (x<0), 5 (x≥0) → LHL=1, RHL=5 → break at 0
removable (hole): f(x)=(x²−1)/(x−1) = x+1 for x≠1, but f(1) undefined
                  limit = 2 exists, yet f(1) doesn't → not continuous
12. Differentiable ⇒ continuous; & the extreme-value theorem 
Differentiable ⇒ Continuous   (but Continuous ⇏ Differentiable)

f(x)=|x| is continuous at 0 but has a sharp corner → not differentiable there.

Extreme Value Theorem: a continuous function on a closed interval [a,b] always attains a global max and min — which is why we check stationary points and endpoints.

GATE-style: f=x+1 (x<2), k (x=2), 3x−3 (x>2): LHL=3, RHL=3 → continuity needs k=3
13. Differentiation from first principles

Differentiation = finding the derivative (the instantaneous slope / rate of change). It's the limit of the average slope as the step shrinks:

f′(x) = lim(h→0) [ f(x+h) − f(x) ] / h

Derive d/dx(x²):

[(x+h)² − x²]/h = (x² + 2xh + h² − x²)/h = (2xh + h²)/h = 2x + h  →  (h→0)  2x
14. Derivative rules you must know 
constant:        d/dx(c) = 0
power:           d/dx(xⁿ) = n·xⁿ⁻¹
constant×:       d/dx(c·f) = c·f′
sum:             d/dx(f + g) = f′ + g′
exponential:     d/dx(eˣ) = eˣ
log (natural):   d/dx(ln x) = 1/x
e.g. d/dx(5x³ − 2x² + 7x − 9) = 15x² − 4x + 7
15. First derivative = direction (increasing / decreasing) 
f′(x) > 0 → increasing (uphill)
f′(x) < 0 → decreasing (downhill)
f′(x) = 0 → flat / stationary

Find where a function increases by solving f′(x) > 0:

f′(x) = 3x² − 12 > 0 ⇒ x² > 4 ⇒ x < −2 or x > 2  → increasing on (−∞,−2)∪(2,∞)
16. First-derivative sign test

Classify a critical point c by how f′ changes around it (no second derivative needed):

f′: − → +  across c  →  local MINIMUM (down then up = valley)
f′: + → −  across c  →  local MAXIMUM (up then down = hilltop)
no sign change       →  neither (e.g. an inflection)
f=x²−4x+5, c=2: f′(1)=−2 (<0), f′(3)=+2 (>0) → −→+ → minimum
17. Second derivative = curvature (bending)

The second derivative is the rate of change of the slope — it tells how the curve bends:

f″(x) > 0 → bends upward like a bowl ∪
f″(x) < 0 → bends downward like a hill ∩

That's why the second-derivative test works: a flat point of a bowl is a minimum, of a hill a maximum. (First derivative finds the candidate; second derivative classifies it.)

18. Key ML derivatives

Two that drive model training:

sigmoid:  σ(x) = 1/(1+e⁻ˣ)  →  σ′(x) = σ(x)·(1 − σ(x))     (logistic regression, NNs)
squared error: L = (y − ŷ)²  →  dL/dŷ = −2(y − ŷ)            (pushes ŷ toward y)

These feed straight into gradient descent's weight updates.

19. What univariate optimization is

Optimizing a function of one variable — find the x that minimizes (or maximizes) f(x).

min f(x),  x ∈ ℝ      (one knob to turn)

It's the simplest case and the foundation for the multivariate version.

20. Stationary points: f′(x) = 0

At a smooth peak or valley the tangent is flat, so the necessary condition is:

f′(x*) = 0     (x* is a stationary / critical point)

This finds candidates — they could be a min, a max, or an inflection. The next test decides which.

21. Second-derivative test 
f′(x*) = 0  and  f″(x*) > 0  → local MINIMUM (curves up, ∪)
f′(x*) = 0  and  f″(x*) < 0  → local MAXIMUM (curves down, ∩)
f″(x*) = 0                   → test fails (inconclusive)
22. Worked example 
f(x) = x² − 4x + 5
f′(x) = 2x − 4 = 0  → x* = 2
f″(x) = 2 > 0       → local (and here global) minimum
f(2) = 4 − 8 + 5 = 1

So the minimum value is 1 at x = 2.

23. When f″ = 0 — the test is inconclusive

Then check the sign of f′ around the point (or higher derivatives):

f′ goes − → + across x*  → minimum
f′ goes + → − across x*  → maximum
f′ keeps the same sign    → inflection (neither)
f(x)=x³ at x=0: f′=3x²=0, f″=0; f′ stays ≥0 → inflection, not an extremum
24. Global optima & closed intervals

On a closed interval [a, b], the global optimum is the best among:

interior stationary points (f′=0)  AND  the endpoints a, b

Endpoints can win even when the derivative isn't zero there — always check them.

25. Convex (unimodal) functions

If f″(x) ≥ 0 everywhere, f is convex — a single bowl. Then any stationary point is the global minimum (no other valleys to worry about).

x², eˣ, x² − 4x + 5 are convex → one global min
26. Concave functions (the mirror of convex)

Concave is convex flipped upside-down — a single dome ∩. The chord between any two points lies below the curve.

f″(x) ≤ 0 everywhere → concave ;  f″(x) < 0 → strictly concave
f(x) = −x² : f″ = −2 < 0 → concave (clean hill)

The key payoff mirrors convexity: for a concave function any local maximum is the global maximum. (A function that is both convex and concave is a straight line.)

27. Numerical method: Newton's method

When f′(x)=0 can't be solved by hand, iterate using both derivatives:

xₙ₊₁ = xₙ − f′(xₙ) / f″(xₙ)

Fast (quadratic) convergence near the optimum, but needs f″ and a good start.

28. Bracketing methods: bisection & golden section
  • Bisection on f′: f′ changes sign across a min — halve the bracket repeatedly.
  • Golden-section search: for a unimodal f, shrink an interval [a,b] using golden-ratio probes — no derivatives needed.

These need the function to be unimodal on the bracket.

29. Where it's used
  • line search inside larger optimizers (pick the best step size)
  • tuning a single hyperparameter
  • 1-D cost/profit/physics problems
30. Taylor series — a local polynomial approximation GATE extra

A Taylor (Maclaurin) series rebuilds a function near a point from its derivatives there:

f(x) ≈ f(0) + f′(0)x + f″(0)·x²/2! + f‴(0)·x³/3! + …

More terms = better fit. The famous expansions (the small-x forms from the foundation cards are just their first terms):

eˣ = 1 + x + x²/2! + …      sin x = x − x³/3! + …
cos x = 1 − x²/2! + …       ln(1+x) = x − x²/2 + x³/3 − …

Why it matters here: the 1st-order Taylor step f(x+Δx) ≈ f(x) + f′(x)Δx with Δx = −η f′(x) gives f(x+Δx) ≈ f(x) − η[f′(x)]² ≤ f(x) — that's why a small step along −slope reduces f (gradient descent), and adding the f″ term gives Newton's method.

31. Geometric & infinite series GATE extra

A series is the sum of a sequence. A geometric series has a constant ratio r:

a + ar + ar² + ar³ + … = a/(1 − r)   only if |r| < 1  (else it diverges)
1 + ½ + ¼ + … = 1/(1−½) = 2 · ½ + ¼ + … = (½)/(1−½) = 1 · 1+2+4+… diverges (r=2)

Used across DA: geometric-distribution probabilities, and reinforcement-learning discounted return G = R₀ + γR₁ + γ²R₂ + … which converges when 0 < γ < 1.

32. Common GATE traps
  • f′(x*)=0 guarantees a minimum? No — could be max or inflection; check f″.
  • f″=0 means inflection? Not necessarily — the test is just inconclusive.
  • Forget the endpoints on [a,b]? Mistake — the global optimum may be there.
  • Newton's method always converges? No — needs a good start & f″≠0.

6 Main formulas

Derivatives

From first principles:

$$ f'(x)=\lim_{h\to0}\frac{f(x+h)-f(x)}{h} $$

Standard rules — the ones you compute with:

$$ \tfrac{d}{dx}(x^n)=n x^{n-1},\quad \tfrac{d}{dx}(e^x)=e^x,\quad \tfrac{d}{dx}(\ln x)=\tfrac1x,\quad (f+g)'=f'+g' $$

Product, quotient & chain rules:

$$ (fg)'=f'g+fg',\qquad \Big(\tfrac{f}{g}\Big)'=\tfrac{f'g-fg'}{g^2},\qquad \tfrac{d}{dx}f(g(x))=f'(g)\,g'(x) $$

Optimization conditions

First-order (necessary):

$$ f'(x^*)=0 $$

Second-order test:

$$ f''(x^*)\gt0 \Rightarrow \text{local min}, \qquad f''(x^*)\lt0 \Rightarrow \text{local max}, \qquad f''(x^*)=0 \Rightarrow \text{inconclusive} $$

Sigmoid derivative (used in ML): $\sigma'(x)=\sigma(x)\big(1-\sigma(x)\big)$.

7 Step-by-step method
  1. Differentiate to get $f'(x)$.
  2. Solve $f'(x)=0$ for the stationary points.
  3. Compute $f''$ at each: positive → min, negative → max.
  4. If $f''=0$, use the first-derivative sign change (or higher derivatives).
  5. Compare values to find the global optimum on the range.
8 Worked example
✦ Minimize $f(x)=x^2-4x+7$

$f'(x)=2x-4=0 \Rightarrow x^*=2$.   $f''(x)=2\gt0$ → it's a minimum.

Minimum value $f(2)=4-8+7=3$.

✦ Two stationary points

$f(x)=x^3-3x$: $f'=3x^2-3=0\Rightarrow x=\pm1$. $f''=6x$: at $x=1$, $f''=6\gt0$ (min); at $x=-1$, $f''=-6\lt0$ (max).

9 GATE cheat sheet
⚡ High-value facts
  • Minima/maxima of a smooth $f$ occur only where $f'(x)=0$.
  • Second-derivative test: $f''\gt0$ → min, $f''\lt0$ → max.
  • For a quadratic $ax^2+bx+c$: vertex at $x=-b/(2a)$; min if $a\gt0$, max if $a\lt0$.
  • Convex (e.g. upward parabola) ⇒ the single stationary point is the global min.
  • Concave ($f''\le0$, e.g. downward parabola) ⇒ local max = global max; convex ($f''\ge0$) ⇒ local min = global min.
  • Don't forget endpoints when the domain is a closed interval.
10 Common traps
✕ Watch out

Trap: "$f'(x)=0$ means it's a minimum." Reality: it could be a max or an inflection — check $f''$.

Trap: "$f''=0$ means an inflection." Reality: inconclusive; it can still be a min/max (e.g. $x^4$ at 0).

Trap: "Ignore endpoints." Reality: on a closed interval the optimum can sit at an endpoint.

11 Connections with other topics
  • Types of optimization: this is the unconstrained, single-variable case.
  • Multivariate: $f'=0$ becomes $\nabla f=0$; $f''$ becomes the Hessian.
  • Gradient descent: the iterative way to reach $f'(x)=0$ without solving it by hand.
12 Practice problems

Q1. Minimize $f(x)=x^2-6x+10$. Find $x^*$ and the minimum value.

$f'=2x-6=0\Rightarrow x^*=3$; $f(3)=9-18+10=1$.

Q2. $f'(x_0)=0$ and $f''(x_0)=-4$. Min or max?

Maximum ($f''\lt0$).

Q3. Vertex of $3x^2-12x+1$?

$x=-b/(2a)=12/6=2$; it's a minimum since $a=3\gt0$.

Quick MCQ

At a stationary point, $f''(x)\gt0$ indicates a…
A max needs $f''\lt0$.
✅ Correct — curving upward ($f''\gt0$) is a local minimum.
Saddles appear in multivariate problems, not 1-D.
It's decisive: positive curvature → min.

  • Quadratic vertex: $x=-b/(2a)$.
