Multivariate Optimization

Many variables, same idea — but now "slope zero" becomes the gradient being zero, and "curvature" becomes a matrix: the Hessian.

2 Why this topic exists

Real models have many parameters. Optimizing $f(x_1,\dots,x_n)$ needs the multivariable version of the derivative test: set the gradient to zero to find candidates, then use the Hessian to tell minima from maxima and saddle points.

3 Game / visual imagination
🎮 Imagine it

The function is a terrain over the $(x,y)$ plane. The gradient $\nabla f$ is an arrow pointing in the steepest-uphill direction; at a peak, valley, or pass it's the zero arrow. A saddle is a mountain pass — uphill one way, downhill the other — neither a min nor a max.

4 Basic definitions
  • Gradient $\nabla f$: the vector of partial derivatives $\big(\partial f/\partial x_1,\dots\big)$ — steepest-ascent direction.
  • Critical point: where $\nabla f=0$ (every partial derivative is zero).
  • Hessian $H$: the matrix of second partial derivatives. Positive definite ⇒ minimum; negative definite ⇒ maximum; indefinite ⇒ saddle.
📖 Study in Depth — optimizing f(x₁,…,xₙ): gradient, Hessian, Lagrange & KKT, from zero
1. What multivariate optimization is

Optimizing a function of several variables — the real-world case (a model has many weights).

min f(x₁, x₂, …, xₙ)        x is a vector

The single derivative becomes a vector of partial derivatives — the gradient.

2. Partial derivatives — slope in one direction

A partial derivative differentiates with respect to one variable while holding the others constant:

∂f/∂x = how f changes if only x moves (y fixed)
∂f/∂y = how f changes if only y moves (x fixed)
f(x,y)=x²+y² :  ∂f/∂x = 2x (y² is constant → 0) ,  ∂f/∂y = 2y

On a 3-D hill at position (x,y), ∂f/∂x is the slope if you step only east-west.

3. Computing partial derivatives — mixed terms

Treat the "other" variable as a number, then differentiate normally:

f(x,y) = x²y + 3xy²
∂f/∂x : y is constant → 2xy + 3y²
∂f/∂y : x is constant → x² + 6xy

ML meaning: for a loss J(w₁,…,wₙ), ∂J/∂wᵢ says how the loss changes if only weight wᵢ moves — positive ⇒ decrease wᵢ, negative ⇒ increase it. That's each step of gradient descent.

4. The gradient ∇f 
∇f = ( ∂f/∂x₁ , ∂f/∂x₂ , … , ∂f/∂xₙ )

It points in the direction of steepest ascent, and its magnitude is the slope there. To go downhill, move along −∇f (that's gradient descent).

5. First-order condition: ∇f = 0

At a smooth min or max, every partial derivative is zero simultaneously:

∇f(x*) = 0   ⇔   ∂f/∂xᵢ = 0 for all i   (x* is a critical point)

Necessary, not sufficient — could be a min, max, or saddle.

6. The Hessian matrix

The matrix of all second partial derivatives — it describes the local curvature:

H = [ ∂²f/∂xᵢ∂xⱼ ]      (symmetric n×n)
2-D: H = [ f_xx  f_xy ; f_xy  f_yy ]
7. Second-order test via definiteness 
H positive definite  → local MINIMUM
H negative definite  → local MAXIMUM
H indefinite         → SADDLE point (min in some directions, max in others)
H semidefinite/singular → inconclusive
8. Read definiteness from eigenvalues 
all eigenvalues > 0 → positive definite → min
all eigenvalues < 0 → negative definite → max
mixed signs         → indefinite        → saddle

(2×2 shortcut: positive definite ⇔ f_xx > 0 and det H > 0.)

9. The 2-variable second-derivative (D) test

For a critical point of f(x,y), the practical GATE test uses the discriminant D (it's just det H):

D = f_xx · f_yy − (f_xy)²
D > 0 and f_xx > 0 → local MINIMUM
D > 0 and f_xx < 0 → local MAXIMUM
D < 0              → SADDLE point
D = 0              → inconclusive
f=x²+y² at (0,0): f_xx=2, f_yy=2, f_xy=0 → D=4>0, f_xx>0 → min
f=x²−y² at (0,0): D = 2(−2) − 0 = −4 < 0 → saddle

(This is the eigenvalue/definiteness test in 2-D: D>0,f_xx>0 ⇔ positive definite.)

10. Worked example — a clean minimum 
f(x,y) = x² + y²
∇f = (2x, 2y) = 0 → critical point (0,0)
H = [2 0 ; 0 2] → eigenvalues 2,2 > 0 → positive definite → MINIMUM
f(0,0) = 0
11. Worked example — a saddle point 
f(x,y) = x² − y²
∇f = (2x, −2y) = 0 → critical point (0,0)
H = [2 0 ; 0 −2] → eigenvalues 2, −2 (mixed) → indefinite → SADDLE

Down along y, up along x — a mountain pass, neither a peak nor a valley.

12. Convexity in n dimensions

If the Hessian is positive semidefinite everywhere, f is convex — then any critical point is the global minimum.

convex losses: least squares, logistic regression, ridge → reliable global optimum
non-convex: neural nets → many local minima & saddles
13. Constrained optimization — Lagrange multipliers

For an equality constraint, the gradient of f must align with the constraint's gradient at the optimum:

min f(x)  s.t. g(x) = 0
form  L = f(x) − λ g(x) ,  set ∇L = 0 and g(x)=0, solve for x and λ

λ (the multiplier) measures how much the optimum improves if the constraint is loosened.

14. Inequality constraints — KKT (preview)

With inequality constraints g(x) ≤ 0, the optimum satisfies the KKT conditions (the generalization of Lagrange):

stationarity, primal/dual feasibility, and complementary slackness (λᵢ·gᵢ=0)

This is the backbone of constrained convex optimization — e.g. the SVM dual.

15. Connection to ML
  • training minimizes a multivariate loss over the weight vector
  • gradient descent uses −∇f; Newton's method uses H⁻¹∇f
  • Lagrange/KKT underpin SVM and other constrained models
16. Traps, practice & memory box
  • ∇f=0 means minimum? No — could be max or saddle; check the Hessian.
  • Indefinite Hessian = max? No — it's a saddle.
  • Lagrange is for inequality constraints? No — equalities; inequalities use KKT.

6 Main formulas
$$ \textbf{Necessary: } \nabla f(x^*)=0 $$
$$ H \succ 0 \Rightarrow \text{min}, \qquad H \prec 0 \Rightarrow \text{max}, \qquad H \text{ indefinite} \Rightarrow \text{saddle} $$

Definiteness is read from the Hessian's eigenvalues: all positive → min, all negative → max, mixed signs → saddle.

2-variable shortcut (D-test): $D=f_{xx}f_{yy}-f_{xy}^2$;   $D>0,f_{xx}>0\Rightarrow$ min,   $D>0,f_{xx}<0\Rightarrow$ max,   $D<0\Rightarrow$ saddle.

7 Step-by-step method
  1. Compute the gradient $\nabla f$ (all first partials).
  2. Solve $\nabla f=0$ → critical points.
  3. Build the Hessian $H$ (second partials) at each point.
  4. Check definiteness via eigenvalues: all $+$ → min, all $-$ → max, mixed → saddle.
8 Worked example
✦ $f(x,y)=x^2+y^2$

$\nabla f=(2x,\ 2y)=0 \Rightarrow (x,y)=(0,0)$.

$H=\begin{bmatrix}2&0\\0&2\end{bmatrix}$, eigenvalues $2,2\gt0$ → positive definite → minimum at the origin.

✦ A saddle: $f(x,y)=x^2-y^2$

$\nabla f=(2x,-2y)=0\Rightarrow(0,0)$. $H=\begin{bmatrix}2&0\\0&-2\end{bmatrix}$, eigenvalues $2,-2$ (mixed) → saddle.

9 GATE cheat sheet
⚡ High-value facts
  • Critical points: solve $\nabla f=0$.
  • Hessian eigenvalues — all $+$ → min, all $-$ → max, mixed signs → saddle.
  • $2\times2$ Hessian: positive definite iff the top-left entry $\gt0$ and $\det(H)\gt0$.
  • Saddle points have $\det(H)\lt0$.
  • Convex $f$ ⇒ Hessian is positive semi-definite everywhere; any critical point is a global min.
10 Common traps
✕ Watch out

Trap: "$\nabla f=0$ means a minimum." Reality: could be a max or a saddle — check the Hessian.

Trap: "Positive Hessian entries ⇒ minimum." Reality: need positive definiteness (e.g. $\det(H)\gt0$ too), not just positive entries.

Trap: "Saddles can't have $\nabla f=0$." Reality: they do — that's exactly why the Hessian test is needed.

11 Connections with other topics
  • Univariate: $f'=0$ generalizes to $\nabla f=0$; $f''$ to the Hessian.
  • Linear algebra: definiteness = signs of the Hessian's eigenvalues (this subject leans on it).
  • Gradient descent: follows $-\nabla f$ downhill toward a minimum.
12 Practice problems

Q1. Critical point of $f(x,y)=(x-1)^2+(y+2)^2$?

$\nabla f=(2(x-1),2(y+2))=0\Rightarrow(1,-2)$; Hessian $=2I$ (PD) → minimum.

Q2. Hessian eigenvalues are $3$ and $-1$. What kind of point?

Mixed signs → saddle point.

Q3. $2\times2$ Hessian with top-left $=2$ and $\det(H)=-5$. Min, max, or saddle?

$\det(H)\lt0$ → saddle.
