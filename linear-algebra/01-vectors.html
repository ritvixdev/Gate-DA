Start here. Everything in linear algebra — matrices, rank, eigenvalues, PCA — is built on this one idea. We'll go from "what is a vector" all the way to basis and dimension, in plain words.

2 Why this topic exists

A vector is just an ordered list of numbers — but that simple idea lets us describe movement, data, and directions in space with one language.

Once you can talk about directions and combinations of directions, you can ask the big questions the rest of GATE linear algebra answers: How many independent directions are there? Which directions matter? Can I rebuild any point from a few building blocks? That's span, independence, basis and dimension — all in this topic.

4 Basic definitions

Vector: an ordered list of numbers, e.g. $v=(3,1)$. Each number is a component (the amount of movement along one axis).

$\mathbb{R}^n$: the space of all vectors with $n$ real components. $(1,2)\in\mathbb{R}^2$, $(3,4,5)\in\mathbb{R}^3$. The symbol $\in$ means "belongs to".

Zero vector ($0$): all components zero, e.g. $(0,0)$ — "no movement", the origin.

Vector space: a set of vectors that is closed — add any two, or scale any one, and the result is still inside the set. ($\mathbb{R}^2$ is the flat plane; $\mathbb{R}^3$ is 3-D space.)

The 8 axioms (just recognize them for GATE): commutativity $u+v=v+u$; associativity $(u+v)+w=u+(v+w)$; a zero vector $v+0=v$; an inverse $v+(-v)=0$; $1\cdot v=v$; $c(dv)=(cd)v$; $c(u+v)=cu+cv$; $(c+d)v=cv+dv$.

Subspace: a smaller vector space sitting inside a bigger one — for example a line through the origin inside $\mathbb{R}^2$.

Span: all vectors you can build from a given set by scaling and adding. Linear combination: one such built vector, $c_1v_1+c_2v_2+\dots$.

Linearly independent: no vector in the set can be built from the others (no waste). Dependent: at least one is redundant.

Basis: a set that is independent and spans the space — the minimum team that reaches everywhere. Dimension: how many vectors are in a basis.

📖 Study in Depth — learn vectors from zero, one idea at a time

Linear algebra is the maths of movement, direction, space and transformation. In a game world, everything has a position — the player, the enemy, the camera, a push of force. Vectors are the language for all of it. Open any card below and go at your own pace.

1. What is a vector, really?

A vector is simply a list of numbers.

Example:

v = [1, 2]

This means:

move 1 step in the x direction
move 2 steps in the y direction

So on a 2D game map, [1, 2] means:

from the origin (0, 0)
go right 1
go up 2

Visually:

y
↑
2 |        ● [1, 2]
1 |
0 |__ __ __→ x
   0  1  2

So [1, 2] is not just two numbers — it is an arrow from (0, 0) to (1, 2).

2. Components, subscripts & transpose

Each number inside a vector is called a component.

v = [v₁, v₂, v₃]

v₁ = first component
v₂ = second component
v₃ = third component

The small numbers ₁, ₂, ₃ are subscripts — just labels ("v one", "v two"), not powers.

A vector can be written as a row or a column:

row vector:     v = [1, 2]

column vector:  v = [1
                     2]

The little ᵀ means transpose — it flips a row into a column (or back):

[v₁, v₂, v₃]ᵀ  =  [v₁
                   v₂
                   v₃]

For GATE, remember one thing: ᵀ = transpose = swap row and column.

3. What is ℝⁿ, and "belongs to" (∈)?

This symbol turns up everywhere:

ℝⁿ

Read it as:

R n

It means:

the space of all vectors with n real-number components

ℝ means real numbers — normal numbers like:

1,  2,  -3,  0,  4.5,  1/2,  √2

So:

ℝ² = all 2D vectors
ℝ³ = all 3D vectors
ℝⁿ = all n-dimensional vectors

Examples:

[1, 2]      belongs to ℝ²
[3, 4, 5]   belongs to ℝ³
[1, 0, 2, 7] belongs to ℝ⁴

The symbol ∈ means belongs to. So:

[1, 2] ∈ ℝ²

reads "[1, 2] belongs to 2D real vector space".

4. Adding vectors & scaling them

Addition is component by component.

u = [1, 2]
v = [3, 1]

u + v = [1+3, 2+1] = [4, 3]

Picture it tip-to-tail: follow arrow u, then from its tip follow v. The final arrow is u + v.

A scalar is just a number. Multiplying stretches or flips the arrow:

2 · [1, 2]    = [2, 4]      bigger, same direction
0.5 · [2, 4]  = [1, 2]      smaller, same direction
-1 · [1, 2]   = [-1, -2]    flipped, opposite way
0 · [1, 2]    = [0, 0]      collapses to zero

🎮 Imagine: the vector is your movement direction and the scalar is a speed multiplier — 5 · [1, 0] is "move right, five times as fast".

5. Vector space & closure

A vector space is a collection of vectors where two moves are always safe: add any two, or scale any one — and the answer stays inside the same set.

That "stays inside" property is called closure.

Example — ℝ² is a vector space:

[1, 2] + [3, 4] = [4, 6]    still in ℝ²
5 · [1, 2]      = [5, 10]   still in ℝ²

🎮 Imagine: a game map is "closed" if your character can move and still stay on the map. Add → still inside. Scale → still inside. That is closure.

6. Subspace + the 3-point test

A subspace is a smaller vector space living inside a bigger one — for example a line through the origin inside ℝ².

To check if a set W is a subspace, test only three things:

  1. Does W contain the zero vector? If no — stop, not a subspace.
  2. Is it closed under addition?
  3. Is it closed under scaling?

Quick test:

W = { (x, y) : x + y = 1 }

check zero:  (0, 0) → 0 + 0 = 0 ≠ 1
so (0, 0) is NOT in W
→ NOT a subspace   (stop here)
7. Span & linear combinations

A linear combination means: scale some vectors, then add them.

c₁·v₁ + c₂·v₂ + ... + cₖ·vₖ

The span is every vector you can build this way.

Example:

v₁ = [1, 0]   (right)
v₂ = [0, 1]   (up)

2·v₁ + 3·v₂ = [2, 3]
→ you can reach ANY point, so span{v₁, v₂} = ℝ²

These two arrows cover the whole plane.

8. Independent vs dependent

Vectors are independent when no vector is redundant — none can be built from the others.

The test (only the all-zero solution works):

c₁·v₁ + c₂·v₂ + ... + cₖ·vₖ = 0
only when  c₁ = c₂ = ... = cₖ = 0

When one vector is redundant, they are dependent:

v₁ = [1, 0]   v₂ = [0, 1]   v₃ = [2, 3]

v₃ = 2·v₁ + 3·v₂   ← adds no new direction
→ dependent

Memory: independent = no waste; dependent = at least one vector is redundant.

9. GATE shortcut: more than n ⇒ dependent

In ℝⁿ you can have at most n independent vectors. So any set with more than n vectors is automatically dependent — no calculation needed.

3 vectors in ℝ²  →  always dependent
4 vectors in ℝ³  →  always dependent
5 vectors in ℝ⁴  →  always dependent

Count first — this is one of the fastest marks in a GATE question.

10. Basis & dimension

A basis is the "perfect team": independent and it spans the space — enough vectors to reach everywhere, with none wasted.

Standard basis of ℝ²:

e₁ = [1, 0]
e₂ = [0, 1]

any [x, y] = x·e₁ + y·e₂

The dimension is the number of vectors in a basis:

dim(ℝ²) = 2
dim(ℝ³) = 3
dim(ℝⁿ) = n

Every basis of the same space has the same size — that count is the dimension.

6 Main formulas
$$ \operatorname{span}\{v_1,\dots,v_k\} = \{\, c_1 v_1 + c_2 v_2 + \dots + c_k v_k \;:\; c_i \in \mathbb{R} \,\} $$

Independence test — the vectors are independent only if the only way to get zero is all-zero scalars:

$$ c_1 v_1 + c_2 v_2 + \dots + c_k v_k = 0 \;\Longrightarrow\; c_1=c_2=\dots=c_k=0 $$

Subspace test — a set $W$ is a subspace exactly when all three hold:

$$ 0 \in W \qquad u,v \in W \Rightarrow u+v \in W \qquad u\in W,\ c\in\mathbb{R} \Rightarrow cu \in W $$
7 Step-by-step method

To check if a set $W$ is a subspace:

  1. Does $W$ contain the zero vector? If no, stop — it is not a subspace.
  2. Take two general members $u,v\in W$; check $u+v$ still satisfies $W$'s condition.
  3. Take $u\in W$ and any scalar $c$; check $cu$ still satisfies the condition.
  4. All three pass → it is a subspace.

To check independence of vectors in $\mathbb{R}^n$:

  1. If you have more than $n$ vectors, they are automatically dependent — done.
  2. Otherwise set $c_1v_1+\dots+c_kv_k=0$ and solve for the $c_i$.
  3. Only the all-zero solution → independent. Any non-zero solution → dependent.
8 Worked example
✦ Is $W=\{(x,y): x+y=0\}$ a subspace of $\mathbb{R}^2$?
  1. Zero: $(0,0)$ gives $0+0=0$ ✓ contained.
  2. Add: if $x_1+y_1=0$ and $x_2+y_2=0$, then $(x_1+x_2)+(y_1+y_2)=0$ ✓.
  3. Scale: $c(x+y)=cx+cy=0$ ✓.

All three hold → yes, it's a subspace (the line $y=-x$ through the origin).

✦ Are $(1,0),(0,1),(2,3)$ independent?

They live in $\mathbb{R}^2$ and there are 3 of them — more than $n=2$ — so they are dependent. Indeed $(2,3)=2(1,0)+3(0,1)$.

9 GATE shortcuts
⚡ High-value facts
  • In $\mathbb{R}^n$, more than $n$ vectors are always dependent. (3 in $\mathbb{R}^2$, 4 in $\mathbb{R}^3$ → dependent.)
  • A subspace must contain $0$. If $0\notin W$, stop — not a subspace.
  • $\dim(\mathbb{R}^n)=n$; every basis of a space has the same number of vectors.
  • Two vectors are dependent iff one is a scalar multiple of the other (they're parallel).
  • The span of $k$ vectors has dimension = number of independent ones among them.
10 Common traps
✕ Watch out

Trap: "A line is always a subspace." Reality: only if it passes through the origin. $x+y=1$ is not a subspace.

Trap: "The zero vector can be an eigenvector / a basis vector." Reality: the zero vector is never independent and never in a basis.

Trap: "More vectors means a bigger span / dimension." Reality: extra dependent vectors add nothing; dimension counts independent directions only.

11 Connections with other topics
  • Matrices: a matrix is a machine that moves vectors around.
  • Rank: the rank of a matrix = number of independent directions in its columns (this topic's idea, applied).
  • Linear equations: "does $Ax=b$ have a solution?" is really "is $b$ in the span of $A$'s columns?"
  • Eigenvectors: special directions that don't rotate under a matrix.
  • Basis & dimension reappear everywhere — null space, column space, eigenspaces.
12 Practice problems

Q1. Is $(2,3,4)$ in $\mathbb{R}^2$?

No — it has 3 components, so it lives in $\mathbb{R}^3$.

Q2. Is $W=\{(x,y): x+y=5\}$ a subspace of $\mathbb{R}^2$?

No — $(0,0)$ gives $0\neq5$, so $W$ doesn't contain the zero vector. Stop there.

Q3. Are $(1,0),(0,1),(4,5)$ independent?

Dependent — 3 vectors in $\mathbb{R}^2$ ($>n$) are always dependent.

Q4. What is $\dim(\operatorname{span}\{(1,2,3),(2,4,6)\})$?

$1$ — the second vector is $2\times$ the first, so they're parallel; the span is a line.

Q5. Write $(5,7)$ as a combination of $(1,0)$ and $(0,1)$.

$(5,7)=5(1,0)+7(0,1)$.

Quick MCQ

Which of the following is not a subspace of $\mathbb{R}^2$?
A line through the origin — it is a subspace.
The trivial subspace — it is a subspace.
$\mathbb{R}^2$ is a subspace of itself.
✅ Correct — $x+y=1$ excludes $(0,0)$, so not a subspace.
