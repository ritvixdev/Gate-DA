Vectors &
Vector Spaces

The language of linear algebra. Vectors, subspaces, span, and linear independence are the foundation everything else is built on.

🔵 What is a vector?

A vector is an ordered list of numbers. In $\mathbb{R}^n$ a vector $v$ has $n$ components and is usually written as a column.

$$ v = \begin{bmatrix} v_1 \\ v_2 \\ \vdots \\ v_n \end{bmatrix}, \qquad u + v = \begin{bmatrix} u_1+v_1 \\ \vdots \\ u_n+v_n \end{bmatrix}, \qquad c\,v = \begin{bmatrix} c\,v_1 \\ \vdots \\ c\,v_n \end{bmatrix} $$

Think of a vector as an arrow from the origin: adding is tip-to-tail, scaling stretches or flips it. Drag the arrows below to feel it.

🌐 Vector space ($\mathbb{R}^n$)

A vector space is a set $V$ where you can add vectors and scale them, and the result always stays inside $V$ (it is closed under both operations). Formally, eight axioms hold:

  • $u+v = v+u$  (commutativity)
  • $(u+v)+w = u+(v+w)$  (associativity)
  • $v + 0 = v$  (zero vector exists)
  • $v + (-v) = 0$  (additive inverse)
  • $1\cdot v = v$,   $c(dv) = (cd)v$
  • $c(u+v) = cu+cv$,   $(c+d)v = cv+dv$  (distributivity)
🧲 Subspace

A subset $W \subseteq \mathbb{R}^n$ is a subspace if it satisfies three things:

$$ \text{(1) } 0 \in W \qquad \text{(2) } u,v \in W \Rightarrow u+v \in W \qquad \text{(3) } u \in W,\ c \in \mathbb{R} \Rightarrow cu \in W $$
✦ Worked example

Is $W = \{(x,y) : x+y = 0\}$ a subspace of $\mathbb{R}^2$?

  1. Does $(0,0)$ satisfy $x+y=0$? Yes ✓
  2. If $x_1+y_1=0$ and $x_2+y_2=0$, then $(x_1+x_2)+(y_1+y_2)=0$ ✓
  3. $c(x+y)=cx+cy=0$ ✓

→ Yes — $W$ is a subspace (a line through the origin).

⚡ GATE check

Every subspace must contain the zero vector. If $0 \notin W$, stop — it is not a subspace.

🌿 Span & linear independence

Span is the set of all linear combinations of some vectors:

$$ \operatorname{span}\{v_1,\dots,v_k\} = \{ c_1 v_1 + c_2 v_2 + \dots + c_k v_k : c_i \in \mathbb{R} \} $$

Vectors are linearly independent when the only way to combine them into the zero vector is the trivial way:

$$ c_1 v_1 + \dots + c_k v_k = 0 \;\Longrightarrow\; c_1 = c_2 = \dots = c_k = 0 $$
✦ Are these independent?

$v_1=(1,0),\ v_2=(0,1),\ v_3=(2,3)$.

Since $v_3 = 2v_1 + 3v_2$, they are linearly dependent. In $\mathbb{R}^2$ any 3 vectors are always dependent — at most 2 can be independent.

🧠 Memory trick

Independent = no vector can be built from the others. Like a team where no member is redundant.

🏛️ Basis & dimension

A basis of $V$ is a set of vectors that is both (1) linearly independent and (2) spans $V$.

$$ \text{Standard basis of } \mathbb{R}^3:\quad e_1=(1,0,0),\ e_2=(0,1,0),\ e_3=(0,0,1) \qquad \dim(\mathbb{R}^n)=n $$
⚡ GATE fact

Every basis of a space has the same number of vectors — that number is the dimension, and it cannot change with your choice of basis.

Practice

Q1. Which of the following is not a subspace of $\mathbb{R}^2$?
A line through the origin — it is a subspace.
The trivial subspace — it is a subspace.
$\mathbb{R}^2$ is a subspace of itself.
✅ Correct — $x+y=1$ does not contain $(0,0)$, so it is not a subspace.
Q2. The dimension of $\operatorname{span}\{(1,2,3),(2,4,6)\}$ is:
✅ Correct — $(2,4,6)=2(1,2,3)$, so the vectors are parallel and the span is a 1-D line.
The two vectors are dependent (one is a multiple of the other), so dim is 1, not 2.
Span dimension can't exceed the number of independent vectors.
Dimension counts independent directions, not entries.

GATE takeaway — always check the zero vector first; if 0 ∉ W, the set is not a subspace.
