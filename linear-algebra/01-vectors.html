Vectors &
Vector Spaces

Start here. Everything in linear algebra — matrices, rank, eigenvalues, PCA — is built on this one idea. We'll go from "what is a vector" all the way to basis and dimension, in plain words.

2 Why this topic exists

A vector is just an ordered list of numbers — but that simple idea lets us describe movement, data, and directions in space with one language.

Once you can talk about directions and combinations of directions, you can ask the big questions the rest of GATE linear algebra answers: How many independent directions are there? Which directions matter? Can I rebuild any point from a few building blocks? That's span, independence, basis and dimension — all in this topic.

4 Basic definitions
  • Vector: an ordered list of numbers, e.g. $v=(3,1)$. Each number is a component (the amount of movement along one axis).
  • $\mathbb{R}^n$: the space of all vectors with $n$ real components — $(1,2)\in\mathbb{R}^2$, $(3,4,5)\in\mathbb{R}^3$. The symbol $\in$ means "belongs to".
  • Zero vector ($0$): all components zero, e.g. $(0,0)$ — "no movement", the origin.
  • Vector space: a set of vectors that is closed — add any two, or scale any one, and the result is still inside the set. ($\mathbb{R}^2$ is the flat plane; $\mathbb{R}^3$ is 3-D space.)
  • Subspace: a smaller vector space sitting inside a bigger one — for example a line through the origin inside $\mathbb{R}^2$.
  • Span: all vectors you can build from a given set by scaling and adding.
  • Linear combination: one such built vector, $c_1v_1+c_2v_2+\dots$.
  • Linearly independent: no vector in the set can be built from the others (no waste). Dependent: at least one is redundant.
  • Basis: a set that is independent and spans the space — the minimum team that reaches everywhere.
  • Dimension: how many vectors are in a basis.

The 8 axioms (just recognize them for GATE): commutativity $u+v=v+u$; associativity $(u+v)+w=u+(v+w)$; a zero vector $v+0=v$; an inverse $v+(-v)=0$; $1\cdot v=v$; $c(dv)=(cd)v$; $c(u+v)=cu+cv$; $(c+d)v=cv+dv$.

📖 Study in Depth — vectors from zero, one idea per card
1. What is a vector?

A vector is a list of numbers.

Example:

v = [1, 2]

This means:

move 1 step in the x direction
move 2 steps in the y direction

So in a 2D game map, [1, 2] means:

from origin (0, 0)
go right 1
go up 2

Visually:

y
↑
2 |        ● [1, 2]
1 |
0 |__ __ __→ x
   0  1  2

So [1, 2] is not just two numbers — it is an arrow from (0, 0) to (1, 2).

2. What do x and y mean?

In 2D, [x, y] means:

x = left / right movement
y = up / down movement

Usually:

positive x = right
negative x = left
positive y = up
negative y = down

Examples:

[3, 0]   = move 3 right
[-2, 0]  = move 2 left
[0, 4]   = move 4 up
[0, -5]  = move 5 down
[2, 3]   = move 2 right and 3 up

🎮 Game imagination: the player starts at the origin [0, 0]. The input vector [2, 3] means: right 2 tiles, up 3 tiles.

3. What is a component?

In v = [1, 2], the numbers 1 and 2 are called components.

1 = first component  = x component
2 = second component = y component

For a 3D vector:

v = [v₁, v₂, v₃]

v₁ = first component
v₂ = second component
v₃ = third component

The small numbers are called subscripts: v₁ means "v one", v₂ means "v two". They are not powers — they are labels.

4. Row vector vs column vector

You may see a row vector:

v = [1, 2]

Or a column vector:

v = [1
     2]

In books, vectors are often written as columns:

v = [v₁
     v₂
     ...
     vₙ]

When they write v = [v₁, v₂, ..., vₙ]ᵀ, the T means transpose — turn a row into a column:

[v₁, v₂, v₃]ᵀ  =  [v₁
                   v₂
                   v₃]

For GATE, just remember: ᵀ = transpose = flip row / column.

5. What is ℝⁿ?

This symbol is very important:

ℝⁿ

Read it as:

R n

It means:

the space of all vectors with n real-number components

ℝ means real numbers — normal numbers like:

1,  2,  -3,  0,  4.5,  1/2,  √2

So:

ℝ² = all 2D vectors
ℝ³ = all 3D vectors
ℝⁿ = all n-dimensional vectors

Examples:

[1, 2]       belongs to ℝ²
[3, 4, 5]    belongs to ℝ³
[1, 0, 2, 7] belongs to ℝ⁴

The symbol ∈ means belongs to. So [1, 2] ∈ ℝ² means "[1, 2] belongs to 2D real vector space".

6. Vector addition

Suppose:

u = [1, 2]
v = [3, 1]

Then add component by component:

u + v = [1+3, 2+1] = [4, 3]

🎮 Game imagination:

u = right 1, up 2
v = right 3, up 1
total = right 4, up 3

Visually: first follow arrow u, then from its tip follow arrow v. The final arrow is u + v. This is called tip-to-tail addition.

7. Scalar multiplication

A scalar is just a normal number.

Example:

c = 3
v = [2, 1]

c · v = 3 · [2, 1] = [6, 3]

It means stretch the vector 3 times.

More examples:

2 · [1, 2]    = [2, 4]      bigger, same direction
0.5 · [2, 4]  = [1, 2]      smaller, same direction
-1 · [1, 2]   = [-1, -2]    flipped, opposite direction
0 · [1, 2]    = [0, 0]      becomes the zero vector

🎮 Imagine: vector = movement direction, scalar = speed multiplier. 5 · [1, 0] = [5, 0] means move faster to the right.

8. Zero vector

The zero vector means no movement.

2D:  0 = [0, 0]
3D:  0 = [0, 0, 0]

It is the origin point.

For GATE, remember: every vector space and every subspace must contain the zero vector. If the zero vector is missing, it is not a subspace.

9. Vector space

A vector space is a collection of vectors where two operations are safe:

1. add two vectors
2. multiply a vector by a scalar

...and after doing them, the answer still stays inside the same space. This property is called closure.

Example — ℝ² is a vector space:

u = [1, 2]   v = [3, 4]

u + v = [4, 6]   still in ℝ²
5 · u = [5, 10]  still in ℝ²

So ℝ² is closed under addition and scaling.

10. What is closure?

Closure means: you do something inside the world, and the result stays inside the same world.

🎮 Game imagination: imagine a game map. If your character moves and still stays inside the map, the map is "closed" for that movement.

For vector spaces:

add vectors   → still inside
scale vector  → still inside

That is closure.

11. The 8 vector space axioms GATE extra

Do not fear the word axiom — an axiom is just a rule that must be true. For GATE you usually only need to recognize them, not prove them.

Let:

u, v, w = vectors
c, d    = scalars
0       = zero vector

Rule 1 — Commutativity (order does not matter):

u + v = v + u

[1,2] + [3,4] = [4,6]
[3,4] + [1,2] = [4,6]   same answer

Rule 2 — Associativity (grouping does not matter):

(u + v) + w = u + (v + w)

Rule 3 — Zero vector (adding zero does nothing):

v + 0 = v

Rule 4 — Additive inverse (every vector has an opposite):

v + (-v) = 0

v  = [2, 3]
-v = [-2, -3]
[2,3] + [-2,-3] = [0, 0]

Rule 5 — Scalar identity (multiplying by 1 changes nothing):

1 · v = v

Rule 6 — Scalar associativity:

c(d·v) = (c·d)·v        e.g.  2(3v) = 6v

Rule 7 — Scalar distributes over vector addition:

c(u + v) = c·u + c·v

2([1,2] + [3,4]) = 2[4,6]      = [8,12]
2[1,2] + 2[3,4]  = [2,4]+[6,8] = [8,12]

Rule 8 — Scalar addition distributes:

(c + d)v = c·v + d·v    e.g.  (2+3)v = 2v + 3v
12. Subspace

A subspace is a smaller vector space inside a bigger vector space. Example: a line through the origin inside ℝ².

Think of ℝ² as the whole flat game map. One subspace is a line through the origin:

W = { (x, y) : x + y = 0 }

This means all points where x + y = 0. Some examples:

(1, -1)   because  1 + (-1) = 0
(2, -2)   because  2 + (-2) = 0
(0, 0)    because  0 + 0    = 0

This is the line y = -x, and it passes through the origin — so it is a subspace.

13. Subspace test

To check if something is a subspace, test only these 3 things:

  1. Does it contain the zero vector?
  2. Is it closed under addition?
  3. Is it closed under scalar multiplication?

Most important shortcut: if the zero vector is not present, it is NOT a subspace — stop immediately.

Example:

W = { (x, y) : x + y = 1 }

check zero:  (0, 0) → 0 + 0 = 0
but the condition says x + y = 1
so (0, 0) is NOT inside W

→ W is NOT a subspace   (no need to check anything else)
14. Span

Span means: all possible vectors you can create using the given vectors.

Formula:

span{v₁, v₂, ..., vₖ} = { c₁v₁ + c₂v₂ + ... + cₖvₖ }

Do not fear the formula. It just says: scale each vector by any number, add them, and see all the places you can reach.

🎮 Game imagination:

v₁ = [1, 0]   move right
v₂ = [0, 1]   move up

2v₁ + 3v₂ = [2, 0] + [0, 3] = [2, 3]

You can reach any point in 2D, so span{[1,0], [0,1]} = ℝ² — these two vectors span the whole plane.

15. Linear combination

This expression is called a linear combination:

c₁v₁ + c₂v₂ + ... + cₖvₖ

Meaning: scale the vectors, then add them.

Example:

v₁ = [1, 0]   v₂ = [0, 1]

3v₁ + 4v₂ = [3, 0] + [0, 4] = [3, 4]

So [3, 4] is a linear combination of [1, 0] and [0, 1].

16. Linear independence

This is extremely important for GATE. Vectors are linearly independent if no vector is useless or a duplicate.

Formal rule:

c₁v₁ + c₂v₂ + ... + cₖvₖ = 0

has only one solution:
c₁ = c₂ = ... = cₖ = 0

This is called the trivial solution — the only way to make zero is to use nothing.

🎮 Game imagination:

v₁ = [1, 0]   right
v₂ = [0, 1]   up

Can one be made from the other?  No.
Right cannot create up. Up cannot create right.
→ they are independent.
17. Linear dependence

Vectors are dependent if one vector can be built from the others.

Example:

v₁ = [1, 0]   v₂ = [0, 1]   v₃ = [2, 3]

v₃ = 2v₁ + 3v₂ = [2, 0] + [0, 3] = [2, 3]

So v₃ gives no new direction — it is already built from v₁ and v₂. Therefore v₁, v₂, v₃ are linearly dependent.

Memory trick:

independent = no vector is redundant
dependent   = at least one vector is redundant
18. Basis

A basis is the perfect team of vectors. A set is a basis if:

1. it spans the space
2. it is linearly independent

So a basis has enough vectors to cover the whole space, but no extra useless vector.

For ℝ², the standard basis is:

e₁ = [1, 0]    gives the x direction
e₂ = [0, 1]    gives the y direction

Any vector can be built from them:

[x, y] = x[1,0] + y[0,1]

[5, 7] = 5[1,0] + 7[0,1]

They span ℝ² and are independent, so {[1,0], [0,1]} is a basis of ℝ².

19. Dimension

Dimension means the number of vectors in a basis.

ℝ² → basis has 2 vectors → dim(ℝ²) = 2
ℝ³ → basis has 3 vectors → dim(ℝ³) = 3

General:

dim(ℝⁿ) = n

6 Main formulas

Vector operations

Addition & scaling — add componentwise, scale every component:

$$ u+v=(u_1+v_1,\dots,u_n+v_n),\qquad c\,v=(c\,v_1,\dots,c\,v_n) $$

Linear combination — scale each vector then add:

$$ c_1 v_1 + c_2 v_2 + \dots + c_k v_k $$

Length (norm) — Pythagoras in $n$ dimensions:

$$ \|v\|=\sqrt{v_1^2+\dots+v_n^2} $$

Dot product — multiply matching components and add; zero means perpendicular:

$$ u\cdot v=\sum_i u_i v_i $$

Span, independence & subspace

Span — every vector you can build from the set:

$$ \operatorname{span}\{v_1,\dots,v_k\}=\{\, c_1 v_1+\dots+c_k v_k \;:\; c_i\in\mathbb{R} \,\} $$

Independence test — independent only if the only way to get zero is all-zero scalars:

$$ c_1 v_1+\dots+c_k v_k=0 \;\Longrightarrow\; c_1=\dots=c_k=0 $$

Subspace test — a set $W$ is a subspace exactly when all three hold:

$$ 0\in W \qquad u,v\in W\Rightarrow u+v\in W \qquad u\in W,\ c\in\mathbb{R}\Rightarrow cu\in W $$

Dimension — the number of vectors in any basis; $\dim(\mathbb{R}^n)=n$.

7 Step-by-step method

To check if a set $W$ is a subspace:

  1. Does $W$ contain the zero vector? If no, stop — it is not a subspace.
  2. Take two general members $u,v\in W$; check $u+v$ still satisfies $W$'s condition.
  3. Take $u\in W$ and any scalar $c$; check $cu$ still satisfies the condition.
  4. All three pass → it is a subspace.

To check independence of vectors in $\mathbb{R}^n$:

  1. If you have more than $n$ vectors, they are automatically dependent — done.
  2. Otherwise set $c_1v_1+\dots+c_kv_k=0$ and solve for the $c_i$.
  3. Only the all-zero solution → independent. Any non-zero solution → dependent.
8 Worked example
✦ Is $W=\{(x,y): x+y=0\}$ a subspace of $\mathbb{R}^2$?
  1. Zero: $(0,0)$ gives $0+0=0$ ✓ contained.
  2. Add: if $x_1+y_1=0$ and $x_2+y_2=0$, then $(x_1+x_2)+(y_1+y_2)=0$ ✓.
  3. Scale: $c(x+y)=cx+cy=0$ ✓.

All three hold → yes, it's a subspace (the line $y=-x$ through the origin).

✦ Are $(1,0),(0,1),(2,3)$ independent?

They live in $\mathbb{R}^2$ and there are 3 of them — more than $n=2$ — so they are dependent. Indeed $(2,3)=2(1,0)+3(0,1)$.

9 GATE cheat sheet
⚡ High-value facts
  • In $\mathbb{R}^n$, more than $n$ vectors are always dependent. (3 in $\mathbb{R}^2$, 4 in $\mathbb{R}^3$ → dependent.)
  • A subspace must contain $0$. If $0\notin W$, stop — not a subspace.
  • $\dim(\mathbb{R}^n)=n$; every basis of a space has the same number of vectors.
  • Two vectors are dependent iff one is a scalar multiple of the other (they're parallel).
  • The span of $k$ vectors has dimension = number of independent ones among them.
🔢 The "more than n" rule, in detail
  • In $\mathbb{R}^2$ the maximum number of independent vectors is $2$ — so any 3 vectors in $\mathbb{R}^2$ are always dependent.
  • In $\mathbb{R}^3$ the maximum is $3$ — so any 4 vectors in $\mathbb{R}^3$ are always dependent.
  • General rule: in $\mathbb{R}^n$, more than $n$ vectors are always dependent.

Quick examples: 3 vectors in $\mathbb{R}^2$ → dependent; 4 in $\mathbb{R}^3$ → dependent; 5 in $\mathbb{R}^4$ → dependent. Count first — it is a fast mark.

📋 Quick recall — everything for GATE
  • Vector = ordered list of numbers.
  • $\mathbb{R}^n$ = all vectors with $n$ real components.
  • Subspace = a smaller vector space inside a bigger one; it must contain the zero vector.
  • Span = all vectors you can create; a linear combination is scaled vectors added together.
  • Independent = no vector can be built from the others; dependent = at least one is redundant.
  • Basis = independent + spans; dimension = number of vectors in a basis, with $\dim(\mathbb{R}^n)=n$.
  • In $\mathbb{R}^n$, more than $n$ vectors are always dependent.
10 Common traps
✕ Watch out

Trap: "A line is always a subspace." Reality: only if it passes through the origin. $x+y=1$ is not a subspace.

Trap: "The zero vector can be an eigenvector / a basis vector." Reality: the zero vector is never independent and never in a basis.

Trap: "More vectors means a bigger span / dimension." Reality: extra dependent vectors add nothing; dimension counts independent directions only.

11 Connections with other topics
  • Matrices: a matrix is a machine that moves vectors around.
  • Rank: the rank of a matrix = number of independent directions in its columns (this topic's idea, applied).
  • Linear equations: "does $Ax=b$ have a solution?" is really "is $b$ in the span of $A$'s columns?"
  • Eigenvectors: special directions that don't rotate under a matrix.
  • Orthogonality: the dot product of two vectors measures length and angle; a zero dot product means perpendicular.
  • SVD & PCA: data is a cloud of vectors; PCA finds the few directions that capture most of its spread.
  • Basis & dimension reappear everywhere — null space, column space, eigenspaces.
12 Practice problems

Q1. Is $(2,3,4)$ in $\mathbb{R}^2$?

No — it has 3 components, so it lives in $\mathbb{R}^3$.

Q2. Is $W=\{(x,y): x+y=5\}$ a subspace of $\mathbb{R}^2$?

No — $(0,0)$ gives $0\neq5$, so $W$ doesn't contain the zero vector. Stop there.

Q3. Are $(1,0),(0,1),(4,5)$ independent?

Dependent — 3 vectors in $\mathbb{R}^2$ ($>n$) are always dependent.

Q4. What is $\dim(\operatorname{span}\{(1,2,3),(2,4,6)\})$?

$1$ — the second vector is $2\times$ the first, so they're parallel; the span is a line.

Q5. Write $(5,7)$ as a combination of $(1,0)$ and $(0,1)$.

$(5,7)=5(1,0)+7(0,1)$.

Quick MCQ

Which of the following is not a subspace of $\mathbb{R}^2$?
A line through the origin — it is a subspace.
The trivial subspace — it is a subspace.
$\mathbb{R}^2$ is a subspace of itself.
✅ Correct — $x+y=1$ excludes $(0,0)$, so not a subspace.
