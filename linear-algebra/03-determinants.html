Topic 3 · GATE weight: High Light

Determinants

Think of it as the matrix's health meter: one number that says whether the matrix can be reversed, how much it scales area/volume, and whether it flips space.

2 Why this topic exists

The determinant squeezes a whole square matrix into a single number that answers the most important yes/no question: is this matrix invertible? If $\det=0$, the matrix crushes space and can't be undone. The same number also measures how much area (2-D) or volume (3-D) is scaled, and its sign tells you if orientation flipped.

3 Game / visual imagination
🎮 Imagine it

Take a rubber unit square (area 1) and push it through the matrix portal. $|\det|$ = the area of the squished/stretched shape. $\det=2$ → area doubles; $\det=\tfrac12$ → area halves; $\det=0$ → it collapses to a line (info lost, not invertible); $\det<0$ → space got mirror-flipped.

The sign is about orientation:

  • $\det>0$ → orientation preserved (rotation/stretch, no mirror).
  • $\det<0$ → orientation flipped (a reflection happened).
  • $\det=0$ → space collapsed to a line/point; columns are dependent.

In 2-D the two columns of $A$ form a parallelogram, and $\det$ is its signed area. Play with the Matrix Transformer: the shaded square is $|\det|$, it turns coral when $\det<0$, and the Singular preset collapses it to a line ($\det=0$).

4 Basic definitions
  • Determinant $\det(A)$ (also $|A|$): a single number showing how much a square matrix scales area/volume; defined only for square matrices.
  • Singular: $\det=0$ — not invertible, dependent columns, space collapses. Non-singular: $\det\neq0$ — invertible, independent columns.
  • Minor $M_{ij}$: the determinant left after deleting entry $a_{ij}$'s row and column.
  • Cofactor $C_{ij}$: a minor with a $+/-$ sign from the checkerboard, $C_{ij}=(-1)^{i+j}M_{ij}$.
  • Cofactor expansion: computing $\det$ by expanding along any row or column using cofactors.
  • Adjugate (adjoint): the transpose of the cofactor matrix; appears in the inverse formula.
📖 Study in Depth — determinants from zero, one idea per card
1. What is a determinant?

A determinant is a single number computed from a square matrix, written det(A) or |A| (same thing).

A = [1  2        det(A) or |A|
     3  4]

Important: only square matrices (2×2, 3×3, n×n) have a determinant — a 2×3 has none.

2. What the determinant tells you

For a square matrix, det(A) answers three things at once:

  • Is the matrix invertible?
  • How much does area / volume change?
  • Does orientation flip?

Most important rule:

det(A) = 0   →  singular     →  inverse does NOT exist
det(A) ≠ 0   →  non-singular →  inverse exists
3. Game imagination: the rubber tile

Picture a unit square rubber tile (area 1). A matrix stretches, squashes, rotates, flips or collapses it. The determinant says what happened to the area:

det = 2     area becomes 2× bigger
det = 0.5   area becomes half
det = -3    area 3× bigger AND orientation flipped
det = 0     area becomes zero — space collapsed to a line/point
4. Determinant = area scaling (2D)

In 2D, |det(A)| is the area scaling factor.

|det(A)| = 5   any area becomes 5× bigger
|det(A)| = 1   area unchanged
|det(A)| = 0   area collapses to zero

det = 0 means the 2D map became a 1D road or a point — you cannot recover the plane, so there is no inverse.

5. Determinant = volume scaling (3D)

In 3D, |det(A)| is the volume scaling factor.

det = 8    volume 8× bigger
det = -2   volume 2× bigger, orientation flipped
det = 0    3D space collapsed into a plane, line, or point
6. Positive, negative, zero determinant 
det > 0   orientation preserved   (e.g. rotation + stretch)
det < 0   orientation flipped     (a reflection / mirror)
det = 0   space collapsed: singular, dependent columns, rank < n

The sign is the orientation; the size is the scaling.

7. The 2×2 formula: ad − bc 
A = [a  b        det(A) = ad − bc
     c  d]        (main diagonal − other diagonal)
A = [1  2        det = 1×4 − 2×3 = −2
     3  4]

So area becomes 2×, orientation flips, and (det ≠ 0) the matrix is invertible.

8. The 2×2 visual: signed area

The two columns of A form a parallelogram; det is its signed area.

A = [1  0     columns [1,0] and [0,1] → unit square
     0  1]    det = 1

A = [2  0     stretch x×2, y×3
     0  3]    det = 6  → area 6×
9. Why det = 0 means collapse 
A = [1  2        det = 1×4 − 2×2 = 0
     2  4]

columns: [1,2] and [2,4] = 2·[1,2]  (same direction!)

Both columns point the same way, so they span only a line, not an area — area 0. Hence det = 0 ⇔ dependent columns, rank < n, no inverse.

10. The 3×3 determinant (cofactor expansion) 
A = [a  b  c
     d  e  f
     g  h  i]

det = a(ei−fh) − b(di−fg) + c(dh−eg)

This is cofactor expansion along the first row; the signs alternate + − +.

11. What is a minor? GATE extra

The minor of an entry is the smaller determinant left after deleting that entry's row and column.

minor of a = |e  f|  = ei − fh
             |h  i|

minor of b = |d  f|  = di − fg
             |g  i|
12. Cofactor = sign × minor GATE extra

A cofactor attaches the checkerboard sign to a minor.

sign pattern:  + − +
               − + −
               + − +

So det = a·(+minor_a) + b·(−minor_b) + c·(+minor_c) = a(ei−fh) − b(di−fg) + c(dh−eg).

13. Worked 3×3 example 
A = [2  1  3
     0  4  2
     1  0  5]

det = 2·|4 2; 0 5| − 1·|0 2; 1 5| + 3·|0 4; 1 0|
    = 2(20) − 1(−2) + 3(−4)
    = 40 + 2 − 12
    = 30

Volume scales by 30, orientation preserved, matrix invertible.

14. Expand along any row or column

You may expand along any row or column — pick the one with the most zeros to save work.

A = [2  1  3
     0  4  2     ← row 2 / column 1 each have a zero
     1  0  5]

GATE shortcut: more zeros = faster expansion.

15. Determinant of a diagonal matrix 
D = [2  0  0
     0  3  0      det(D) = 2 × 3 × 5 = 30
     0  0  5]

Diagonal determinant = product of the diagonal — no expansion needed.

16. Determinant of a triangular matrix

Upper or lower triangular — same rule:

U = [2  7  8        L = [2  0  0
     0  3  4             7  3  0
     0  0  5]            8  4  5]

det(U) = det(L) = 2 × 3 × 5 = 30

A huge GATE time-saver.

17. Determinant of the identity 
det(Iₙ) = 1

The identity changes nothing, so it scales area/volume by 1.

18. Determinant of the zero matrix 
det(O) = 0

It collapses everything to the origin, so it is never invertible.

19. Property: det(AB) = det(A)·det(B) 
det(AB) = det(A) × det(B)

Scalings multiply: if A scales by 4 and B by 5, then AB scales by 20.

det(A²) = det(A)²,   det(Aᵏ) = det(A)ᵏ

Note det(AB) = det(BA) even though AB ≠ BA — both equal det(A)det(B).

20. Property: det(Aᵀ) = det(A)

Transpose does not change the determinant.

A = [1  2        det(A)  = 1×4 − 2×3 = −2
     3  4]

Aᵀ = [1  3       det(Aᵀ) = 1×4 − 3×2 = −2
21. Property: det(A⁻¹) = 1 / det(A) 
det(A⁻¹) = 1 / det(A)      (when A is invertible)

Why? A A⁻¹ = I ⇒ det(A)·det(A⁻¹) = 1. So if det(A) = 5, then det(A⁻¹) = 1/5.

22. Property: det(cA) = cⁿ·det(A) (trap!)

For an n × n matrix, multiplying the whole matrix by c multiplies every one of the n rows by c:

det(cA) = cⁿ · det(A)      (NOT c · det(A))
A is 3×3, det(A) = 4
det(2A) = 2³ × 4 = 8 × 4 = 32

GATE trap: the power is the matrix size n.

23. Row operations and the determinant
  • Swap two rows → determinant changes sign.
  • Multiply a row by k → determinant × k.
  • Add a multiple of one row to another → determinant unchanged.
R₂ → R₂ − 3R₁ on [1 2; 3 4]  gives  [1 2; 0 −2]
det stays −2.   (used in Gaussian elimination)

The same three rules hold for columns.

24. When the determinant is automatically 0
  • Two identical rows (or columns) → det = 0.
  • A completely zero row (or column) → det = 0.
  • One row/column is a multiple of another (dependent) → det = 0.
[1 2; 2 4]  → row2 = 2·row1 → det = 0
25. Determinant and rank 
det(A) ≠ 0   ⇔   rank(A) = n   (full rank)
det(A) = 0   ⇔   rank(A) < n

For a 3×3 with det = 0, the rank is 0, 1, or 2 — never 3.

26. Determinant, inverse & the 2×2 formula 
A = [a  b        A⁻¹ = 1/(ad−bc) · [ d  −b
     c  d]                            −c   a]

The denominator ad − bc is det(A). If det(A) = 0 the denominator is 0 — so the inverse cannot exist. That is exactly why det = 0 ⇒ no inverse.

27. Determinant of an orthogonal matrix

If AᵀA = I then det(A)² = 1, so:

det(A) = ±1

+1 = rotation (orientation preserved); −1 = reflection (flipped). Orthogonal maps preserve area/volume.

28. Determinant of idempotent / nilpotent GATE extra 
idempotent (A² = A):  det = 0 or 1
nilpotent  (Aᵏ = 0):  det = 0

From det(A)² = det(A) ⇒ det ∈ {0,1}; and det(A)ᵏ = 0 ⇒ det = 0.

29. Determinant of a scalar matrix 
A = cIₙ        det(cIₙ) = cⁿ
5·I₃ = diag(5,5,5)   det = 5³ = 125
30. Determinant of a block triangular matrix GATE extra 
M = [A  B        M = [A  0
     0  D]            C  D]

det(M) = det(A) · det(D)   (A, D square blocks)
31. Determinant by row reduction

Reduce to triangular form with row ops (the "add a multiple" op keeps det unchanged), then multiply the diagonal.

[1 2 3; 2 5 6; 3 8 10]
 → R₂−2R₁, R₃−3R₁ → [1 2 3; 0 1 0; 0 2 1]
 → R₃−2R₂          → [1 2 3; 0 1 0; 0 0 1]

det = 1 × 1 × 1 = 1
32. Determinant as an independence test

Put n vectors of ℝⁿ as columns; det ≠ 0 ⇔ they are independent.

v₁=[1,2], v₂=[3,4] → A = [1 3; 2 4]
det = 1×4 − 3×2 = −2 ≠ 0  → independent
33. det(A) = product of eigenvalues 
det(A) = λ₁ · λ₂ · … · λₙ          (product of eigenvalues)
tr(A)  = λ₁ + λ₂ + … + λₙ          (sum — that is the trace)
eigenvalues 2, 3, 5  →  det = 30,  trace = 10

(The characteristic equation that produces these λ lives in the Eigenvalues topic.)

34. Determinant vs trace — don't confuse them 
trace(A)       = sum of diagonal     = sum of eigenvalues
determinant(A) = area/volume scale  = product of eigenvalues
A = [2 0; 0 3]   trace = 5,   det = 6
35. Determinant and orientation GATE extra 
I = [1 0; 0 1]      det = 1   → orientation preserved
R = [1 0; 0 −1]     det = −1  → reflection (flipped)

Positive det = no mirror flip; negative det = mirror flip.

36. Adjoint (adjugate) & the inverse formula GATE extra

The cofactor matrix C has entries Cᵢⱼ = (−1)^{i+j}·minorᵢⱼ. Its transpose is the adjugate:

adj(A) = Cᵀ
A · adj(A) = adj(A) · A = det(A) · I
A⁻¹ = adj(A) / det(A)        (det(A) ≠ 0)

2×2 shortcut:

A = [a b; c d]  →  adj(A) = [d −b; −c a]  →  A⁻¹ = (1/(ad−bc))·adj(A)

GATE fact: for an n×n matrix, det(adj A) = det(A)^{n−1}.

37. Common GATE shortcuts
  • Triangular / diagonal → det = product of diagonal.
  • Any zero row/column, or repeated/proportional rows → det = 0.
  • det = 0 ⇒ singular, rank-deficient, dependent, no unique solution.
  • det ≠ 0 ⇒ invertible, full rank, independent, unique Ax=b solution.
  • det(cA) = cⁿ det(A)  ·  det(AB) = det(A)det(B).
38. Important true / false
  • det(A) = 0 ⇒ A invertible? False — it is NOT invertible.
  • det(A+B) = det(A) + det(B)? False — det is multiplicative, not additive (e.g. det(I+I)=4 ≠ 2).
  • det(kA) = k·det(A)? False — it is kⁿ·det(A).
  • Triangular det = sum of diagonal? False — it is the product (sum is the trace).
  • det(Aᵀ)=det(A), orthogonal ⇒ det=±1, det=∏λ? All true.
39. Determinant decision tree 
1. Square?            no → not defined
2. Triangular/diag?   yes → product of diagonal
3. Zero row/col?      yes → 0
4. Repeated/proportional row/col? yes → 0
5. 2×2?               ad − bc
6. 3×3?               cofactor expansion / row-reduce
7. Eigenvalues given? product of eigenvalues
8. Product/inverse/transpose/scalar? use properties

6 Main formulas

Computing the determinant

$2\times2$ — main diagonal minus anti-diagonal:

$$ \det\begin{bmatrix}a&b\\c&d\end{bmatrix}=ad-bc $$

$3\times3$ — cofactor expansion along the first row:

$$ \det\begin{bmatrix}a&b&c\\d&e&f\\g&h&i\end{bmatrix}=a(ei-fh)-b(di-fg)+c(dh-eg) $$

Cofactor — a minor times its checkerboard sign:

$$ C_{ij}=(-1)^{i+j}M_{ij} $$

Key properties

Product & transpose — $\det$ of a product multiplies; transpose keeps it:

$$ \det(AB)=\det(A)\det(B),\qquad \det(A^{\mathsf T})=\det(A) $$

Inverse & triangular — invertible iff $\det\neq0$; a triangular det is the product of its diagonal:

$$ \det(A^{-1})=\frac{1}{\det(A)},\qquad \det(\text{triangular})=\prod_i a_{ii} $$

Inverse via adjugate:

$$ A^{-1}=\frac{1}{\det(A)}\operatorname{adj}(A) $$
7 Step-by-step methods

Cofactor expansion (3×3):

  1. Pick the row or column with the most zeros (less work).
  2. For each entry, cross out its row and column → the leftover $2\times2$ determinant is its minor.
  3. Apply the checkerboard signs.
  4. Sum: $\text{entry}\times(\text{signed minor})$ across the line.

Row-reduction method (fast):

  1. Use $R_i\to R_i+kR_j$ (these don't change $\det$) to make it triangular.
  2. Track any row swaps (each flips the sign) and any row scaling (multiplies $\det$).
  3. $\det=$ product of the diagonal of the triangular form (adjusted for swaps/scales).
8 Worked examples
✦ 3×3 by cofactors

$\det\begin{bmatrix}2&1&3\\0&4&2\\1&0&5\end{bmatrix}=2(4\cdot5-2\cdot0)-1(0\cdot5-2\cdot1)+3(0\cdot0-4\cdot1)$

$=2(20)-1(-2)+3(-4)=40+2-12=\mathbf{30}$.

✦ Cramer's rule for $2x+y=5,\ x+3y=7$

$A=\begin{bmatrix}2&1\\1&3\end{bmatrix},\ \det A=5\neq0$ ⇒ unique solution.

$x=\dfrac{\det\begin{bmatrix}5&1\\7&3\end{bmatrix}}{\det A}=\dfrac{8}{5},\qquad y=\dfrac{\det\begin{bmatrix}2&5\\1&7\end{bmatrix}}{\det A}=\dfrac{9}{5}.$

9 GATE cheat sheet
⚡ Properties to memorize
  • $\det(AB)=\det A\det B$  ⇒  $\det(A^k)=(\det A)^k$, and $\det(AB)=\det(BA)$.
  • $\det(A^{\mathsf T})=\det A$;   $\det(A^{-1})=1/\det A$.
  • $\det(cA)=c^{\,n}\det A$ for an $n\times n$ matrix (size is the exponent!).
  • $\det(cI_n)=c^n$;   block-triangular $\det\begin{bmatrix}A&B\\0&D\end{bmatrix}=\det A\,\det D$.
  • Triangular / diagonal → $\det=$ product of the diagonal; $\det=\prod\lambda_i$.
  • Orthogonal matrix → $\det=\pm1$. Idempotent → $\det\in\{0,1\}$. Nilpotent → $\det=0$.
⚡ Row / column operations
OperationEffect on $\det$
Swap two rows (or columns)sign flips ($\times(-1)$)
Multiply a row by $k$$\det\times k$
$R_i\to R_i+kR_j$unchanged

$\det=0$ instantly if: a row/column is all zeros, two rows/columns are equal, or one row/column is a multiple of another.

10 Common traps
✕ Watch out

Trap: "$\det(A+B)=\det A+\det B$." Reality: false — det is multiplicative, not additive. (E.g. $A=B=I_2$: LHS $=4$, RHS $=2$.)

Trap: "$\det(2A)=2\det A$." Reality: $\det(2A)=2^n\det A$ — e.g. $2^3$ for a $3\times3$.

Trap: "Triangular determinant is the sum of the diagonal." Reality: it's the product (the sum is the trace).

Trap: "Non-square matrices have determinants." Reality: only square matrices do.

11 Connections with other topics
🔑 The invertibility chain (GATE gold)

For an $n\times n$ matrix $A$, all of these say the same thing:

$\det(A)\neq0 \iff A^{-1}$ exists $\iff \operatorname{rank}(A)=n \iff$ columns are independent $\iff Ax=0$ has only $x=0 \iff Ax=b$ has a unique solution.

  • Rank: square $\det=0 \iff$ rank $<n$ (dependent columns).
  • Eigenvalues: $\det=\prod\lambda$; a zero eigenvalue ⇒ $\det=0$.
  • Trace: don't confuse — trace is the sum of the diagonal/eigenvalues; det is the product.
  • Linear equations: $\det\neq0$ ⇒ unique solution; $\det=0$ ⇒ none or infinite (check ranks).
12 Practice problems

Q1. $\det(A)=5$ for a $3\times3$ matrix. Find $\det(2A)$.

$2^3\cdot5=40$.

Q2. $\det\begin{bmatrix}2&4&7\\0&3&8\\0&0&5\end{bmatrix}=?$

Upper-triangular → $2\cdot3\cdot5=30$.

Q3. $\det A=5,\ \det B=3$. Find $\det(AB)$ and $\det(A^{-1})$.

$\det(AB)=15$; $\det(A^{-1})=1/5$.

Q4. Eigenvalues of $A$ are $2,-1,4$. Find $\det A$.

$\det A=2\cdot(-1)\cdot4=-8$ (negative ⇒ orientation flips).

Q5. Is $A=\begin{bmatrix}2&6\\1&3\end{bmatrix}$ invertible?

$\det=2\cdot3-6\cdot1=0$ → singular, not invertible (column 2 $=3\times$ column 1).

Q6. If $A$ is orthogonal, what values can $\det A$ take?

$\pm1$ (from $\det(A^{\mathsf T}A)=\det I=1$ ⇒ $(\det A)^2=1$).

✅ Correct — both equal $\det A\,\det B$.

13 Final memory box
🧠 Remember
  • $|\det|$ = area/volume scale; sign = orientation; $\det=0$ ⇒ collapse, not invertible.
  • $2\times2$: $ad-bc$. Triangular/diagonal: product of diagonal.
  • $\det(AB)=\det A\det B$, $\det(A^{\mathsf T})=\det A$, $\det(A^{-1})=1/\det A$, $\det(cA)=c^n\det A$.
  • $\det=\prod\lambda_i$; zero row / equal rows / proportional rows ⇒ $\det=0$.
  • Invertible $\iff \det\neq0 \iff$ full rank $\iff$ independent columns.

GATE takeaway — det(cA) = cⁿ·det(A), triangular det = product of the diagonal, and det(A+B) ≠ det(A)+det(B).
