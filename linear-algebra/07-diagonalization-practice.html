Diagonalization

8 questions. Tap any option — you'll see why it's right or wrong, and the correct one lights up.

Q1. A $3\times3$ matrix has eigenvalues $2,4,7$ (all distinct). Diagonalizable?
✅ Correct — $n$ distinct eigenvalues guarantee $n$ independent eigenvectors.
Distinct eigenvalues always give diagonalizability.
Symmetry is sufficient but not necessary; distinct $\lambda$ already suffice.
Distinct eigenvalues settle it without the full matrix.
Q2. In $A=PDP^{-1}$, what are $P$ and $D$?
Swapped.
✅ Correct — $P$'s columns are eigenvectors; $D$'s diagonal holds the matching eigenvalues.
$P$ holds eigenvectors.
$D$ is diagonal with eigenvalues.
Q3. Which matrix is not diagonalizable?
Already diagonal.
Symmetric ⇒ always diagonalizable.
✅ Correct — repeated $\lambda=1$ but only one eigenvector (defective).
$I$ is already diagonal.
Q4. $A$ is diagonalizable iff…
Invertibility is unrelated — a defective invertible matrix exists.
✅ Correct — needs a full set of $n$ independent eigenvectors.
Same issue as A.
Distinct is sufficient but not necessary (e.g. $I$).
Q5. If $A=PDP^{-1}$, then $A^k=?$
The $P$'s telescope — only $D$ gets the power.
Needs $P^{-1}$ on the right, not $P$.
✅ Correct — $A^k=PD^kP^{-1}$; only the diagonal gets raised to $k$.
Powers aren't linear in $k$.
Q6. Every symmetric matrix can be diagonalized…
Symmetric matrices are always diagonalizable.
✅ Correct — the spectral theorem: $A=QDQ^{\mathsf T}$ with orthogonal $Q$.
No invertibility needed.
Real symmetric matrices diagonalize over the reals.
Q7. A repeated eigenvalue means the matrix is…
Not necessarily — $I$ has a repeated eigenvalue and is diagonal.
Not always — defective matrices repeat $\lambda$ with too few eigenvectors.
✅ Correct — it works iff geometric = algebraic multiplicity for that $\lambda$.
Repetition says nothing about $\det$.
Q8. $D=\operatorname{diag}(1,2)$, $A=PDP^{-1}$. $A^5$ uses which diagonal?
That's $5\times$; you need $\lambda^5$.
✅ Correct — $D^5=\operatorname{diag}(1^5,2^5)=\operatorname{diag}(1,32)$.
That's $D^1$, not $D^5$.
Each entry is raised to the 5th power individually.
