Topic 2 · GATE weight: Very High

Matrices

A matrix is a table of numbers — and, more powerfully, a machine that transforms vectors. This is the workhorse of GATE linear algebra.

2 Why this topic exists

Vectors are arrows; a matrix is a machine that moves those arrows — it can stretch, rotate, flip, shear, or project them. The same object also stores data tables and packs a whole system of equations into one symbol.

Almost every later topic is "matrices, asked a different way": determinants, rank, inverse, eigenvalues, systems of equations, PCA. Master the operations here and the rest gets easy.

4 Basic definitions
  • Matrix: a rectangular table of numbers. An $m\times n$ matrix has $m$ rows and $n$ columns (rows first — remember "RC cola"). Entry $a_{ij}$ sits in row $i$, column $j$.
  • Square: rows = columns. Column vector: $n\times1$. Row vector: $1\times n$.

A matrix can also just be a data table (rows = samples, columns = features). And it acts on vectors: e.g. $\begin{bmatrix}0&-1\\1&0\end{bmatrix}$ sends $(1,0)\mapsto(0,1)$ — a $90°$ rotation. Same object, two views: a table of data and a transformation machine.

📖 Study in Depth — matrices from zero, one idea per card

Think of matrices as the next level after vectors. A vector is an arrow; a matrix is a machine that changes arrows — it can move, stretch, rotate, flip, shear or project them, and it can also hold a data table or a system of equations. Matrices connect to almost everything in GATE DA: linear equations, rank, determinant, inverse, eigenvalues, orthogonality, projection, PCA, least squares and ML. Open each card below in order.

1. What is a matrix?

A matrix is a rectangular table of numbers.

A = [1  2
     3  4]

This has 2 rows and 2 columns, so its size is 2 × 2 ("two by two").

B = [1  2  3
     4  5  6]      →  B is a 2 × 3 matrix

An m × n matrix has m rows and n columns — rows first, columns second.

Memory trick: RC cola — Rows first, Columns second.

2. Matrix entry: what aᵢⱼ means

Every number has a position, written aᵢⱼ ("a i j") = the entry in row i, column j.

A = [1  2
     3  4]

a₁₁ = row 1, col 1 = 1
a₁₂ = row 1, col 2 = 2
a₂₁ = row 2, col 1 = 3
a₂₂ = row 2, col 2 = 4

Always row first, column second. For example, in

A = [5  8  2
     7  1  9]

a₂₃ = row 2, col 3 = 9
a₁₂ = row 1, col 2 = 8
3. Matrix as a data table

At the simplest level a matrix is just a table. Marks of students:

          Math  CS  Stats
Amit       80   90   70
Ravi       60   85   75
Sita       95   88   91

As a matrix (rows = samples, columns = features):

M = [80  90  70
     60  85  75
     95  88  91]

But in linear algebra the more powerful meaning is: matrix = transformation machine.

4. Matrix as a transformation machine

A matrix acts on a vector to produce a new vector.

A = [2  0      v = [1
     0  2]          3]

Av = [2×1 + 0×3     = [2
      0×1 + 2×3]       6]

So [1, 3] becomes [2, 6] — the vector is stretched by 2.

🎮 Imagine: the vector is the player's position and the matrix is a transformation portal — walk through it and your coordinates change.

5. Matrix size (and row / column vectors) 
A = [1  2  3
     4  5  6]      →  2 × 3

B = [1
     2
     3]            →  3 × 1   (a column vector)

C = [1  2  3]      →  1 × 3   (a row vector)

So a column vector is an n × 1 matrix and a row vector is a 1 × n matrix.

6. Why matrix size matters

Multiplication is allowed only when the inner dimensions match.

A is m × n
B is n × p
→ AB is possible, and AB is m × p

(m × n)(n × p) = m × p
        ↑    ↑  these must match
A = 2 × 3,  B = 3 × 4   → AB = 2 × 4   ✓
A = 2 × 3,  B = 2 × 4   → not possible (3 ≠ 2)

Memory: outside numbers become the answer size; inside numbers must match.

7. Matrix multiplication (worked) 
A = [1  2      B = [5  6
     3  4]          7  8]

Each entry of C = AB is a row of A dotted with a column of B.

C₁₁ = [1 2]·[5;7] = 1×5 + 2×7 = 19
C₁₂ = [1 2]·[6;8] = 1×6 + 2×8 = 22
C₂₁ = [3 4]·[5;7] = 3×5 + 4×7 = 43
C₂₂ = [3 4]·[6;8] = 3×6 + 4×8 = 50

AB = [19  22
      43  50]
8. Why multiplication looks strange

Matrix multiplication is not just number multiplication — it means applying one transformation after another.

A = rotate,  B = stretch
AB means: first apply B, then apply A

Order matters: "rotate then stretch" is not the same as "stretch then rotate". That is why AB ≠ BA in general — one of the biggest GATE traps.

9. AB ≠ BA

For numbers 2×3 = 3×2, but for matrices order usually matters.

A = [1  2      B = [1  0
     0  1]          3  1]

AB = [7  2          BA = [1  2
      3  1]                3  7]

→ AB ≠ BA

GATE memory: matrix multiplication is associative — A(BC) = (AB)C — but not commutative.

10. Matrix addition

Add matrices only if they are the same size; addition is component-wise.

A = [1  2      B = [5  6
     3  4]          7  8]

A + B = [6   8
         10  12]
allowed:    2×2 + 2×2,   3×4 + 3×4
not allowed: 2×3 + 3×2
11. Scalar multiplication

A scalar multiplies every entry.

A = [1  2          3A = [3   6
     3  4]                9   12]
12. Identity matrix

The identity I is the matrix version of the number 1: AI = IA = A.

I₂ = [1  0       I₃ = [1  0  0
      0  1]            0  1  0
                       0  0  1]

1 on the diagonal, 0 everywhere else. A vector passes through unchanged:

I·[3      [3
   4]  =   4]

🎮 Imagine: the identity is the "no-effect" portal.

13. Diagonal matrix

A diagonal matrix has non-zero values only on the diagonal — it scales each axis independently.

D = [2  0          Dv = [2x
     0  3]                3y]

x is stretched by 2, y by 3. 🎮 Imagine: a stretch machine.

14. Square matrix

A matrix is square when rows = columns (2×2, 3×3, …). Only square matrices can have a determinant, inverse, eigenvalues and trace — most GATE matrix properties are about square matrices.

15. Zero matrix

The zero matrix has all entries 0 and sends every vector to the origin: Ov = 0.

O = [0  0
     0  0]

🎮 Imagine: a black-hole portal — everything collapses to the origin.

16. Transpose (Aᵀ)

Transpose flips rows into columns. If A is m × n, then Aᵀ is n × m.

A = [1  2  3        Aᵀ = [1  4
     4  5  6]              2  5
                           3  6]

Properties:

  • (Aᵀ)ᵀ = A
  • (A + B)ᵀ = Aᵀ + Bᵀ
  • (cA)ᵀ = cAᵀ
  • (AB)ᵀ = BᵀAᵀ — the order reverses (a classic GATE trap).
17. Symmetric matrix

A matrix is symmetric if A = Aᵀ — a mirror image across the diagonal (must be square).

A = [1  2  3
     2  5  6
     3  6  9]      aᵢⱼ = aⱼᵢ

GATE facts:

  • Real symmetric matrices have real eigenvalues.
  • Eigenvectors for distinct eigenvalues are orthogonal.
18. Skew-symmetric matrix

A matrix is skew-symmetric if A = −Aᵀ, i.e. aᵢⱼ = −aⱼᵢ.

A = [ 0   2  -3
     -2   0   4
      3  -4   0]

GATE fact: the diagonal entries are always 0, because aᵢᵢ = −aᵢᵢ ⇒ aᵢᵢ = 0.

19. Upper triangular matrix

Everything below the diagonal is zero.

U = [2  5  7
     0  3  9
     0  0  4]      det(U) = 2×3×4 = 24

GATE fact: det = product of the diagonal entries.

20. Lower triangular matrix

Everything above the diagonal is zero.

L = [2  0  0
     5  3  0
     7  9  4]      det(L) = 2×3×4 = 24

Same rule: det = product of the diagonal.

21. Trace

Trace = sum of the diagonal entries, written tr(A).

A = [1  2  3
     4  5  6        tr(A) = 1 + 5 + 9 = 15
     7  8  9]

Properties:

  • tr(A + B) = tr(A) + tr(B),   tr(cA) = c·tr(A),   tr(Aᵀ) = tr(A)
  • tr(AB) = tr(BA) even though AB ≠ BA.
  • Trace = sum of eigenvalues — if eigenvalues are 2, 3, 5 then tr(A) = 10.
22. Determinant

The determinant is a single number for a square matrix, written det(A) or |A|.

A = [a  b        det(A) = ad − bc
     c  d]

A = [1  2        det = 1×4 − 2×3 = −2
     3  4]

🎮 Imagine: the determinant is the area / volume scaling factor. det = 2 doubles area; det = 0 collapses space to a line or point (the matrix loses a dimension).

GATE rule: A⁻¹ exists iff det(A) ≠ 0. det = 0 → singular (not invertible); det ≠ 0 → non-singular (invertible).

23. Inverse matrix

The inverse is the "undo" button: it reverses the transformation.

A A⁻¹ = A⁻¹ A = I

Just as 5 × 1/5 = 1 for numbers. 🎮 Imagine: A is a portal, A⁻¹ is the reverse portal, and I is the unchanged original.

24. The 2×2 inverse formula 
A = [a  b
     c  d]

A⁻¹ = 1/(ad−bc) · [ d  −b
                     −c   a]      (only if ad−bc ≠ 0)

Example:

A = [1  2      det = −2
     3  4]

A⁻¹ = (1/−2)·[ 4  −2      = [−2    1
              −3   1]         1.5 −0.5]
25. Inverse of a product

(AB)⁻¹ = B⁻¹A⁻¹ — the order reverses.

🎮 Imagine: socks then shoes — to undo, remove shoes first, then socks. Same reversal as transpose: (AB)ᵀ = BᵀAᵀ.

Memory: reverse the order for both inverse and transpose.

26. Orthogonal matrix

A matrix is orthogonal if AᵀA = I, which means A⁻¹ = Aᵀ — the inverse is just the transpose.

🎮 Imagine: a pure rotation / reflection machine — it preserves length, angle and distance.

GATE facts:

  • AᵀA = I and A⁻¹ = Aᵀ
  • det(A) = ±1
  • columns (and rows) are orthonormal = orthogonal (90°) + unit length.
27. Idempotent matrix

Idempotent means A² = A — applying it twice is the same as once.

P = [1  0       P[x      [x        P² = P
     0  0]         y]  =   0]   (a projection onto the x-axis)

🎮 Imagine: a projection / snap-to-ground machine. GATE fact: eigenvalues are only 0 or 1.

28. Nilpotent matrix

Nilpotent means Aᵏ = 0 for some positive integer k — enough applications send everything to zero.

A = [0  1        A² = [0  0
     0  0]              0  0]

GATE facts: all eigenvalues are 0; also det(A) = 0 and tr(A) = 0.

29. Matrix–vector multiplication is the heart 
A = [2  0        Av = [2x      (stretch x×2, y×3)
     0  3]              3y]

A = [0  -1       A[1     [0       (90° rotation:
     1   0]        0] =   1]       [1,0] → [0,1])

So a matrix is not just a table — it is an action on space.

30. How the columns explain a transformation

The columns of A tell you where the basis vectors go.

e₁ = [1      e₂ = [0
      0]            1]

Ae₁ = first column of A
Ae₂ = second column of A
A = [2  1       Ae₁ = [2      Ae₂ = [1
     0  3]             0]            3]

The x-axis direction moves to [2,0] and the y-axis to [1,3] — the matrix transforms the whole grid. This is the best way to see a matrix.

31. Rank

Rank = the number of independent rows/columns = the number of useful directions the matrix keeps.

[1 0; 0 1] keeps 2D as 2D        → rank 2
[1 0; 0 0] collapses to a line   → rank 1
[0 0; 0 0] collapses to a point  → rank 0

GATE: for an n×n matrix, det(A) ≠ 0 ⇔ rank(A) = n ⇔ inverse exists.

32. Singular vs non-singular (the big equivalence)

For an n × n matrix A, all of these say the same thing:

A is invertible
⇔ det(A) ≠ 0
⇔ rank(A) = n
⇔ columns are linearly independent
⇔ Ax = 0 has only the trivial solution

If instead det(A) = 0: A is singular — not invertible, rank < n, dependent columns, space collapses. GATE loves this chain.

33. Eigenvalues & eigenvectors (a first look)

Most vectors change direction under a matrix. A few special ones do not — they only stretch or shrink. Those are eigenvectors; the stretch factor is the eigenvalue λ.

A v = λ v     (A = matrix, v = eigenvector, λ = eigenvalue)
λ = 2   vector doubles
λ = 1   vector unchanged
λ = 0   vector collapses to zero
λ = -1  vector flips direction

🎮 Imagine: through the portal most arrows turn, but eigenvectors keep their direction — only their length changes.

35. Common GATE traps
  • AB = BA? Usually false — AB ≠ BA in general.
  • (AB)ᵀ = AᵀBᵀ? False — it is BᵀAᵀ.
  • (AB)⁻¹ = A⁻¹B⁻¹? False — it is B⁻¹A⁻¹.
  • det(A) = 0 but inverse exists? No — det = 0 means no inverse.
  • A orthogonal ⇒ A⁻¹ = Aᵀ. True.
  • A skew-symmetric ⇒ diagonal entries are 0. True.
  • A triangular ⇒ det = product of diagonal. True.
36. Quick self-check

Q1. Size of [[1,2,3],[4,5,6]]?

2 rows, 3 columns  →  2 × 3

Q2. A = 2×3, B = 3×4 — is AB possible?

Yes — inner 3 = 3.  AB is 2 × 4.

Q3. A = 2×3, B = 2×4 — is AB possible?

No — inner 3 ≠ 2.

Q4. det of [[2,3],[4,5]]?

2×5 − 3×4 = −2  → non-zero, so A is invertible.

Q5. Is [[1,4],[4,9]] symmetric?

Aᵀ = [[1,4],[4,9]] = A  → yes, symmetric.

Q6. Is [[0,5],[−5,0]] skew-symmetric?

−Aᵀ = [[0,5],[−5,0]] = A  → yes, skew-symmetric.
37. Final mental model
  • Vector = arrow; matrix = machine that transforms arrows.
  • Multiplication = applying machines one after another (order matters).
  • Identity = no-change machine; inverse = undo machine.
  • Determinant = area/volume scaling; 0 means a collapse.
  • Rank = number of useful directions left; trace = sum of diagonal = sum of eigenvalues.
  • Transpose = flip rows/columns; orthogonal = rotation/reflection; idempotent = projection; nilpotent = eventually-zero.

6 Main formulas

Size rule: $(m\times n)(n\times p)=(m\times p)$ — inner numbers must match, outer numbers give the answer size.

$$ C = AB,\qquad C_{ij} = \sum_{k=1}^{n} A_{ik}\,B_{kj} \quad (\text{row } i \text{ of } A \cdot \text{column } j \text{ of } B) $$
$$ (AB)^{\mathsf T}=B^{\mathsf T}A^{\mathsf T},\qquad (AB)^{-1}=B^{-1}A^{-1},\qquad \operatorname{tr}(A)=\sum_i a_{ii} $$

For a $2\times2$ matrix the inverse is $\begin{bmatrix}a&b\\c&d\end{bmatrix}^{-1}=\dfrac{1}{ad-bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}$ (only if $ad-bc\neq0$).

7 Step-by-step: multiply two matrices
  1. Check sizes: $A$ is $m\times n$, $B$ is $n\times p$. Inner $n$'s match? Good; answer is $m\times p$.
  2. For entry $C_{ij}$, take row $i$ of $A$ and column $j$ of $B$.
  3. Multiply them term by term and add (a dot product).
  4. Repeat for every $i,j$ to fill the answer.
8 Worked example
✦ Multiply

$A=\begin{bmatrix}1&2\\3&4\end{bmatrix},\ B=\begin{bmatrix}5&6\\7&8\end{bmatrix}$

$C_{11}=1\cdot5+2\cdot7=19,\quad C_{12}=1\cdot6+2\cdot8=22$

$C_{21}=3\cdot5+4\cdot7=43,\quad C_{22}=3\cdot6+4\cdot8=50$

$\Rightarrow AB=\begin{bmatrix}19&22\\43&50\end{bmatrix}$

✦ Order matters: $AB \neq BA$

With $A=\begin{bmatrix}1&2\\0&1\end{bmatrix},\ B=\begin{bmatrix}1&0\\3&1\end{bmatrix}$:   $AB=\begin{bmatrix}7&2\\3&1\end{bmatrix}$ but $BA=\begin{bmatrix}1&2\\3&7\end{bmatrix}$. Different!

9 GATE cheat sheet
⚡ High-value facts
  • Multiplication is associative $A(BC)=(AB)C$ but not commutative $AB\neq BA$.
  • Transpose and inverse reverse order: $(AB)^{\mathsf T}=B^{\mathsf T}A^{\mathsf T}$, $(AB)^{-1}=B^{-1}A^{-1}$.
  • $\operatorname{tr}(AB)=\operatorname{tr}(BA)$ even though $AB\neq BA$. Also $\operatorname{tr}(A)=\sum\lambda_i$.
  • Diagonal / triangular matrix: $\det=$ product of diagonal entries.
  • $A^{-1}$ exists $\iff \det(A)\neq0$.
10 Common traps
✕ Watch out

Trap: "$AB=BA$ always." Reality: almost never — order matters.

Trap: "$(AB)^{\mathsf T}=A^{\mathsf T}B^{\mathsf T}$." Reality: the order reverses: $B^{\mathsf T}A^{\mathsf T}$.

Trap: "$A^2=A$ means $A=I$." Reality: any projection (idempotent) works; eigenvalues are just $0$ or $1$.

Trap: "Any two matrices can be multiplied." Reality: only if inner dimensions match.

11 Connections with other topics
🔑 The invertibility chain (GATE gold)

For a square $A$, these are all the same statement:

$A^{-1}$ exists $\iff \det(A)\neq0 \iff \operatorname{rank}(A)=n \iff$ columns independent $\iff Ax=0$ only at $x=0 \iff Ax=b$ has a unique solution.

  • Determinant & inverse: built from a (square) matrix; $A^{-1}$ exists iff $\det\neq0$.
  • Rank: independent columns/rows of the matrix.
  • Eigenvalues: special directions of the transformation; $\operatorname{tr}=\sum\lambda$, $\det=\prod\lambda$.
  • Linear equations: $Ax=b$ is a matrix acting on the unknown vector $x$.
  • PCA / ML: data lives in matrices; covariance is a symmetric matrix.
12 Practice problems

Q1. Can a $2\times3$ matrix multiply a $2\times2$ matrix (in that order)?

No — inner dimensions $3$ and $2$ don't match. $(2\times3)(2\times2)$ is undefined.

Q2. If $A$ is $4\times3$ and $B$ is $3\times5$, what size is $AB$?

$4\times5$ (outer numbers).

Q3. $\operatorname{tr}\begin{bmatrix}1&2&3\\4&5&6\\7&8&9\end{bmatrix}=?$

$1+5+9=15$.

Q4. For an orthogonal matrix $Q$, what is $Q^{-1}$?

$Q^{-1}=Q^{\mathsf T}$ (its transpose), since $Q^{\mathsf T}Q=I$.

Q5. Is $\begin{bmatrix}1&4\\4&9\end{bmatrix}$ symmetric? Is $\begin{bmatrix}0&5\\-5&0\end{bmatrix}$ skew-symmetric?

First: $A=A^{\mathsf T}$ → yes, symmetric. Second: $A^{\mathsf T}=\begin{bmatrix}0&-5\\5&0\end{bmatrix}=-A$ → yes, skew-symmetric (note the zero diagonal).

Q6. $\operatorname{tr}(AB)$ vs $\operatorname{tr}(BA)$ — equal or not?

Always equal (cyclic property), even though $AB\neq BA$ in general.

Quick MCQ

If $A$ is $3\times3$ with $A^2=A$, which must be true?
$A=I$ works, but many other idempotents do too.
$A=0$ works but isn't forced.
✅ Correct — $A^2=A\Rightarrow\lambda^2=\lambda\Rightarrow\lambda\in\{0,1\}$.
$\det$ could be $0$ or $1$.
13 Final memory box
🧠 Remember
  • Matrix = machine that transforms vectors; columns = where basis arrows land.
  • Size: $(m\times n)(n\times p)=m\times p$; inner must match.
  • $AB\neq BA$, but $A(BC)=(AB)C$ and $\operatorname{tr}(AB)=\operatorname{tr}(BA)$.
  • Reverse order: $(AB)^{\mathsf T}=B^{\mathsf T}A^{\mathsf T}$, $(AB)^{-1}=B^{-1}A^{-1}$.
  • Triangular/diagonal $\det=$ product of diagonal; $A^{-1}$ exists iff $\det\neq0$.

GATE takeaway — AB ≠ BA, and trace equals the sum of eigenvalues: two of the most common traps.
