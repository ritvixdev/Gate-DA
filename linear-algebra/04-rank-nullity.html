Rank & Nullity

Rank counts how many independent directions a matrix keeps. Nullity counts what it crushes to zero. Together they always add up to the number of columns.

2 Why this topic exists

When a matrix transforms space, some directions survive and some get flattened. Rank tells you how much "real information" the matrix carries; nullity tells you how much it destroys. This single idea decides whether equations have solutions and how many.

3 Game / visual imagination
🎮 Imagine it

A matrix is a portal. Rank = how many dimensions come out the other side. Null space = all the input arrows crushed to the origin; nullity = how many independent crushed directions there are.

Picture 3-D space going through the portal:

  • 3-D stays 3-D → rank 3 (nothing lost).
  • 3-D squashed to a plane → rank 2.
  • squashed to a line → rank 1; to a point → rank 0.

So rank = the dimension of the output = how many independent directions survive.

🧊 Rank in 3D drag to rotate · switch the rank
4 Basic definitions
  • Rank: the number of linearly independent rows (= independent columns); equivalently the number of non-zero rows after row-reduction.
  • Null space (kernel): all vectors $x$ with $Ax=0$. Nullity: its dimension.
  • Column space: all outputs $Ax$ (span of the columns). Row space: span of the rows.
  • Pivot: the first non-zero entry of a row after row-reduction — rank = number of pivots.
  • Row-echelon form: zero rows sit at the bottom; each leading entry is further right than the one above.
  • Full rank: rank $=\min(m,n)$ (nothing collapses). Rank-deficient: rank $\lt\min(m,n)$.
  • Free variable: an unknown not pinned by a pivot — each adds a dimension of solutions (nullity counts them).
📖 Study in Depth — rank & nullity from zero, one idea per card
1. What is rank? (intuition)

Picture a 3D game world with full movement (x, y, z = 3 independent directions). A matrix transforms it; rank = how many independent directions survive.

3D stays 3D            → rank 3
3D crushed to a plane  → rank 2
3D crushed to a line   → rank 1
everything → a point   → rank 0

So rank = dimension of the output space after the transformation.

2. Matrix as an information carrier

A matrix maps input x to output Ax. Keeping more independent directions = preserving more information.

[1 0; 0 1] keeps everything       → rank 2
[1 0; 0 0] destroys the y-direction → rank 1
[0 0; 0 0] destroys everything      → rank 0

High rank = more information preserved; low rank = more lost.

3. Formal definition

The rank is the maximum number of linearly independent rows — or columns.

Beautiful fact: row rank = column rank. That common number is rank(A).

4. Independent rows — example 
A = [1  2  3
     2  4  6        R2 = 2·R1  (dependent, no new info)
     1  1  1]       R3 is not a multiple of R1

Useful independent rows are R1 and R3, so rank(A) = 2.

5. Rank via row echelon form (the practical method)
  1. Row-reduce A to echelon form.
  2. Count the non-zero rows.
  3. That count is the rank.

Allowed row ops (swap, scale by non-zero, add a multiple of a row) do not change rank.

6. What is row echelon form?

Echelon form: zero rows at the bottom; each row's first non-zero entry sits to the right of the one above; entries below each pivot are zero.

[1 2 3; 0 1 4; 0 0 5]  → 3 non-zero rows → rank 3
[1 2 3; 0 1 4; 0 0 0]  → 2 non-zero rows → rank 2
7. Pivots — rank = number of pivots

A pivot is the first non-zero entry of a row after reduction.

[1 2 3; 0 5 6; 0 0 0]  → pivots: 1, 5 → rank = 2
8. Worked rank example 
A = [1 2 3; 2 4 6; 1 1 1]
R2 → R2 − 2R1  ⇒ [0,0,0]
R3 → R3 − R1   ⇒ [0,−1,−2]

[1  2  3
 0 -1 -2     → 2 non-zero rows
 0  0  0]    → rank(A) = 2
9. Rank limit: ≤ min(m, n) 
rank(A) ≤ min(m, n)        (A is m×n)

3×5 → rank ≤ 3
5×3 → rank ≤ 3
4×4 → rank ≤ 4

You cannot have more independent rows than rows, nor more independent columns than columns.

10. Full rank

Full rank means rank = min(m, n). For a square n×n matrix, full rank = n, and:

rank(A) = n
⇔ det(A) ≠ 0
⇔ A is invertible
⇔ columns independent
⇔ rows independent
11. Rank and determinant 
det(A) ≠ 0   →  rank = n  (full): invertible, independent, unique solution
det(A) = 0   →  rank < n: singular, dependent, Ax=0 has non-zero solutions

Memory: det ≠ 0 = full-power matrix; det = 0 = collapsed matrix.

12. Rank by the largest non-zero minor GATE extra

rank(A) = the order of the largest square submatrix with non-zero determinant.

A = [1 2 3; 2 4 6; 1 1 1]
full 3×3 det = 0  → rank < 3
2×2 [1 2; 1 1] det = −1 ≠ 0  → rank ≥ 2
⇒ rank = 2

Handy for small or parameter matrices.

13. Rank by eigenvalues (careful)

For diagonal / triangular / symmetric / diagonalizable matrices, rank = number of non-zero eigenvalues.

D = diag(3, 0, 5)  → non-zero: 3, 5 → rank 2

Caution: for defective (non-diagonalizable) matrices this can fail — algebraic multiplicity matters. GATE usually gives the friendly cases.

14. Null space

The null space is every input that the matrix sends to zero:

Null(A) = { x ∈ ℝⁿ : Ax = 0 }

🎮 Imagine: the portal's "disappearing directions" — non-zero vectors that come out as 0 are destroyed directions.

15. Nullity

Nullity = dimension of the null space = number of independent directions the matrix destroys.

nullity 0  → destroys no non-zero direction
nullity 1  → destroys one independent direction
nullity 2  → destroys two
16. The Rank–Nullity Theorem 
rank(A) + nullity(A) = n        (n = number of COLUMNS)

The input x has n components, and the input space splits into surviving directions (rank) + destroyed directions (nullity).

Memory: columns = input variables, so rank + nullity = columns (not rows!).

17. Rank–nullity — game imagination 
3D input (n = 3):
rank 2 → nullity 1   (2 survive, 1 crushed)
rank 1 → nullity 2   (1 survives, 2 crushed)
18. Example: rank and nullity 
A = [1 0; 0 0]  maps [x,y] → [x,0]

x survives → rank = 1
n = 2 columns → nullity = 2 − 1 = 1
destroyed direction: [0,1]  (since A[0,1]ᵀ = 0)
Null(A) = span{[0,1]}
19. Solving for a null space 
A = [1  2]   (1×2)
Ax = 0  ⇒  x₁ + 2x₂ = 0  ⇒  x₁ = −2x₂

let x₂ = t:  x = t·[−2, 1]
Null(A) = span{[−2, 1]},  nullity = 1
rank = 1  ✓  (rank + nullity = 1 + 1 = 2 = columns)
20. Free variables = nullity

When solving Ax = 0, the number of free variables = nullity, and the number of pivot variables = rank.

pivot variables + free variables = total variables
        rank      +    nullity     =    columns
21. Pivot variable vs free variable 
x₁ + 2x₂ + 3x₃ = 0
      x₂ + 4x₃ = 0

pivots: x₁, x₂   → rank = 2
free:   x₃       → nullity = 1
variables = 3    → 2 + 1 = 3  ✓
22. Rank of the transpose 
rank(A) = rank(Aᵀ)     (row rank = column rank)

If A is 3×5 with rank 3, then Aᵀ (5×3) also has rank 3.

23. Rank of a product 
rank(AB) ≤ min(rank(A), rank(B))

🎮 Imagine: if one portal already crushes 3D to a line, a later portal cannot recover the lost information.

rank(A)=2, rank(B)=5 → rank(AB) ≤ 2
24. Rank under invertible multiplication

If P and Q are invertible, rank(PAQ) = rank(A) — invertible maps only rotate/relabel, they destroy no information.

This is why elementary row operations preserve rank (they are multiplication by invertible elementary matrices).

25. rank(AᵀA) = rank(A) 
rank(AᵀA) = rank(AAᵀ) = rank(A)

Key in least squares, projections, PCA and the normal equations.

26. The four fundamental subspaces

For an m×n matrix A:

  • Column space C(A) — span of columns; all outputs Ax; lives in ℝᵐ; dim = rank.
  • Row space C(Aᵀ) — span of rows; lives in ℝⁿ; dim = rank.
  • Null space N(A) — all x with Ax = 0; lives in ℝⁿ; dim = n − rank.
  • Left null space N(Aᵀ) — all y with Aᵀy = 0; lives in ℝᵐ; dim = m − rank.
27. Four subspaces — table
SubspaceDefinitionLives inDimension
Column space C(A)span of columnsℝᵐrank
Row space C(Aᵀ)span of rowsℝⁿrank
Null space N(A)Ax = 0ℝⁿn − rank
Left null space N(Aᵀ)Aᵀy = 0ℝᵐm − rank
28. Orthogonality of the subspaces 
Row space    ⊥  Null space        (both in ℝⁿ)
Column space ⊥  Left null space   (both in ℝᵐ)

Why? If Ax = 0 then every row · x = 0, so x ⊥ every row ⇒ x ⊥ row space.

Trap: do NOT say "column space ⊥ null space" — for m ≠ n they live in different spaces.

29. Worked: rank, nullity & null space (2×3) 
A = [1 2 3; 2 4 6]   (R2 = 2R1)  → rank = 1
n = 3 columns → nullity = 2

x₁ + 2x₂ + 3x₃ = 0 ⇒ x₁ = −2x₂ − 3x₃
Null(A) = span{ [−2,1,0], [−3,0,1] }   → 2 basis vectors
rank + nullity = 1 + 2 = 3  ✓
30. Rank & nullity for square matrices 
n×n:  rank + nullity = n

rank = n → nullity 0, det ≠ 0, invertible, Ax=0 only x=0
rank < n → nullity > 0, det = 0, singular, Ax=0 has non-zero solutions
3×3 with rank 2 → nullity = 1 (one free variable)

So Ax = 0 has a non-trivial solution ⇔ nullity > 0 ⇔ (square) det(A) = 0.

31. Rank in data & machine learning

For a data matrix (rows = samples, columns = features), rank = number of truly independent features.

features: height, weight, BMI, age
BMI depends on height & weight → that column is redundant → dependent columns

Low rank ⇒ redundancy / correlated features ⇒ dimensionality can be reduced — this is the basis of PCA, SVD, feature selection.

32. Rank in linear regression 
Xβ = y   (X = feature matrix)

If columns of X are dependent, rank(X) < #features → multicollinearity: the parameter solution is non-unique and the model is unstable. Rank tells you whether features are genuinely independent.

33. GATE shortcuts
  • rank = non-zero rows in echelon form = number of pivots.
  • rank ≤ min(m, n);   rank(A) = rank(Aᵀ).
  • rank(AB) ≤ min(rank A, rank B).
  • rank + nullity = n (columns).
  • Square: det ≠ 0 ⇔ rank = n ⇔ nullity = 0.
  • Ax=0 has a non-trivial solution ⇔ nullity > 0.
  • Row space ⊥ null space; column space ⊥ left null space.
34. Common GATE traps
  • rank can exceed the number of rows? No — rank ≤ min(m,n).
  • rank + nullity = number of rows? No — it equals the number of columns.
  • det = 0 ⇒ rank = 0? No — only rank < n (a 3×3 could be rank 1 or 2).
  • rank(AB) = rank(A)·rank(B)? No — it is ≤ min.
  • nullity = number of zero rows? No — nullity = free variables = columns − rank.
  • column space ⊥ null space (always)? No — row space ⊥ null space.

36. Cheatsheet
  • Rank = independent rows/cols = pivots = dim of column space = dim of row space.
  • Null space = all x with Ax = 0; nullity = its dimension = free variables.
  • rank + nullity = n;   rank ≤ min(m,n);   rank(A)=rank(Aᵀ).
  • rank(AB) ≤ min(rank A, rank B); invertible P,Q ⇒ rank(PAQ)=rank(A).
  • Square: det ≠ 0 ⇔ rank n ⇔ nullity 0.
  • Row space ⊥ null space; column space ⊥ left null space.
37. Best mental model
  • Input space = all directions before the matrix.
  • Rank = directions that survive; nullity = directions crushed to zero.
  • rank + nullity = total input directions (columns).

6 Main formula

Rank–nullity theorem — independent directions plus collapsed directions = total columns:

$$ \operatorname{rank}(A)+\operatorname{nullity}(A)=n \quad (n=\text{number of columns}) $$

Rank bounds — row rank = column rank, and rank can't exceed the smaller dimension:

$$ \operatorname{rank}(A)=\operatorname{rank}(A^{\mathsf T}),\qquad \operatorname{rank}(A)\le\min(m,n) $$

Solving $Ax=b$ (rank test) — with $r=\operatorname{rank}(A)$ and $r'=\operatorname{rank}([A\,|\,b])$:

$$ r\neq r'\Rightarrow\text{no solution},\quad r=r'=n\Rightarrow\text{unique},\quad r=r'<n\Rightarrow\infty\ (n-r\text{ free}) $$
7 Step-by-step: find rank & nullity
  1. Row-reduce $A$ to row-echelon form.
  2. Count the non-zero rows → that's the rank.
  3. Nullity $=n-\operatorname{rank}$ where $n$ is the number of columns.
8 Worked example
✦ Find the rank

$A=\begin{bmatrix}1&2&3\\2&4&6\\1&1&1\end{bmatrix}$. Row 2 $=2\times$ Row 1, so it vanishes.

Row-reduce → two non-zero rows remain ⇒ $\operatorname{rank}(A)=2$, and nullity $=3-2=1$.

✦ Find the null space (solve $Ax=0$)

$A=\begin{bmatrix}1&2&3\\2&4&6\end{bmatrix}$. Both rows give the same equation $x_1+2x_2+3x_3=0$, so $\operatorname{rank}=1$.

Let the free variables be $x_2=s,\ x_3=t$. Then $x_1=-2s-3t$, so

$x=s\begin{bmatrix}-2\\1\\0\end{bmatrix}+t\begin{bmatrix}-3\\0\\1\end{bmatrix}$. Two basis vectors ⇒ nullity $=2$; check $1+2=3=n$ ✓.

9 GATE cheat sheet
⚡ High-value facts
  • $\operatorname{rank}(A)+\operatorname{nullity}(A)=n$ (columns). Memorize this above all.
  • rank = number of pivots = pivot variables; nullity = number of free variables.
  • $\operatorname{rank}(A)=$ order of the largest square submatrix with non-zero determinant (handy for matrices with parameters).
  • $\operatorname{rank}(A)=\operatorname{rank}(A^{\mathsf T})$; rank = number of non-zero eigenvalues only for diagonalizable cases — safer: use echelon form.
  • $\operatorname{rank}(AB)\le\min(\operatorname{rank}A,\operatorname{rank}B)$; multiplying by invertible $P,Q$ keeps it: $\operatorname{rank}(PAQ)=\operatorname{rank}(A)$ (row operations preserve rank).
  • $\operatorname{rank}(A^{\mathsf T}A)=\operatorname{rank}(AA^{\mathsf T})=\operatorname{rank}(A)$ — used in least squares, projections, PCA.
10 Common traps
✕ Watch out

Trap: "Nullity $=m-\operatorname{rank}$." Reality: it's $n-\operatorname{rank}$ — use the number of columns.

Trap: "Rank can exceed $\min(m,n)$." Reality: never.

Trap: "$\det(A)=0$ means $\operatorname{rank}=0$." Reality: it means rank $<n$; a $3\times3$ singular matrix can have rank 1 or 2.

Trap: "$\operatorname{rank}(AB)=\operatorname{rank}A\cdot\operatorname{rank}B$ or $\operatorname{rank}(A+B)=\operatorname{rank}A+\operatorname{rank}B$." Reality: both false — only inequalities hold.

Trap: "Nullity = number of zero rows." Reality: nullity = free variables = columns $-$ rank.

11 Connections with other topics
🔑 The invertibility chain (GATE gold)

For a square $A$ ($n\times n$):

$\operatorname{rank}(A)=n \iff \det(A)\neq0 \iff A^{-1}$ exists $\iff$ columns independent $\iff$ nullity $=0 \iff Ax=0$ only at $x=0$.

  • Vectors: rank = independent directions (this is the independence idea applied).
  • Determinant: square $\det=0 \iff$ rank $<n$.
  • Linear equations: compare $\operatorname{rank}(A)$ with $\operatorname{rank}([A|b])$ to count solutions.
  • Four fundamental subspaces: column/row spaces have dim = rank; null spaces use nullity.
SubspaceDefinitionDimension
Column space $C(A)$span of columns$\operatorname{rank}$
Null space $N(A)$$\{x:Ax=0\}$$n-\operatorname{rank}$
Row space $C(A^{\mathsf T})$span of rows$\operatorname{rank}$
Left null space $N(A^{\mathsf T})$$\{y:A^{\mathsf T}y=0\}$$m-\operatorname{rank}$
🔑 Orthogonality of the four subspaces

Row space $\perp$ Null space (both live in $\mathbb{R}^n$).   Column space $\perp$ Left null space (both in $\mathbb{R}^m$).

Why: if $Ax=0$, then every row dotted with $x$ is $0$, so $x$ is perpendicular to the row space.

  • Data science: rank = number of truly independent features. Low rank ⇒ redundancy / multicollinearity; in regression $X\beta=y$ a rank-deficient $X$ gives non-unique, unstable parameters — the motivation for PCA and feature selection.
12 Practice problems

Q1. A $5\times8$ matrix has rank 3. Find its nullity.

Nullity $=n-\operatorname{rank}=8-3=5$ ($n$ = columns = 8).

Q2. A $4\times4$ matrix has rank 4. Is it invertible? What is its nullity?

Yes, invertible (full rank); nullity $=0$.

Q3. What is the maximum possible rank of a $3\times7$ matrix?

$\min(3,7)=3$.

Q4. $A$ is $4\times7$ with rank 3. Find the nullity.

$7-3=4$ (use columns).

Q5. $A$ is $5\times3$ with rank 3. Find the nullity, and what does it say about $Ax=0$?

$3-3=0$; so $Ax=0$ has only the trivial solution $x=0$.

Q6. $A$ is $3\times3$ with $\det(A)=0$. What can you say (and not say) about its rank?

Rank $<3$ (singular, dependent columns, $Ax=0$ has non-trivial solutions). You can't pin the exact rank — it could be 0, 1, or 2 without more info.

Quick MCQ

A $5\times8$ matrix $A$ has rank 3. What is its nullity?
Nullity $=8-3=5$ (use columns).
That's the rank, not the nullity.
✅ Correct — $n-\operatorname{rank}=8-3=5$.
8 is the column count, not the nullity.
13 Final memory box
🧠 Remember
  • Rank = independent directions kept; nullity = crushed directions.
  • $\operatorname{rank}+\operatorname{nullity}=n$ (columns).
  • Rank = non-zero rows in echelon form; $\operatorname{rank}(A)=\operatorname{rank}(A^{\mathsf T})\le\min(m,n)$.
  • Square invertible $\iff$ full rank $\iff \det\neq0 \iff$ nullity 0.

GATE takeaway — rank + nullity = number of columns; this is the identity most rank questions hinge on.
