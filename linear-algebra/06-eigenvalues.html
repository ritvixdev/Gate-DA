Eigenvalues &
Eigenvectors

The most important topic in GATE DA linear algebra. Eigenvalues reveal the "natural directions" of a matrix — the ones it only stretches, never rotates.

2 Why this topic exists

Most directions get knocked sideways when a matrix acts on them. A rare few only get longer or shorter, staying on their own line. Those special directions (eigenvectors) and their stretch factors (eigenvalues) capture the true behaviour of the transformation — and power diagonalization, stability analysis, and PCA.

4 Basic definitions
  • Eigenvector: a non-zero vector $x$ with $Ax=\lambda x$ — it keeps its direction.
  • Eigenvalue $\lambda$: the stretch factor for that eigenvector.
  • Characteristic equation: $\det(A-\lambda I)=0$ — the equation whose roots are the eigenvalues.
  • Eigenspace: all eigenvectors for a given $\lambda$ (plus zero) — a subspace.
📖 Study in Depth — eigenvalues & eigenvectors from zero, one idea per card
1. The core idea

A matrix usually changes a vector's direction. But a few special vectors come out on the same line — only their length changes. Those are eigenvectors; the scaling factor is the eigenvalue.

A x = λ x        (A = matrix, x = eigenvector, λ = eigenvalue)

Important: x ≠ 0 — the eigenvector is never the zero vector.

2. What is λ?

λ ("lambda") is the eigenvalue = the scaling factor along the eigenvector's direction.

λ = 2    doubles        λ = 1    unchanged
λ = 3    triples        λ = 0    collapses to zero
λ = 0.5  halves         λ = -1   flips direction
3. Game imagination

A matrix is a transformation portal. Shoot arrows in: most come out turned. But some special arrows come out on the same line (longer, shorter, or reversed).

eigenvector = special direction
eigenvalue  = power multiplier in that direction
4. Simple visual example 
A = [2  0      maps [x; y] → [2x; 3y]
     0  3]

A·[1;0] = [2;0] = 2·[1;0]   → eigenvector [1;0], λ = 2
A·[0;1] = [0;3] = 3·[0;1]   → eigenvector [0;1], λ = 3

The x-axis stretches by 2, the y-axis by 3.

5. Why eigenvectors matter

Eigenvectors reveal a matrix's natural directions — the ones it only stretches/shrinks. They power PCA, dimensionality reduction, covariance analysis, Markov chains, stability analysis, SVD, spectral clustering, and connect to trace, determinant, rank, invertibility and diagonalization.

6. The eigenvector cannot be zero

Ax = λx technically holds for x = 0 (both sides 0), but the zero vector has no direction.

GATE point: an eigenvector must be non-zero (eigenvalue 0 is allowed; eigenvector 0 is not).

7. How to find eigenvalues 
Ax = λx
Ax − λx = 0   →   Ax − λIx = 0   (since Ix = x)
(A − λI)x = 0

This homogeneous system has a non-zero solution only if the matrix is singular:

det(A − λI) = 0      ← the characteristic equation
8. The step-by-step method
  1. Form A − λI.
  2. Compute det(A − λI).
  3. Set it to 0 and solve for λ (the eigenvalues).
  4. For each λ, solve (A − λI)x = 0.
  5. The non-zero solutions x are the eigenvectors.

Defined only for square matrices.

9. Worked example — eigenvalues 
A = [4  1
     2  3]

A − λI = [4−λ   1
           2   3−λ]

det = (4−λ)(3−λ) − 2 = λ² − 7λ + 10 = 0
(λ − 5)(λ − 2) = 0   →   λ = 5, 2
10. Worked example — eigenvector for λ = 5 
A − 5I = [−1   1
           2  −2]

−x₁ + x₂ = 0  →  x₂ = x₁
x = t·[1; 1]   →  eigenvector [1; 1]

Every non-zero multiple ([2;2], [−5;−5], …) is the same eigen-direction.

11. Worked example — eigenvector for λ = 2 
A − 2I = [2  1
          2  1]

2x₁ + x₂ = 0  →  x₂ = −2x₁
x = t·[1; −2]   →  eigenvector [1; −2]

Result: direction [1;1] stretches ×5; direction [1;−2] stretches ×2.

12. Check with trace & determinant 
A = [4 1; 2 3]:  tr = 4+3 = 7,  det = 12−2 = 10
eigenvalues 5, 2:  5+2 = 7 ✓ (= trace),  5×2 = 10 ✓ (= det)

Shortcut: sum of eigenvalues = trace; product = determinant.

13. The characteristic polynomial (2×2 fast form) 
A = [a b; c d]
λ² − tr(A)·λ + det(A) = 0
A = [4 1; 2 3]: λ² − 7λ + 10 = 0  →  λ = 5, 2

Much faster than expanding the determinant for 2×2 matrices.

14. Triangular / diagonal eigenvalues (shortcut)

For diagonal, upper- or lower-triangular matrices, the eigenvalues are simply the diagonal entries — no work needed.

[2 5 7; 0 3 8; 0 0 4]  → eigenvalues 2, 3, 4
diag(5, −1, 7)         → eigenvalues 5, −1, 7
15. Trace / determinant & the missing eigenvalue 
tr(A) = λ₁ + λ₂ + … + λₙ ;   det(A) = λ₁·λ₂·…·λₙ

Find a missing eigenvalue:

3×3, trace 9, eigenvalues 2, 3, λ₃  →  2+3+λ₃ = 9 → λ₃ = 4
3×3, det 24,  eigenvalues 2, 4, λ₃  →  2·4·λ₃ = 24 → λ₃ = 3
16. Zero eigenvalue ⟺ singular 
0 is an eigenvalue of A  ⟺  det(A) = 0  ⟺  A is singular

Because det = product of eigenvalues: one zero eigenvalue forces det 0 → not invertible, rank < n, Ax=0 has non-trivial solutions.

17. How eigenvalues transform (key rules)

If Ax = λx, then for the same eigenvector x:

  • Aᵏ has eigenvalue λᵏ  (A² → λ², A³ → λ³).
  • A⁻¹ has eigenvalue 1/λ (if A is invertible, so λ ≠ 0).
  • A + cI has eigenvalue λ + c.
  • cA has eigenvalue cλ.
  • Aᵀ has the same eigenvalues as A (eigenvectors may differ).
18. Eigenvalues of basic special matrices 
Identity I   →  every eigenvalue = 1
Zero O       →  every eigenvalue = 0
Diagonal     →  the diagonal entries
Triangular   →  the diagonal entries
19. Symmetric matrices (very friendly)

If A = Aᵀ (real symmetric):

  • all eigenvalues are real;
  • eigenvectors for distinct eigenvalues are orthogonal;
  • the matrix is always diagonalizable.

Covariance matrices are symmetric — this is the foundation of PCA.

20. Orthogonal matrices: |λ| = 1

If AᵀA = I, length is preserved, so ||Ax|| = ||x||, giving |λ| = 1.

real orthogonal eigenvalues → +1 or −1
(complex ones lie on the unit circle)
21. Idempotent & nilpotent eigenvalues GATE extra 
idempotent (A² = A):  λ² = λ  →  λ = 0 or 1
nilpotent  (Aᵏ = 0):  λᵏ = 0  →  λ = 0 (all of them)
22. Complex eigenvalues

A real matrix can have complex eigenvalues. The 90° rotation has no real eigen-direction:

R = [0  −1
     1   0]   →  eigenvalues  i  and  −i

Complex eigenvalues of real matrices come in conjugate pairs a ± bi.

23. Eigenvalues in PCA (data meaning)

A covariance matrix is symmetric; its eigenvectors = principal directions and eigenvalues = variance along them.

λ₁ = 100, λ₂ = 5  →  data spreads far along direction 1, little along 2

PCA keeps the top eigenvectors. (Full treatment in the SVD & PCA topic.)

24. Eigenvalues & stability

Repeatedly applying A (x, Ax, A²x, …) — eigenvalues govern long-term behaviour:

|λ| < 1  → that direction decays
|λ| > 1  → it grows
λ < 0    → it flips sign each step

Used in dynamical systems, Markov chains and iterative algorithms.

25. Projection matrices

Projection matrices are idempotent (P² = P), so their eigenvalues are 0 or 1.

P = [1 0; 0 0]:  P[1;0]=[1;0] (λ=1, kept),  P[0;1]=[0;0] (λ=0, destroyed)
26. Quadratic & bilinear forms

A quadratic form packs a degree-2 expression into a symmetric matrix:

Q(x) = xᵀA x = Σ aᵢⱼ xᵢ xⱼ     (A symmetric)
e.g. 2x² + 3y² + 4xy  →  A = [2 2; 2 3]   (diagonal = squares, off-diag = half the cross term)

A bilinear form is xᵀA y. The matrix of a quadratic form is always taken symmetric.

27. Definiteness (positive/negative definite)

Classify a symmetric matrix A by the sign of xᵀAx — read straight off its eigenvalues:

positive definite (PD):      xᵀAx > 0 ∀ x≠0   ⟺  all eigenvalues > 0
positive semidefinite (PSD): xᵀAx ≥ 0          ⟺  all eigenvalues ≥ 0
negative definite:           xᵀAx < 0          ⟺  all eigenvalues < 0
indefinite:                  mixed signs       ⟺  eigenvalues of both signs

Sylvester's criterion: A is PD ⟺ every leading principal minor is > 0.

Why it matters: covariance matrices are PSD; the Hessian's definiteness decides minima/maxima in optimization.

28. Common GATE traps
  • Zero vector is an eigenvector? No — eigenvectors are non-zero.
  • Every real matrix has real eigenvalues? No — only real symmetric ones are guaranteed real.
  • Eigenvalues of A+B = eigenvalues of A + eigenvalues of B? No (true only for A + cI).
  • det(A)=0 ⇒ 0 is an eigenvalue? True.
  • Triangular ⇒ eigenvalues are diagonal entries? True.
  • Idempotent ⇒ λ ∈ {0,1}; nilpotent ⇒ all λ = 0; symmetric ⇒ real λ & orthogonal eigenvectors? All true.

30. Cheatsheet
  • Ax = λx, x ≠ 0; eigenvalues from det(A − λI) = 0; eigenvectors from (A − λI)x = 0.
  • 2×2: λ² − tr·λ + det = 0;   sum λ = trace, product λ = det.
  • Triangular/diagonal → diagonal entries; 0 is an eigenvalue ⟺ det = 0 ⟺ singular.
  • Aᵏ→λᵏ, A⁻¹→1/λ, A+cI→λ+c, cA→cλ, Aᵀ same λ.
  • Symmetric → real λ, orthogonal eigenvectors; orthogonal → |λ|=1; idempotent → 0/1; nilpotent → 0.
31. Best mental model 
Matrix = transformation machine
Eigenvector = special direction that stays on its own line
Eigenvalue  = the multiplier on that direction
λ>1 stretch · 0<λ<1 shrink · λ=1 unchanged · λ=0 collapsed · λ<0 flipped

And remember: trace = Σλ, determinant = Πλ, and a zero eigenvalue means the matrix is singular.

6 Main formulas
$$ Ax=\lambda x,\qquad x\neq 0 $$
$$ \det(A-\lambda I)=0 \ \ (\text{eigenvalues}),\qquad (A-\lambda I)x=0 \ \ (\text{eigenvectors}) $$
7 Step-by-step method
  1. Form $A-\lambda I$ (subtract $\lambda$ from the diagonal).
  2. Compute $\det(A-\lambda I)$ → a polynomial in $\lambda$.
  3. Set it to $0$ and solve → the eigenvalues.
  4. For each $\lambda$, solve $(A-\lambda I)x=0$ → the eigenvectors.
  5. Non-zero solutions are the eigenvectors.
8 Worked example
✦ Full example

$A=\begin{bmatrix}4&1\\2&3\end{bmatrix}$:   $\det(A-\lambda I)=(4-\lambda)(3-\lambda)-2=\lambda^2-7\lambda+10=0$.

$(\lambda-5)(\lambda-2)=0 \Rightarrow \lambda_1=5,\ \lambda_2=2$.

$\lambda=5$: $(A-5I)x=0\Rightarrow x=(1,1)^{\mathsf T}$.   $\lambda=2$: $x=(1,-2)^{\mathsf T}$.

9 GATE cheat sheet
⚡ GATE gold
  • $\sum\lambda_i=\operatorname{tr}(A)$ and $\prod\lambda_i=\det(A)$ — fast checks.
  • Triangular/diagonal matrix → eigenvalues are the diagonal entries.
  • $A^k$ has $\lambda^k$; $A^{-1}$ has $1/\lambda$; $A+cI$ has $\lambda+c$; $A^{\mathsf T}$ has the same $\lambda$.
  • Symmetric matrix → all real eigenvalues, orthogonal eigenvectors.
  • If $0$ is an eigenvalue, $A$ is singular ($\det=0$).
10 Common traps
✕ Watch out

Trap: "The zero vector is an eigenvector." Reality: eigenvectors must be non-zero ($\lambda$ can be $0$, but $x$ can't).

Trap: "Real matrices have real eigenvalues." Reality: they can be complex (e.g. a rotation).

Trap: "Eigenvalues of $A+B$ are eigenvalues of $A$ plus those of $B$." Reality: false in general.

11 Connections with other topics
  • Determinant: $\det A=\prod\lambda$. Trace: $\operatorname{tr}A=\sum\lambda$.
  • Rank: a zero eigenvalue means the matrix is rank-deficient.
  • Diagonalization: needs enough independent eigenvectors.
  • PCA: principal components are eigenvectors of the covariance matrix.
12 Practice problems

Q1. Eigenvalues are $2,3,5$. Find the trace and determinant.

Trace $=2+3+5=10$; determinant $=2\cdot3\cdot5=30$.

Q2. $A$ is triangular with diagonal $1,4,7$. Eigenvalues?

$1,4,7$ (the diagonal entries).

Q3. Eigenvalues of $A$ are $2,3$. Eigenvalues of $A^{-1}$?

$\tfrac12,\tfrac13$ (reciprocals).

Quick MCQ

If the eigenvalues of $A$ are $2$ and $3$, the eigenvalues of $A^{-1}$ are:
Inversion gives reciprocals, not negatives.
✅ Correct — $A^{-1}$ has eigenvalues $1/\lambda$.
$4,9$ are eigenvalues of $A^2$.
$A^{-1}$ flips each $\lambda$ to its reciprocal.
13 Final memory box
🧠 Remember
  • Eigenvector = direction that doesn't rotate; eigenvalue = stretch factor.
  • $Ax=\lambda x$; find $\lambda$ from $\det(A-\lambda I)=0$.
  • Trace $=\sum\lambda$, determinant $=\prod\lambda$.
  • $A^k\to\lambda^k$, $A^{-1}\to1/\lambda$; triangular → diagonal entries.
  • Symmetric matrices are safest: real $\lambda$, orthogonal eigenvectors.

GATE takeaway — Σλ = trace, Πλ = det, and Aᵏ has eigenvalues λᵏ — these appear almost every year.
