Matrices

8 questions. Tap any option — you'll see why it's right or wrong, and the correct one lights up.

Q1. $A$ is $4\times3$, $B$ is $3\times5$. Size of $AB$?
The inner $3$'s match (good), but the answer uses the outer numbers.
✅ Correct — $(4\times3)(3\times5)=4\times5$ (outer numbers).
Right size but wrong order — rows of $A$ come first.
It's defined — the inner dimensions $3$ and $3$ match.
Q2. Can a $2\times3$ matrix multiply a $2\times2$ matrix (in that order)?
No — the product isn't defined here.
✅ Correct — inner dimensions $3$ and $2$ don't match, so $(2\times3)(2\times2)$ is undefined.
Undefined — you can't even form it.
Squareness isn't the rule; matching inner dimensions is.
Q3. $(AB)^{\mathsf T}=?$
Common trap — the order must reverse.
Transpose changes the matrix unless it's symmetric.
✅ Correct — transpose reverses order: $(AB)^{\mathsf T}=B^{\mathsf T}A^{\mathsf T}$.
Both factors transpose, and the order flips.
Q4. $(AB)^{-1}=?$ (both invertible)
Order must reverse (socks-then-shoes).
Both factors invert and the order flips.
✅ Correct — $(AB)^{-1}=B^{-1}A^{-1}$.
Not equal in general since $AB\neq BA$.
Q5. $\operatorname{tr}\begin{bmatrix}2&9\\4&5\end{bmatrix}=?$
$20$ doesn't come from this matrix — trace is just the diagonal sum $2+5$.
$13=9+4$ is the off-diagonal sum; trace uses the diagonal.
✅ Correct — trace $=2+5=7$ (sum of the diagonal).
Not the trace — add the diagonal entries: $2+5=7$.
Q6. If $A^2=A$ (idempotent), its eigenvalues are…
Only if $A=I$; in general a projection has $0$'s too.
✅ Correct — $\lambda^2=\lambda\Rightarrow\lambda(\lambda-1)=0\Rightarrow\lambda\in\{0,1\}$.
Only if $A=0$; projections also have eigenvalue $1$.
The relation $\lambda^2=\lambda$ forces $0$ or $1$.
Q7. For an orthogonal matrix $Q$, $Q^{-1}=?$
Only if $Q^2=I$; in general it's the transpose.
No — orthogonal means $Q^{\mathsf T}Q=I$.
✅ Correct — $Q^{\mathsf T}Q=I$ means $Q^{-1}=Q^{\mathsf T}$ (and $\det=\pm1$).
It always exists — the inverse is just the transpose.
Q8. Which statement is always true?
False in general — matrices don't commute.
False — it's $A^2+AB+BA+B^2$, and $AB\neq BA$.
✅ Correct — the trace is cyclic: $\operatorname{tr}(AB)=\operatorname{tr}(BA)$ always.
False — e.g. a reflection has $A^2=I$ but $A\neq\pm I$.
