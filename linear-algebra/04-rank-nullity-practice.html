Rank & Nullity

8 questions. Tap any option — you'll see why it's right or wrong, and the correct one lights up.

Q1. A $5\times8$ matrix has rank $3$. Nullity?
Used rows ($5-3$)? Nullity uses columns.
That's the rank, not the nullity.
✅ Correct — nullity $=n-\operatorname{rank}=8-3=5$.
$8$ is the column count, not the nullity.
Q2. $\operatorname{rank}(A)+\operatorname{nullity}(A)=?$
A classic trap — it's columns, because null space is about the input $x$.
✅ Correct — rank $+$ nullity $=n$ (columns).
That bounds the rank alone, not the sum.
Determinant isn't even defined for non-square matrices.
Q3. Maximum possible rank of a $6\times4$ matrix?
Rank can't exceed the smaller dimension.
✅ Correct — $\operatorname{rank}\le\min(6,4)=4$.
Rank is bounded by $\min(m,n)$, not $m+n$.
That's the entry count, unrelated to rank.
Q4. $A$ is $3\times3$ with $\det A=0$. Which is certain?
Not necessarily $0$ — could be $1$ or $2$.
$\det=0$ means it's not full rank.
✅ Correct — singular ⇒ rank $<n$; exact value (0,1,2) needs more info.
Could be $2$, but also $0$ or $1$ — not certain.
Q5. Nullity equals…
Zero rows relate to rank, not directly nullity.
✅ Correct — nullity = free variables = columns $-$ rank.
Pivots count the rank, not the nullity.
It's columns $-$ rank, not rows $-$ rank.
Q6. $\operatorname{rank}(A^{\mathsf T})$ compared with $\operatorname{rank}(A)$?
✅ Correct — row rank $=$ column rank, so $\operatorname{rank}(A^{\mathsf T})=\operatorname{rank}(A)$.
They're exactly equal.
They're exactly equal.
They're tightly linked — in fact identical.
Q7. $\operatorname{rank}(AB)$ satisfies…
False — rank isn't multiplicative.
✅ Correct — a later map can't recover lost directions: $\operatorname{rank}(AB)\le\min(\operatorname{rank}A,\operatorname{rank}B)$.
False — it's bounded above by the smaller rank.
Backwards — multiplication can only lose rank.
Q8. Which subspaces are orthogonal?
Trap — they may live in different spaces ($\mathbb{R}^m$ vs $\mathbb{R}^n$).
These aren't the orthogonal pair.
✅ Correct — Row space ⊥ Null space (both in $\mathbb{R}^n$); also Column ⊥ Left-null.
Two orthogonal pairs do exist.
