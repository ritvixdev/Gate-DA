Topic 5 · GATE weight: Very High

System of
Linear Equations

When does $Ax=b$ have a solution, and how many? This one question is the heart of applied linear algebra — and a near-certain GATE appearance.

2 Why this topic exists

Most real problems become "solve $Ax=b$": balancing equations, fitting data, circuits, networks. The key skill is not just solving, but knowing in advance whether a solution exists, is unique, or comes in infinite families — decided entirely by ranks.

3 Game / visual imagination
🎮 Imagine it

Each equation is a line (2-D) or plane (3-D). A solution is where they all cross. Lines crossing at one point → unique. Same line twice → infinite (they overlap). Parallel lines → no solution (never meet). Rank just counts how many genuinely different equations you have.

4 Basic definitions

System: $Ax=b$, where $A$ is the coefficient matrix, $x$ the unknowns, $b$ the right side.

Augmented matrix $[A\,|\,b]$: $A$ with the $b$ column attached.

Consistent: at least one solution exists. Inconsistent: none.

Homogeneous: $b=0$ (i.e. $Ax=0$). Free variable: an unknown not pinned down — each one gives an infinite family.

5 Symbols & their meaning
SymbolRead asMeaning
$Ax=b$"A x equals b"the system
$[A\,|\,b]$"augmented matrix"coefficients + right side
$r=\operatorname{rank}(A)$"rank of A"independent equations
$r'=\operatorname{rank}([A|b])$"rank of augmented"with right side included
$n$"n"number of unknowns
6 Main rule (the decider)
$$ r \neq r' \;\Rightarrow\; \textbf{no solution} $$
$$ r = r' = n \;\Rightarrow\; \textbf{unique solution} $$
$$ r = r' < n \;\Rightarrow\; \boldsymbol{\infty}\textbf{ solutions} \ \ (n-r \text{ free variables}) $$
7 Step-by-step: Gaussian elimination
  1. Write the augmented matrix $[A\,|\,b]$.
  2. Use row operations (swap, scale, add multiples) to reach row-echelon form.
  3. Read off $r=\operatorname{rank}(A)$ and $r'=\operatorname{rank}([A|b])$.
  4. Apply the decider rule to classify no / unique / infinite.
  5. Back-substitute to get the actual solution (parameterize free variables if infinite).
8 Worked example
✦ Solve $x+2y=5,\ 2x+4y=10$

$\left[\begin{array}{cc|c}1&2&5\\2&4&10\end{array}\right]\xrightarrow{R_2-2R_1}\left[\begin{array}{cc|c}1&2&5\\0&0&0\end{array}\right]$

$r=r'=1<2$ ⇒ infinitely many solutions. $y$ is free; $x=5-2y$.

9 GATE shortcuts
⚡ High-value facts
  • Homogeneous $Ax=0$ always has the trivial solution $x=0$.
  • $Ax=0$ has non-trivial solutions $\iff \operatorname{rank}(A)<n \iff \det A=0$ (square).
  • Square system has a unique solution for every $b$ $\iff \det A\neq0$.
  • General solution $=$ one particular solution $+$ any null-space vector: $x=x_p+x_h$.
  • Cramer's rule (small square systems, $\det A\neq0$): $x_i=\dfrac{\det(A_i)}{\det(A)}$, where $A_i$ is $A$ with column $i$ replaced by $b$. Clean but costly — use Gaussian elimination for big systems.
10 Common traps
✕ Watch out

Trap: "$\det A=0$ means no solution." Reality: it means not unique — could be none OR infinite; check $r$ vs $r'$.

Trap: "Homogeneous systems can be inconsistent." Reality: never — $x=0$ always works.

Trap: "More equations ⇒ no solution." Reality: only ranks decide, not the count.

11 Connections with other topics
  • Rank: the entire decider rule is built on ranks.
  • Determinant: square unique-solution test is $\det\neq0$.
  • Null space: the $x_h$ part of the solution lives in the null space.
  • Vectors: "$Ax=b$ solvable" means $b$ is in the span of $A$'s columns.
12 Practice problems

Q1. $r=2$, $r'=3$, $n=3$. How many solutions?

$r\neq r'$ ⇒ no solution (inconsistent).

Q2. $r=r'=2$, $n=4$. How many solutions?

$r=r'<n$ ⇒ infinitely many, with $4-2=2$ free variables.

Q3. When does $Ax=0$ (square $A$) have a non-trivial solution?

When $\det A=0$ (equivalently $\operatorname{rank}<n$).

13 Final memory box
🧠 Remember
  • $r\neq r'$ → none; $r=r'=n$ → unique; $r=r'<n$ → infinite ($n-r$ free).
  • $Ax=0$ always has $x=0$; non-trivial iff $\operatorname{rank}<n$ iff $\det=0$.
  • General solution $x=x_p+x_h$ (particular + null space).
  • Solve by Gaussian elimination on $[A\,|\,b]$.

GATE takeaway — compare rank(A), rank([A|b]) and n: it decides no, unique, or infinitely many solutions.
