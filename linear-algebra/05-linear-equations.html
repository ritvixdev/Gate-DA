Topic 5 · GATE weight: Very High

System of
Linear Equations

When does $Ax=b$ have a solution? How many? This is the central problem of applied linear algebra — and a top GATE topic.

⚖️ Types of solutions

Compare the rank of $A$ with the rank of the augmented matrix $[A\,|\,b]$, where $n$ is the number of unknowns:

$$ \operatorname{rank}(A) \neq \operatorname{rank}([A|b]) \;\Rightarrow\; \textbf{no solution} $$ $$ \operatorname{rank}(A) = \operatorname{rank}([A|b]) = n \;\Rightarrow\; \textbf{unique solution} $$ $$ \operatorname{rank}(A) = \operatorname{rank}([A|b]) < n \;\Rightarrow\; \boldsymbol{\infty}\textbf{ solutions} $$
🧠 The golden rule

Let $r=\operatorname{rank}(A)$, $r'=\operatorname{rank}([A|b])$, $n=$ unknowns.

  • $r \neq r'$ → no solution
  • $r=r'=n$ → unique
  • $r=r'
🛠️ Gaussian elimination

Reduce $[A\,|\,b]$ to row-echelon form using elementary row operations: swap rows, scale a row, or add a multiple of one row to another.

✦ Example — solve $x+2y=5,\ 2x+4y=10$

$\left[\begin{array}{cc|c}1&2&5\\2&4&10\end{array}\right] \xrightarrow{R_2-2R_1} \left[\begin{array}{cc|c}1&2&5\\0&0&0\end{array}\right]$

$r=r'=1<2$ → infinitely many solutions. $y$ is free; $x=5-2y$.

🏠 Homogeneous system $Ax=0$

$Ax=0$ always has the trivial solution $x=0$. Non-trivial solutions exist precisely when the columns are dependent:

$$ \text{only trivial} \iff \operatorname{rank}(A)=n \qquad \text{non-trivial} \iff \operatorname{rank}(A)
⚡ GATE trap

“Only the trivial solution” $\iff \operatorname{rank}=n \iff \det\neq0 \iff A$ invertible. All four are equivalent.

🔁 Solution structure
$$ x = x_p + x_h $$

$x_p$ is any one particular solution; $x_h$ is any solution of $Ax=0$ (the null space). The full solution set is a shifted null space.
