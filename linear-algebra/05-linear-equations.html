Topic 5 · GATE weight: Very High

System of
Linear Equations

When does $Ax=b$ have a solution, and how many? This one question is the heart of applied linear algebra — and a near-certain GATE appearance.

2 Why this topic exists

Most real problems become "solve $Ax=b$": balancing equations, fitting data, circuits, networks. The key skill is not just solving, but knowing in advance whether a solution exists, is unique, or comes in infinite families — decided entirely by ranks.

3 Game / visual imagination
🎮 Imagine it

Each equation is a line (2-D) or plane (3-D). A solution is where they all cross. Lines crossing at one point → unique. Same line twice → infinite (they overlap). Parallel lines → no solution (never meet). Rank just counts how many genuinely different equations you have.

4 Basic definitions
  • System: $Ax=b$, where $A$ is the coefficient matrix, $x$ the unknowns, $b$ the right side.
  • Augmented matrix $[A\,|\,b]$: $A$ with the $b$ column attached.
  • Consistent: at least one solution exists. Inconsistent: none.
  • Homogeneous: $b=0$ (i.e. $Ax=0$). Free variable: an unknown not pinned down — each one gives an infinite family.
📖 Study in Depth — solving systems from zero, one idea per card

A system of linear equations means solving many equations at once — written compactly as Ax = b. The whole topic answers two questions: does a solution exist, and how many? Rank, the determinant, and the column/null spaces give the complete answer. Open each card in order.

1. What is a system of linear equations?

A system is several linear equations together — and the question is always: does a common solution exist, and how many?

x + 2y = 5
2x + 4y = 10

In 2D each equation is a line, so solving asks: where do the lines meet?

2. Matrix form: Ax = b

Every linear system packs into one equation:

x + 2y = 5        A = [1  2     x = [x     b = [5
3x + 4y = 6            3  4]         y]         6]

[1 2; 3 4]·[x; y] = [5; 6]   ⇒   Ax = b

A = coefficient matrix, x = unknowns, b = constants.

3. The augmented matrix [A|b]

Attach b to A as an extra column:

[A | b] = [1  2 | 5
           3  4 | 6]

Key names: A = coefficient matrix, [A|b] = augmented matrix. Comparing their ranks decides everything.

4. Game imagination

Each equation is a rule/wall; the solution is the position satisfying all rules at once.

No solution       = rules contradict
Unique solution   = exactly one position fits
Infinite solutions = many positions fit
5. Geometric meaning in 2D 
Unique:   lines cross once     x+y=3 , x−y=1
No soln:  parallel, different  x+y=3 , x+y=5
Infinite: same line            x+2y=5 , 2x+4y=10

The second of each pair shows the give-away: same left side with a different (or proportional) right side.

6. Geometric meaning in 3D

With three unknowns each equation is a plane:

planes meet at a point  → unique
planes meet on a line   → infinite
planes share no common point → no solution

Solving a system is really geometry.

7. The golden rule (rank test) 
r  = rank(A) ,  r' = rank([A|b]) ,  n = number of unknowns

r ≠ r'        → No solution
r = r' = n    → Unique solution
r = r' < n    → Infinitely many solutions

This single rule answers every system question in GATE.

8. Why rank decides the solutions

Rank = number of independent equations; n = number of variables to pin down.

enough independent equations (r = n)  → all variables fixed → unique
too few (r < n)                       → free variables remain → infinite
[A|b] gains rank over A               → a contradiction row    → none
9. What each case looks like

Contradiction (r ≠ r'):

[0 0 | 5]  →  0 = 5  →  impossible  →  no solution

All variables fixed (r = r' = n):

2 independent equations, 2 unknowns → unique

Free variable (r = r' < n):

2 independent equations, 3 unknowns → free variables = n − r = 1 → infinite
10. Gaussian elimination

The main method: row-reduce [A|b] to echelon form using operations that do not change the solution set:

1. swap two rows
2. multiply a row by a non-zero scalar
3. add a multiple of one row to another

🎮 Imagine: the same mission rewritten with simpler instructions.

11. Row echelon form & back substitution 
[1  2  3 | 4
 0  1  5 | 6
 0  0  2 | 8]

Once triangular, solve from the bottom up (back substitution): the last row gives the last variable, then substitute upward.

12. Worked: unique solution 
x + y = 3 , x − y = 1
[1 1 | 3; 1 −1 | 1]  →  R2−R1  →  [1 1 | 3; 0 −2 | −2]

−2y = −2 → y = 1 ;  x + 1 = 3 → x = 2

rank(A)=rank([A|b])=2=n  →  unique:  x=2, y=1
13. Worked: infinitely many solutions 
x + 2y = 5 , 2x + 4y = 10
[1 2 | 5; 2 4 | 10]  →  R2−2R1  →  [1 2 | 5; 0 0 | 0]

rank = rank' = 1 < 2  →  infinite
let y = t:  x = 5 − 2t

[x; y] = [5; 0] + t·[−2; 1],   t ∈ ℝ
14. Worked: no solution 
x + 2y = 5 , 2x + 4y = 11
[1 2 | 5; 2 4 | 11]  →  R2−2R1  →  [1 2 | 5; 0 0 | 1]

last row: 0 = 1  →  impossible
rank(A)=1 ≠ rank([A|b])=2  →  no solution
15. Homogeneous systems Ax = 0

When the right side is all zeros, Ax = 0 always has the trivial solution x = 0.

x + 2y = 0 , 3x + 4y = 0   →   x = 0, y = 0 always works

The real question is whether a non-trivial (non-zero) solution also exists.

16. Trivial vs non-trivial solution 
trivial:     x = 0  (all variables zero)
non-trivial: at least one variable ≠ 0  (e.g. x=2, y=−1)

GATE asks: "does Ax = 0 have a non-trivial solution?" = is there a non-zero x with Ax = 0?

17. Homogeneous golden rule 
Ax = 0 has ONLY the trivial solution  ⟺  rank(A) = n

Because rank + nullity = n: if rank = n then nullity = 0 (no free variable → only x = 0). If rank < n, free variables exist → non-trivial solutions.

18. Homogeneous system for a square matrix

For n×n A, these are all equivalent:

Ax = 0 only trivial  ⟺  rank = n  ⟺  det(A) ≠ 0
                     ⟺  invertible ⟺ columns independent ⟺ nullity 0

And the opposite chain:

Ax = 0 has non-trivial  ⟺  rank < n  ⟺  det(A) = 0  ⟺  singular
19. Non-homogeneous systems (b ≠ 0)

With b ≠ 0, a solution is not guaranteed — Ax = b may have none, one, or infinitely many solutions (unlike the homogeneous case, which always has at least x = 0).

20. Solution structure: x = xₚ + xₕ 
x = xₚ + xₕ
xₚ = one particular solution of Ax = b
xₕ = any solution of Ax = 0  (the null space)

One real solution, plus every "invisible" null-space move that doesn't change the output.

21. Why x = xₚ + xₕ works 
A xₚ = b ,  A xₕ = 0
A(xₚ + xₕ) = Axₚ + Axₕ = b + 0 = b  ✓

So once you have one solution, adding any null-space vector keeps you a solution — the full solution set is a shifted null space.

22. Example of the solution structure 
x + 2y = 5  →  [x; y] = [5; 0] + t·[−2; 1]

xₚ = [5; 0]      (check: 5 + 2·0 = 5 ✓)
xₕ = t·[−2; 1]   (solves x + 2y = 0)

general = particular + homogeneous.

23. Free variables and pivot variables 
x₁ + 2x₂ + 3x₃ = 5
        x₂ + 4x₃ = 6

pivot variables: x₁, x₂   → count = rank
free variable:   x₃       → count = nullity = n − rank

Free variables = n − rank(A); if any exist (and the system is consistent), there are infinitely many solutions.

24. Worked: 3-variable unique solution 
x+y+z=6 , 2x+3y+z=11 , x+2y+3z=14
R2−2R1, R3−R1 → [1 1 1|6; 0 1 −1|−1; 0 1 2|8]
R3−R2          → [1 1 1|6; 0 1 −1|−1; 0 0 3|9]

3z = 9 → z = 3 ;  y − 3 = −1 → y = 2 ;  x + 5 = 6 → x = 1
rank = rank' = 3 = n → unique:  (1, 2, 3)
25. Overdetermined & underdetermined systems 
Overdetermined: more equations than unknowns (A is 5×2)
   → often NO exact solution → least squares (regression)

Underdetermined: fewer equations than unknowns (A is 2×5)
   → if consistent, infinitely many solutions (free variables)
26. More unknowns than equations (key result)

For Ax = 0 with n > m (more unknowns than equations):

rank(A) ≤ m < n  ⇒  rank < n  ⇒  nullity > 0
⇒  a non-trivial solution ALWAYS exists
e.g. 2 equations, 3 unknowns → Ax = 0 has non-zero solutions
27. Square system Ax = b 
A is n×n:
det(A) ≠ 0 → unique solution for EVERY b  (x = A⁻¹b)
det(A) = 0 → not unique: no solution OR infinitely many (depends on b)
              for Ax = 0 specifically → infinitely many
28. Consistent vs inconsistent & row types 
consistent   ⟺ rank(A) = rank([A|b])   (a solution exists)
inconsistent ⟺ rank(A) ≠ rank([A|b])   (no solution)

Watch these rows after reduction:

[0 0 0 | 5]  contradiction → no solution
[0 0 0 | 0]  no new info   → dependency / free variables
[0 1 3 | 4]  pivot row     → useful equation
29. Reduced row echelon form (RREF)

Gauss–Jordan goes one step further than echelon form, making pivots 1 with zeros above and below:

[1 0 | 2
 0 1 | 1]   →   reads off directly:  x = 2, y = 1
30. Cramer's Rule

For a square system with det(A) ≠ 0:

xᵢ = det(Aᵢ) / det(A)     (Aᵢ = A with column i replaced by b)
2x+y=5 , x+3y=7 :  det(A)=5
x = |5 1; 7 3|/5 = 8/5 ,  y = |2 5; 1 7|/5 = 9/5

Great for theory and small systems; for big systems use elimination.

31. Connection with the column space

Ax is a combination of A's columns: Ax = x₁·col₁ + x₂·col₂ + …. So:

Ax = b has a solution  ⟺  b lies in the column space of A

Solving the system = "can b be built from the columns of A?" — the deepest way to see it.

32. Column-space example 
A = [1 2; 2 4]   (col₂ = 2·col₁ → column space is a line)

b = [3; 6] = 3·[1;2]  → in the column space → Ax = b solvable
b = [3; 7]            → not on that line   → no solution

This matches the rank test exactly.

33. Connection with the null space 
if A xₚ = b, every solution is  x = xₚ + z,  where A z = 0 (z in Null(A))

nullity 0  → unique solution
nullity > 0 → infinitely many (move freely along the null space)
34. Number-of-solutions summary
ResultCondition
No solutionb ∉ C(A) ; rank(A) ≠ rank([A|b])
Unique solutionb ∈ C(A) ; rank(A) = n ; nullity 0
Infinitely manyb ∈ C(A) ; rank(A) < n ; nullity > 0
35. Where this shows up in GATE DA / ML

Linear regression is a system: Xβ = y (X = data matrix, β = parameters, y = targets). Usually there are far more rows (data points) than columns (parameters), so an exact solution rarely exists.

Systems of equations lead directly to least squares, projection, normal equations and optimization.

36. Least-squares preview

When Ax = b has no exact solution (b outside the column space), find the x that makes Ax closest to b:

minimize || Ax − b ||²   ⇔   project b onto the column space of A

The full method (projection / normal equations) comes in the Orthogonality and SVD & PCA topics.

37. LU decomposition

Factor a square matrix into a lower × upper triangular matrix:

A = L U      (L lower-triangular, U upper-triangular)

It comes straight out of Gaussian elimination (U = the echelon form; L records the multipliers). To solve Ax = b:

1. forward-substitute  Ly = b   (find y)
2. back-substitute     Ux = y   (find x)

Efficient when solving for many right-hand sides b with the same A. (Row swaps give PA = LU.)

38. Common GATE traps
  • rank(A)=rank([A|b]) ⇒ unique? No — that only guarantees a solution exists; unique also needs rank = n.
  • rank(A) < n ⇒ no solution? No — if consistent it means infinitely many.
  • Homogeneous Ax=0 can have no solution? No — x = 0 always works.
  • Square det(A)=0 ⇒ Ax=b has no solution? No — it means not unique (none or infinite).
  • A invertible ⇒ unique solution for every b? True.
  • More unknowns than equations (homogeneous) ⇒ non-trivial exists? True.
39. Practice problems

Q1. r=2, r'=3, n=4?

r ≠ r' → no solution

Q2. r=3, r'=3, n=3?

r = r' = n → unique

Q3. r=2, r'=2, n=5?

r = r' < n → infinite; free variables = 5 − 2 = 3

Q4. A is 3×5, Ax = 0 — only trivial?

rank ≤ 3 < 5 → nullity > 0 → non-trivial solutions exist

Q5. A is 4×4, det ≠ 0?

Ax=b: unique for every b ; Ax=0: only trivial

Q6. A is 4×4, det = 0?

Ax=0: non-trivial solutions ; Ax=b: none or infinitely many
40. Cheatsheet
  • Write it as Ax = b; compare r = rank(A) and r' = rank([A|b]) with n = unknowns.
  • r ≠ r' → none; r = r' = n → unique; r = r' < n → infinite.
  • Free variables = n − rank(A).
  • Ax=0 always has x = 0; only trivial ⟺ rank = n; non-trivial ⟺ rank < n.
  • Square: rank = n ⟺ det ≠ 0 ⟺ invertible.
  • General solution x = xₚ + xₕ; solvable ⟺ b ∈ C(A).
41. Best mental model 
A = machine ,  x = input/control ,  b = target output
Ax = b  →  find the input x that produces target b
  • No solution = target b is unreachable.
  • Unique = exactly one input reaches b.
  • Infinite = many inputs reach the same b (move along the null space).

So x = xₚ + xₕ = one valid input + invisible moves that don't change the output.

5 Symbols & their meaning
SymbolRead asMeaning
$Ax=b$"A x equals b"the system
$[A\,|\,b]$"augmented matrix"coefficients + right side
$r=\operatorname{rank}(A)$"rank of A"independent equations
$r'=\operatorname{rank}([A|b])$"rank of augmented"with right side included
$n$"n"number of unknowns
6 Main rule (the decider)
$$ r \neq r' \;\Rightarrow\; \textbf{no solution} $$
$$ r = r' = n \;\Rightarrow\; \textbf{unique solution} $$
$$ r = r' < n \;\Rightarrow\; \boldsymbol{\infty}\textbf{ solutions} \ \ (n-r \text{ free variables}) $$
7 Step-by-step: Gaussian elimination
  1. Write the augmented matrix $[A\,|\,b]$.
  2. Use row operations (swap, scale, add multiples) to reach row-echelon form.
  3. Read off $r=\operatorname{rank}(A)$ and $r'=\operatorname{rank}([A|b])$.
  4. Apply the decider rule to classify no / unique / infinite.
  5. Back-substitute to get the actual solution (parameterize free variables if infinite).
8 Worked example
✦ Solve $x+2y=5,\ 2x+4y=10$

$\left[\begin{array}{cc|c}1&2&5\\2&4&10\end{array}\right]\xrightarrow{R_2-2R_1}\left[\begin{array}{cc|c}1&2&5\\0&0&0\end{array}\right]$

$r=r'=1<2$ ⇒ infinitely many solutions. $y$ is free; $x=5-2y$.

9 GATE cheat sheet
⚡ High-value facts
  • Homogeneous $Ax=0$ always has the trivial solution $x=0$.
  • $Ax=0$ has non-trivial solutions $\iff \operatorname{rank}(A)<n \iff \det A=0$ (square).
  • Square system has a unique solution for every $b$ $\iff \det A\neq0$.
  • General solution $=$ one particular solution $+$ any null-space vector: $x=x_p+x_h$.
  • Cramer's rule (small square systems, $\det A\neq0$): $x_i=\dfrac{\det(A_i)}{\det(A)}$, where $A_i$ is $A$ with column $i$ replaced by $b$. Clean but costly — use Gaussian elimination for big systems.
10 Common traps
✕ Watch out

Trap: "$\det A=0$ means no solution." Reality: it means not unique — could be none OR infinite; check $r$ vs $r'$.

Trap: "Homogeneous systems can be inconsistent." Reality: never — $x=0$ always works.

Trap: "More equations ⇒ no solution." Reality: only ranks decide, not the count.

11 Connections with other topics
  • Rank: the entire decider rule is built on ranks.
  • Determinant: square unique-solution test is $\det\neq0$.
  • Null space: the $x_h$ part of the solution lives in the null space.
  • Vectors: "$Ax=b$ solvable" means $b$ is in the span of $A$'s columns.
12 Practice problems

Q1. $r=2$, $r'=3$, $n=3$. How many solutions?

$r\neq r'$ ⇒ no solution (inconsistent).

Q2. $r=r'=2$, $n=4$. How many solutions?

$r=r'<n$ ⇒ infinitely many, with $4-2=2$ free variables.

Q3. When does $Ax=0$ (square $A$) have a non-trivial solution?

When $\det A=0$ (equivalently $\operatorname{rank}<n$).

Q4. For which $k$ does $x+2y=3,\ 2x+ky=7$ have no solution?   (A) $k=3$   (B) $k=4$   (C) $k=5$   (D) $k=6$

(B) $k=4$ — $\det=k-4=0$ at $k=4$; then $R_2-2R_1$ gives $[0\,0\,|\,1]$, so $r'\neq r$ → no solution.
13 Final memory box
🧠 Remember
  • $r\neq r'$ → none; $r=r'=n$ → unique; $r=r'<n$ → infinite ($n-r$ free).
  • $Ax=0$ always has $x=0$; non-trivial iff $\operatorname{rank}<n$ iff $\det=0$.
  • General solution $x=x_p+x_h$ (particular + null space).
  • Solve by Gaussian elimination on $[A\,|\,b]$.

GATE takeaway — compare rank(A), rank([A|b]) and n: it decides no, unique, or infinitely many solutions.
