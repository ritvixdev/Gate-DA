GATE Questions

Real GATE DA 2024, 2025 & 2026 questions from the Linear Algebra part of our syllabus, tagged by year. Attempt each (MCQ, MSQ or numerical) and check the official answer.

GATE DA 2026 Principal components (PCA) MCQ 1 mark
Q11. For a classification problem, PCA is used to reduce the feature-space dimensionality from 100 to 10. What is true about the angle $\theta$ between the first and the tenth principal components?
Single correct (MCQ).
Principal components are eigenvectors of the symmetric covariance matrix, so any two distinct PCs are mutually orthogonal: $\theta=90^\circ$.
GATE DA 2026 Matrix powers MCQ 1 mark
Q21. Let $M=\begin{pmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{pmatrix}$ with $\theta=\dfrac{2\pi}{5}$, and $I_2=\begin{pmatrix}1&0\\0&1\end{pmatrix}$. Which of the following equals $M^{2026}$?
Single correct (MCQ).
$M$ is a rotation by $\theta=\tfrac{2\pi}{5}$, so $M^5=I_2$ (a full $2\pi$ turn). Since $2026=5(405)+1$, $M^{2026}=M^{1}=M$.
GATE DA 2026 Subspaces & geometry MCQ 1 mark
Q22. Let $S_1=\{x=(x_1,x_2,x_3)^{\mathsf T}\in\mathbb{R}^3 : x^{\mathsf T}x\le16\}$ and let $S_2$ be a $2$-dimensional subspace of $\mathbb{R}^3$. What is the area of $S_1\cap S_2$?
Single correct (MCQ).
$S_1$ is a ball of radius $4$ ($x^{\mathsf T}x\le16$). A $2$-D subspace is a plane through the origin, which cuts the ball in a disk of radius $4$, so the area is $\pi(4)^2=16\pi$.
GATE DA 2026 Eigenvalues MCQ 2 marks
Q46. Let $\gamma_1,\gamma_2,\gamma_3$ be the eigenvalues of $\begin{bmatrix}1&0&0\\0&\cos t&\sin t\\0&-\sin t&\cos t\end{bmatrix}$, where $t\in[-\pi,\pi]$. Which option lists all values of $t$ with $\gamma_1+\gamma_2+\gamma_3=1+\sqrt2$?
Single correct (MCQ).
The eigenvalues are $1$ and $\cos t\pm i\sin t$, so the sum is $1+2\cos t$. Setting $1+2\cos t=1+\sqrt2$ gives $\cos t=\tfrac1{\sqrt2}$, i.e. $t=\pm\tfrac\pi4$.
GATE DA 2026 Projection matrix MSQ 2 marks
Q52. Let $M=I_n-\dfrac1n\mathbf{1}\mathbf{1}^{\mathsf T}$, where $\mathbf{1}=(1,1,\dots,1)^{\mathsf T}\in\mathbb{R}^n$ and $I_n$ is the identity of order $n$. Which is/are correct?
Multiple correct (MSQ) — select all that apply.
$M$ is the centering matrix: symmetric ($M^{\mathsf T}=M$, A) and idempotent $M^2=M$, hence a projection (D). Its eigenvalues are $1$ ($n-1$ times) and $0$, so $\operatorname{Trace}(M)=n-1$ (C false) and $M^2=M\ne I_n$ (B false).
GATE DA 2026 Eigenvalues NAT 2 marks
Q65. Let $M=I_n-\dfrac1n\mathbf{1}\mathbf{1}^{\mathsf T}$ with $\mathbf{1}=(1,\dots,1)^{\mathsf T}\in\mathbb{R}^n$. The value of $\displaystyle\max_{x\in S}\,x^{\mathsf T}Mx$, where $S=\{x\in\mathbb{R}^n:x^{\mathsf T}x=1\}$, is ______.
Numerical answer (NAT) — type a value and press Check.
The centering matrix $M$ has eigenvalues $0$ (eigenvector $\mathbf{1}$) and $1$ ($n-1$ times). Over unit vectors, $x^{\mathsf T}Mx$ is maximized by the largest eigenvalue, $1$.
GATE DA 2025 Solution of equations MCQ 1 mark
Q12. The number of additions and multiplications involved in performing Gaussian elimination on an $n\times n$ upper-triangular matrix is of the order
Single correct (MCQ).
For an upper-triangular system only back-substitution remains: row $i$ needs about $n-i$ multiply-adds, and $\sum_i (n-i)=O(n^2)$. (Full Gaussian elimination on a general matrix is $O(n^3)$.)
GATE DA 2025 Eigenvalues MCQ 1 mark
Q13. The sum of the elements in each row of $A\in\mathbb{R}^{n\times n}$ is $1$. If $B=A^3-2A^2+A$, which ONE statement is correct (for $x\in\mathbb{R}^n$)?
Single correct (MCQ).
Row sums $1\Rightarrow A$ has eigenvalue $1$ (eigenvector $\mathbf{1}$). $B=A(A-I)^2$ has eigenvalue $1\cdot(1-1)^2=0$, so $B$ is singular and $Bx=0$ has infinitely many solutions.
GATE DA 2025 Orthonormal basis MSQ 1 mark
Q25. Which of the following statements is/are correct?
Multiple correct (MSQ) — select all that apply.
Orthonormal bases are not unique (B). Orthonormal $\Rightarrow$ linearly independent (D). Independence does not imply orthonormality (C false).
GATE DA 2025 Eigenvalues MSQ 1 mark
Q28. Let $A=I_n+xx^{\mathsf T}$, where $I_n$ is the identity and $x\in\mathbb{R}^n$ with $x^{\mathsf T}x=1$. Which is/are correct?
Multiple correct (MSQ) — select all that apply.
$xx^{\mathsf T}$ has eigenvalues $1$ (once) and $0$ ($n-1$ times), so $A$ has eigenvalues $2$ and $1$ — all positive. Hence full rank and invertible (A, B); no zero or negative eigenvalues.
GATE DA 2025 Rank MCQ 2 marks
Q37. Let $A\in\mathbb{R}^{n\times n}$ satisfy $A^3=A$. Which ONE statement is ALWAYS correct?
Single correct (MCQ).
$A^3=A$ makes $A$ diagonalizable with eigenvalues in $\{-1,0,1\}$, so $\operatorname{rank}(A)=\operatorname{rank}(A^2)$. The others can fail (e.g. $A=I$).
GATE DA 2025 Positive definiteness MCQ 2 marks
Q38. Let $\{x_1,\dots,x_n\}$ be linearly independent in $\mathbb{R}^n$, and let $A\in\mathbb{R}^{n\times n}$ have $(i,j)$-entry $A_{ij}=x_i^{\mathsf T}x_j$. Which ONE statement is correct?
Single correct (MCQ).
$A$ is the Gram matrix of independent vectors, hence symmetric positive-definite and invertible (A). So no singular value is $0$, $\det\neq0$, and $z^{\mathsf T}Az>0$ for all $z\neq0$.
GATE DA 2025 Projections MSQ 2 marks
Q50. Let $x_1,\dots,x_5$ be orthonormal in $\mathbb{R}^{10}$ and $A=x_1x_1^{\mathsf T}+\dots+x_5x_5^{\mathsf T}$. Which is/are correct?
Multiple correct (MSQ) — select all that apply.
$A$ is the orthogonal projection onto a 5-D subspace: symmetric with eigenvalues $1$ (five times) and $0$ (five times). So singular values $=$ eigenvalues $\in\{0,1\}$ (A, B); having zero eigenvalues, $\det=0$ and $A$ is not invertible.
GATE DA 2025 Orthogonal matrices MSQ 2 marks
Q52. An $n\times n$ real matrix $A$ satisfies $\lVert Ax\rVert_2=\lVert x\rVert_2$ for all $x\in\mathbb{R}^n$. Which is/are ALWAYS correct?
Multiple correct (MSQ) — select all that apply.
Norm-preserving $\Rightarrow A$ is orthogonal (A) and invertible / full rank (D). But orthogonal matrices include rotations, so $A=I$ is not the only one and eigenvalues may be complex on the unit circle.
GATE DA 2025 PCA / covariance NAT 2 marks
Q60. A dataset $D=\{x^{(1)},\dots,x^{(n)}\}$ with each $x^{(i)}\in\mathbb{R}^{100}$ satisfies $\sum_{i=1}^{n}x^{(i)}=0$. Its covariance matrix has eigenvalues $\lambda_i=100^{\,2-i}$, $1\le i\le100$. Let $u$ ($u^{\mathsf T}u=1$) be the direction of maximum variance. Then $\displaystyle\frac1n\sum_{i=1}^{n}\big(u^{\mathsf T}x^{(i)}\big)^2$ = ______.
Numerical answer (NAT) — type a value and press Check.
For centered data, $\frac1n\sum (u^{\mathsf T}x^{(i)})^2=u^{\mathsf T}Cu$, maximized over unit $u$ by the top eigenvector, giving the largest eigenvalue $\lambda_1=100^{\,2-1}=100$.
GATE DA 2024 Eigenvalues MCQ 1 mark
Q13. Consider the matrix $A=\begin{bmatrix}2&-1\\3&1\end{bmatrix}$. Which ONE of the following statements is TRUE?
Single correct (MCQ).
$\operatorname{tr}(A)=3,\ \det(A)=5\Rightarrow \lambda^2-3\lambda+5=0$, discriminant $9-20=-11<0$, so the eigenvalues are a complex conjugate pair.
GATE DA 2024 Determinants NAT 1 mark
Q35. Consider the matrix $M=\begin{bmatrix}1&2&3\\3&1&3\\4&3&6\end{bmatrix}$. The determinant of $\big(M^2+12M\big)$ is ______.
Numerical answer (NAT) — type a value and press Check.
$M^2+12M=M(M+12I)$, so $\det(M^2+12M)=\det(M)\,\det(M+12I)$. Here $\det(M)=0$ (the rows are linearly dependent), hence the value is $0$.
GATE DA 2024 Subspaces MSQ 2 marks
Q47. Select all choices that are subspaces of $\mathbb{R}^3$ (with $\alpha,\beta\in\mathbb{R}$).
Multiple correct (MSQ) — select all that apply.
A is a span (subspace); C is a homogeneous linear system (subspace). B uses $\alpha^2,\beta^2\ge0$ so it is not closed under scaling; D is affine (does not contain $0$).
GATE DA 2024 Solution of equations MSQ 2 marks
Q48. Which of the following statements is/are TRUE? ($b,c$ are vectors of matching size.)
Multiple correct (MSQ) — select all that apply.
A: an invertible $M$ makes $M^{\mathsf T}$ invertible too (unique, not infinite). C: a $2\times3$ system is underdetermined, so it can never have a unique solution. B and D are both achievable.
GATE DA 2024 Projections MSQ 2 marks
Q49. Let $W$ be a subspace of $\mathbb{R}^3$ and $P\in\mathbb{R}^{3\times3}$ the matrix of the orthogonal projection onto $W$. Which is/are TRUE?
Multiple correct (MSQ) — select all that apply.
$\operatorname{null}(P)=W^\perp$, so $\dim=3-\dim W$: for 1-D $W$ it is 2-D (A false), for 2-D $W$ it is 1-D (B true). A projection is idempotent: $P^2=P$, hence $P^3=P$ (C, D true).
GATE DA 2024 Singular values (SVD) NAT 2 marks
Q61. Let $A=\begin{bmatrix}1\\2\\3\\4\\5\end{bmatrix}$ and let $\sigma_1,\dots,\sigma_5$ be the singular values of $M=AA^{\mathsf T}$. The value of $\displaystyle\sum_{i=1}^{5}\sigma_i$ is ______.
Numerical answer (NAT) — type a value and press Check.
$M=AA^{\mathsf T}$ is rank-1 and symmetric PSD; its only non-zero singular value equals $A^{\mathsf T}A=1+4+9+16+25=55$, the rest are $0$. Sum $=55$.

Extra practice questions (quiz-style, not official GATE papers) covering the same Linear Algebra topics.

Practice Set Systems of linear equations MCQ 1 mark
P1. Consider the system: $x+y+z=3$, $2x+2y+2z=6$, $x-y+z=1$. What is the nature of the solution?
Single correct (MCQ).
Equation 2 is just Equation 1 doubled, so it adds no new information — only 2 independent equations remain for 3 unknowns. Solving Eq. 1 and Eq. 3 gives $y=1$ and $x+z=2$, leaving $x$ free, so there are infinitely many solutions.
Practice Set Eigenvalues & eigenvectors MCQ 2 marks
P2. A $2\times2$ rotation matrix $A$ rotates every real vector by $45^\circ$ (all entries of $A$ are real). Select the correct option.
Single correct (MCQ).
For a rotation by $\theta$, the eigenvalues solve $\lambda^2-2\cos\theta\,\lambda+1=0$, giving $\lambda=\cos\theta\pm i\sin\theta$. Since $\theta=45^\circ$ has $\sin\theta\ne0$, the eigenvalues are complex, so no real eigenvectors exist. Real coefficients in the characteristic polynomial do not guarantee real roots.

P3. Suppose $\lambda$ is an eigenvalue of a square matrix $A$ for some eigenvector $x$. Then, for the matrix $A-\lambda I$,
Single correct (MCQ).
By definition $(A-\lambda I)x=0$ for a non-zero $x$. If $A-\lambda I$ were invertible, multiplying both sides by its inverse would force $x=0$, a contradiction. So $A-\lambda I$ is always singular.

Practice Set Eigenvalues — worked example MSQ 2 marks
P5. Consider the matrix $A=\begin{bmatrix}2&1\\1&2\end{bmatrix}$. Select the correct option(s).
Multiple correct (MSQ) — select all that apply.
$A=A^{\mathsf T}$, so it's symmetric. $\operatorname{trace}(A)=4$, $\det(A)=3\Rightarrow\lambda^2-4\lambda+3=0\Rightarrow\lambda=1,3$. Solving $(A-3I)v=0$ gives $v=[1,1]$; solving $(A-I)v=0$ gives $v=[1,-1]$ — both check out, and being eigenvectors of a symmetric matrix for distinct eigenvalues, they are orthogonal.
Practice Set Vectors — convex combinations MCQ 3 marks
P6. Given non-zero vectors $u,v$, let $w=u+\tfrac13(v-u)$. Let $O$ be the origin, $A,B$ the tips of $u,v$, and $X$ the tip of $w$. Which is True of $\triangle OAB$?
Single correct (MCQ).
$w=\tfrac23u+\tfrac13v$, a convex combination ($\alpha+\beta=1$, both $\ge0$) of $u$ and $v$, so $X$ lies exactly on segment $AB$ — one side of the triangle, $1/3$ of the way from $A$ to $B$.
Practice Set Vector norms MCQ 1 mark
P7. A vector of ones in $n$ dimensions is $[1,1,\dots,1]$. The length of such a vector is:
Single correct (MCQ).
$\lVert\mathbf{1}\rVert=\sqrt{1^2+1^2+\dots+1^2}$ ($n$ terms) $=\sqrt{n}$.
Practice Set Vectors — dot product & triangles MCQ 3 marks
P8. Three vectors $a,b,c$ have magnitudes $\lVert a\rVert=3$, $\lVert b\rVert=4$, $\lVert c\rVert=5$. Select the correct option.
Single correct (MCQ).
If $a+b+c=0$ then $c=-(a+b)$, so $\lVert c\rVert^2=\lVert a\rVert^2+\lVert b\rVert^2+2(a\cdot b)$. Plugging in $25=9+16+2(a\cdot b)$ gives $a\cdot b=0$ — a forced right angle between $a$ and $b$. Without the constraint, the magnitudes alone say nothing about angles.
Practice Set Matrices — determinants MCQ 2 marks
P9. Given $A=\begin{bmatrix}1&3\\2&4\end{bmatrix}$. Which statement is correct?
Single correct (MCQ).
$\det(A)=(1)(4)-(3)(2)=-2\ne0$, so $A$ is invertible, full rank, and its columns are linearly independent.

Practice Set Vectors — proportionality MCQ 3 marks
P11. True or False: If $[a,b]$ is a multiple of $[c,d]$ with $abcd\ne0$, then $[a,c]$ is a multiple of $[b,d]$.
Single correct (MCQ).
$[a,b]=k[c,d]$ means $a=kc,\,b=kd$, so $ad=bc$. Dividing both sides by $bd$ (all non-zero) gives $a/b=c/d$, exactly the condition for $[a,c]$ to be a multiple of $[b,d]$.
