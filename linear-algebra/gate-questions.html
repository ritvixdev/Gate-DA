GATE DA 2025 Solution of equations MCQ 1 mark
Q12. The number of additions and multiplications involved in performing Gaussian elimination on an $n\times n$ upper-triangular matrix is of the order
Single correct (MCQ).
For an upper-triangular system only back-substitution remains: row $i$ needs about $n-i$ multiply-adds, and $\sum_i (n-i)=O(n^2)$. (Full Gaussian elimination on a general matrix is $O(n^3)$.)
GATE DA 2025 Eigenvalues MCQ 1 mark
Q13. The sum of the elements in each row of $A\in\mathbb{R}^{n\times n}$ is $1$. If $B=A^3-2A^2+A$, which ONE statement is correct (for $x\in\mathbb{R}^n$)?
Single correct (MCQ).
Row sums $1\Rightarrow A$ has eigenvalue $1$ (eigenvector $\mathbf{1}$). $B=A(A-I)^2$ has eigenvalue $1\cdot(1-1)^2=0$, so $B$ is singular and $Bx=0$ has infinitely many solutions.

Q25. Which of the following statements is/are correct?

Orthonormal bases are not unique (B). Orthonormal $\Rightarrow$ linearly independent (D). Independence does not imply orthonormality (C false).
GATE DA 2025 Eigenvalues MSQ 1 mark
Q28. Let $A=I_n+xx^{\mathsf T}$, where $I_n$ is the identity and $x\in\mathbb{R}^n$ with $x^{\mathsf T}x=1$. Which is/are correct?
Multiple correct (MSQ) — select all that apply.
$xx^{\mathsf T}$ has eigenvalues $1$ (once) and $0$ ($n-1$ times), so $A$ has eigenvalues $2$ and $1$ — all positive. Hence full rank and invertible (A, B); no zero or negative eigenvalues.
GATE DA 2025 Rank MCQ 2 marks
Q37. Let $A\in\mathbb{R}^{n\times n}$ satisfy $A^3=A$. Which ONE statement is ALWAYS correct?
Single correct (MCQ).
$A^3=A$ makes $A$ diagonalizable with eigenvalues in $\{-1,0,1\}$, so $\operatorname{rank}(A)=\operatorname{rank}(A^2)$. The others can fail (e.g. $A=I$).
GATE DA 2025 Positive definiteness MCQ 2 marks
Q38. Let $\{x_1,\dots,x_n\}$ be linearly independent in $\mathbb{R}^n$, and let $A\in\mathbb{R}^{n\times n}$ have $(i,j)$-entry $A_{ij}=x_i^{\mathsf T}x_j$. Which ONE statement is correct?
Single correct (MCQ).
$A$ is the Gram matrix of independent vectors, hence symmetric positive-definite and invertible (A). So no singular value is $0$, $\det\neq0$, and $z^{\mathsf T}Az>0$ for all $z\neq0$.
GATE DA 2025 Projections MSQ 2 marks
Q50. Let $x_1,\dots,x_5$ be orthonormal in $\mathbb{R}^{10}$ and $A=x_1x_1^{\mathsf T}+\dots+x_5x_5^{\mathsf T}$. Which is/are correct?
Multiple correct (MSQ) — select all that apply.
$A$ is the orthogonal projection onto a 5-D subspace: symmetric with eigenvalues $1$ (five times) and $0$ (five times). So singular values $=$ eigenvalues $\in\{0,1\}$ (A, B); having zero eigenvalues, $\det=0$ and $A$ is not invertible.
GATE DA 2025 Orthogonal matrices MSQ 2 marks
Q52. An $n\times n$ real matrix $A$ satisfies $\lVert Ax\rVert_2=\lVert x\rVert_2$ for all $x\in\mathbb{R}^n$. Which is/are ALWAYS correct?
Multiple correct (MSQ) — select all that apply.
Norm-preserving $\Rightarrow A$ is orthogonal (A) and invertible / full rank (D). But orthogonal matrices include rotations, so $A=I$ is not the only one and eigenvalues may be complex on the unit circle.
GATE DA 2025 PCA / covariance NAT 2 marks
Q60. A dataset $D=\{x^{(1)},\dots,x^{(n)}\}$ with each $x^{(i)}\in\mathbb{R}^{100}$ satisfies $\sum_{i=1}^{n}x^{(i)}=0$. Its covariance matrix has eigenvalues $\lambda_i=100^{\,2-i}$, $1\le i\le100$. Let $u$ ($u^{\mathsf T}u=1$) be the direction of maximum variance. Then $\displaystyle\frac1n\sum_{i=1}^{n}\big(u^{\mathsf T}x^{(i)}\big)^2$ = ______.
Numerical answer (NAT) — type a value and press Check.
For centered data, $\frac1n\sum (u^{\mathsf T}x^{(i)})^2=u^{\mathsf T}Cu$, maximized over unit $u$ by the top eigenvector, giving the largest eigenvalue $\lambda_1=100^{\,2-1}=100$.
GATE DA 2024 Eigenvalues MCQ 1 mark
Q13. Consider the matrix $A=\begin{bmatrix}2&-1\\3&1\end{bmatrix}$. Which ONE of the following statements is TRUE?
Single correct (MCQ).
$\operatorname{tr}(A)=3,\ \det(A)=5\Rightarrow \lambda^2-3\lambda+5=0$, discriminant $9-20=-11<0$, so the eigenvalues are a complex conjugate pair.
GATE DA 2024 Determinants NAT 1 mark
Q35. Consider the matrix $M=\begin{bmatrix}1&2&3\\3&1&3\\4&3&6\end{bmatrix}$. The determinant of $\big(M^2+12M\big)$ is ______.
Numerical answer (NAT) — type a value and press Check.
$M^2+12M=M(M+12I)$, so $\det(M^2+12M)=\det(M)\,\det(M+12I)$. Here $\det(M)=0$ (the rows are linearly dependent), hence the value is $0$.
GATE DA 2024 Subspaces MSQ 2 marks
Q47. Select all choices that are subspaces of $\mathbb{R}^3$ (with $\alpha,\beta\in\mathbb{R}$).
Multiple correct (MSQ) — select all that apply.
A is a span (subspace); C is a homogeneous linear system (subspace). B uses $\alpha^2,\beta^2\ge0$ so it is not closed under scaling; D is affine (does not contain $0$).
GATE DA 2024 Solution of equations MSQ 2 marks
Q48. Which of the following statements is/are TRUE? ($b,c$ are vectors of matching size.)
Multiple correct (MSQ) — select all that apply.
A: an invertible $M$ makes $M^{\mathsf T}$ invertible too (unique, not infinite). C: a $2\times3$ system is underdetermined, so it can never have a unique solution. B and D are both achievable.
GATE DA 2024 Projections MSQ 2 marks
Q49. Let $W$ be a subspace of $\mathbb{R}^3$ and $P\in\mathbb{R}^{3\times3}$ the matrix of the orthogonal projection onto $W$. Which is/are TRUE?
Multiple correct (MSQ) — select all that apply.
$\operatorname{null}(P)=W^\perp$, so $\dim=3-\dim W$: for 1-D $W$ it is 2-D (A false), for 2-D $W$ it is 1-D (B true). A projection is idempotent: $P^2=P$, hence $P^3=P$ (C, D true).
GATE DA 2024 Singular values (SVD) NAT 2 marks
Q61. Let $A=\begin{bmatrix}1\\2\\3\\4\\5\end{bmatrix}$ and let $\sigma_1,\dots,\sigma_5$ be the singular values of $M=AA^{\mathsf T}$. The value of $\displaystyle\sum_{i=1}^{5}\sigma_i$ is ______.
Numerical answer (NAT) — type a value and press Check.
$M=AA^{\mathsf T}$ is rank-1 and symmetric PSD; its only non-zero singular value equals $A^{\mathsf T}A=1+4+9+16+25=55$, the rest are $0$. Sum $=55$.
