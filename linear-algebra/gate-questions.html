GATE DA 2024 Eigenvalues
Q1. Consider the matrix $A=\begin{bmatrix}2&-1\\3&1\end{bmatrix}$. Which ONE of the following statements is TRUE?
Real non-negative eigenvalues would need a non-negative discriminant — not the case here.
✓ Correct — $\operatorname{tr}(A)=3$, $\det(A)=5$, so $\lambda^2-3\lambda+5=0$ with discriminant $9-20=-11<0$ ⇒ complex conjugate pair.
A zero eigenvalue would require $\det(A)=0$, but $\det(A)=5\neq0$.
Both real (one negative) would need $\det(A)<0$, but $\det(A)=5>0$.
