0 The complete subject in one picture

Linear algebra is one connected story:

$$\text{Vectors}\to\text{Matrices}\to\text{Determinant}\to\text{Rank and Nullity}\to\text{Linear Equations}$$
$$\text{Eigenvalues}\to\text{Diagonalization}\to\text{Orthogonality and Projection}\to\text{SVD}\to\text{PCA}$$

A vector is an arrow, direction, or data point. A matrix is a machine that transforms vectors. Rank counts how many independent directions survive; nullity counts how many are destroyed. The determinant tells whether any dimension was completely collapsed. An inverse reverses the transformation. An eigenvector is a direction that does not turn; its eigenvalue is how much it's scaled. Orthogonality means perpendicularity. A projection gives the shadow on a subspace. SVD expresses any matrix using rotations and stretches, and PCA keeps the directions containing the most variance.

2 The most important equivalence chain

For a square $n\times n$ matrix $A$:

$$\det(A)\ne0\iff A^{-1}\text{ exists}\iff\operatorname{rank}(A)=n$$

equivalently:

$$\text{columns are independent}\iff N(A)=\{0\}\iff Ax=0\text{ has only }x=0\iff0\text{ is not an eigenvalue}$$

and the opposite chain:

$$\det(A)=0\iff A\text{ is singular}\iff A^{-1}\text{ does not exist}$$
$$\operatorname{rank}(A)\lt n\iff\text{columns are dependent}\iff Ax=0\text{ has a non-zero solution}\iff0\text{ is an eigenvalue}$$
🧠 Memory sentence
  • Non-zero determinant means nothing was destroyed, so the matrix can be reversed.
3 Vectors — length, unit vectors & the sum/difference shortcut

3.1 Length / norm — for $x=[x_1,\ldots,x_n]^T$:

$$\|x\|=\sqrt{x_1^2+\cdots+x_n^2}=\sqrt{x^Tx}$$

Use this when the question says: length, magnitude, distance from origin, unit vector, normalize.

For the all-ones vector $\mathbf1=[1,\ldots,1]^T\in\mathbb R^n$: $\|\mathbf1\|=\sqrt n$, because $\mathbf1^T\mathbf1=n$.

3.2 Unit vector — length $1$; to normalize $x$:

$$\hat x=\frac{x}{\|x\|}$$

3.3 Sum and difference shortcut — if $x+y=a$, $x-y=b$:

$$x=\frac{a+b}2\qquad y=\frac{a-b}2$$
4 Dot product, angle & perpendicularity
$$x^Ty=x_1y_1+\cdots+x_ny_n\qquad\text{(a scalar, not a vector)}$$
$$\cos\theta=\frac{x^Ty}{\|x\|\|y\|}$$
ConditionAngle
$x^Ty\gt0$$\theta\lt90^\circ$
$x^Ty=0$$\theta=90^\circ$
$x^Ty\lt0$$\theta\gt90^\circ$
$$x\perp y\iff x^Ty=0$$

Useful expansion:

$$\|x+y\|^2=\|x\|^2+\|y\|^2+2x^Ty\qquad\|x-y\|^2=\|x\|^2+\|y\|^2-2x^Ty$$
🎯 GATE trigger

If lengths like $3,4,5$ are given together with $a+b+c=0$, write $c=-(a+b)$, expand $\|c\|^2=\|a+b\|^2$ — this very often proves $a^Tb=0$.

5 Linear combinations, span, independence and basis

5.1 Linear combination — $c_1v_1+c_2v_2+\cdots+c_kv_k$ means scaling vectors and adding them.

5.2 Span — every vector reachable using the supplied directions:

$$\operatorname{span}\{v_1,\ldots,v_k\}=\{c_1v_1+\cdots+c_kv_k\}$$

5.3 Independence — vectors are independent when $c_1v_1+\cdots+c_kv_k=0$ forces $c_1=\cdots=c_k=0$: no vector is redundant.

🔍 Quick checks

Two non-zero vectors are dependent if one is a scalar multiple of the other. More than $n$ vectors in $\mathbb R^n$ must be dependent.

5.4 Basis — must be linearly independent and span the whole space. The number of basis vectors is the dimension: $\dim(\mathbb R^n)=n$.

6 Subspaces

A set $S\subseteq\mathbb R^n$ is a subspace if:

$$0\in S\qquad x,y\in S\Rightarrow x+y\in S\qquad x\in S,\ c\in\mathbb R\Rightarrow cx\in S$$
🔍 Fastest exam method

Test zero first — if the zero vector is not present, stop: not a subspace. A span is always a subspace. A homogeneous system $\{x:Ax=0\}=N(A)$ is always a subspace. A non-homogeneous system $\{x:Ax=b\}$, $b\ne0$, is generally not, since zero doesn't satisfy it. Expressions with squared coefficients like $\alpha^2v_1+\beta^2v_2$ are suspicious — they can't scale negatively.

🧠 Memory sentence
  • Subspaces must pass through the origin.
7 Affine and convex combinations
$$w=\alpha u+\beta v,\qquad\alpha+\beta=1\qquad\text{(point on the line through }u,v\text{)}$$

A convex combination additionally requires $\alpha,\beta\ge0$ — then the point lies on the segment between $u$ and $v$.

CoefficientsLocation
Sum $=1$On the complete line
Sum $=1$, both non-negativeOn the segment
More than two non-negative, summing to $1$Inside/on the convex shape

Example: $u+\tfrac13(v-u)=\tfrac23u+\tfrac13v$ — coefficients non-negative and sum to one, so it's on the segment joining $u$ and $v$.

9 Geometry shortcuts

$n$-dimensional cube — each coordinate has two corner choices:

$$\text{number of vertices}=2^n$$

Ball equation — $x^Tx\le r^2$ means $\|x\|\le r$: a solid ball of radius $r$. In $\mathbb R^3$, a two-dimensional subspace is a plane through the origin, and a plane through the centre of a radius-$r$ ball cuts out a disk:

$$\text{area}=\pi r^2$$
10 Matrices — multiplication, transpose, symmetry & trace

An $m\times n$ matrix has $m$ rows, $n$ columns, and transforms $A:\mathbb R^n\to\mathbb R^m$. Input dimension $=$ number of columns; output dimension $=$ number of rows.

10.1 Multiplication — if $A:m\times n$ and $B:n\times p$, then $AB:m\times p$.

🧠 Memory
  • Inner dimensions match; outer dimensions survive. $AB\ne BA$ in general, but $A(BC)=(AB)C$.

10.2 Transpose:

$$(AB)^T=B^TA^T\qquad\text{(the order reverses)}$$

10.3 Symmetric matrix — $A=A^T$. Real symmetric matrices: eigenvalues are real; eigenvectors for distinct eigenvalues are orthogonal; they can be orthogonally diagonalized; Rayleigh-quotient rules apply directly.

10.4 Trace:

$$\operatorname{tr}(A)=\sum_ia_{ii}=\sum_i\lambda_i\qquad\operatorname{tr}(AB)=\operatorname{tr}(BA)$$
🧠 Memory
  • Trace adds eigenvalues.
11 Rotation matrices
$$R(\theta)=\begin{bmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{bmatrix}\qquad R(\alpha)R(\beta)=R(\alpha+\beta)\qquad R(\theta)^k=R(k\theta)$$

$R(2\pi)=I$. If $\theta=\dfrac{2\pi}p$, then $R(\theta)^p=I$.

🎯 Large power trick

Reduce the exponent modulo $p$. Example: for $\theta=\dfrac{2\pi}5$, since $2026=5(405)+1$, $R(\theta)^{2026}=R(\theta)$.

12 Determinants
$$\det\!\begin{bmatrix}a&b\\c&d\end{bmatrix}=ad-bc$$

$|\det(A)|=$ area/volume scale factor. If $\det(A)=0$, at least one dimension was collapsed.

12.1 Key properties:

$$\det(AB)=\det(A)\det(B)\qquad\det(A^T)=\det(A)\qquad\det(A^{-1})=\frac1{\det(A)}\qquad\det(A^k)=\det(A)^k$$

For triangular/diagonal matrices, $\det(A)=\prod_ia_{ii}$. For $A\in\mathbb R^{n\times n}$, $\det(cA)=c^n\det(A)$, and $\det(A)=\prod_i\lambda_i$.

🧠 Memory
  • Determinant multiplies eigenvalues.

12.2 $2\times2$ inverse:

$$A^{-1}=\frac1{ad-bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}$$

Procedure: swap $a,d$; negate $b,c$; divide by $ad-bc$. Valid only if $ad-bc\ne0$.

12.3 Factor before you calculate — never multiply out $A^2+cA$ directly:

$$A^2+cA=A(A+cI)\qquad\det(A^2+cA)=\det(A)\det(A+cI)$$
⚠️ Trap

If $\det(A)=0$, the whole expression is instantly zero — no need to touch $A+cI$ at all.

13 Rank and nullity
$$\operatorname{rank}(A)=\text{number of pivots}=\dim C(A)=\dim\operatorname{Row}(A)$$
$$N(A)=\{x:Ax=0\}\qquad\text{nullity}=\dim N(A)$$

13.1 Rank-nullity theorem — for $A\in\mathbb R^{m\times n}$:

$$\operatorname{rank}(A)+\operatorname{nullity}(A)=n\qquad\text{free variables}=n-r$$
⚠️ Trap

$n$ is the number of columns, not rows.

13.2 Full row vs. column rank — full column rank ($\operatorname{rank}(A)=n$): columns independent, $Ax=0$ has only $x=0$, no free variables. Full row rank ($\operatorname{rank}(A)=m$): the matrix can generate every output in $\mathbb R^m$.

13.3 The four fundamental subspaces — for $A\in\mathbb R^{m\times n}$, rank $r$:

SpaceMeaningLives inDimension
$C(A)$Column space$\mathbb R^m$$r$
$C(A^T)$Row space$\mathbb R^n$$r$
$N(A)$$Ax=0$$\mathbb R^n$$n-r$
$N(A^T)$$A^Ty=0$$\mathbb R^m$$m-r$
$$N(A)\perp\operatorname{Row}(A)\qquad N(A^T)\perp C(A)$$
🧠 Memory sentence
  • Null space is perpendicular to row space, not column space.
14 Linear systems — the solution table

A system is $Ax=b$, augmented matrix $[A\mid b]$. Let $r=\operatorname{rank}(A)$, $r'=\operatorname{rank}([A\mid b])$, $n=$ number of unknowns:

$$r\ne r'\Rightarrow\text{no solution}\qquad r=r'=n\Rightarrow\text{unique solution}\qquad r=r'\lt n\Rightarrow\text{infinitely many solutions}$$

14.1 Recognizing inconsistency — a row $[0\ 0\ 0\mid c]$, $c\ne0$, means $0=c$: impossible, so no solution.

14.2 Homogeneous systems — $Ax=0$ always has the trivial solution $x=0$, so it can never have "no solution." Only $x=0$ if full column rank; infinitely many if rank $\lt n$:

$$\text{non-zero solution exists}\iff\operatorname{rank}(A)\lt n\iff\det(A)=0\ \text{(square }A\text{)}$$
15 Tall and wide systems, and the minimum-norm solution

Tall matrix ($m\gt n$, overdetermined) — more equations than unknowns; may have no solution or a unique solution, but not infinitely many if full column rank.

Wide matrix ($m\lt n$, underdetermined) — more unknowns than equations; a consistent system cannot have a unique solution since $\operatorname{rank}(A)\le m\lt n$, so at least one free variable exists.

For a consistent underdetermined system, the shortest solution is $x_{\min}=A^+b$. If $A$ has full row rank, $A^+=A^T(AA^T)^{-1}$; the fully general SVD formula is $A^+=V\Sigma^+U^T$.

16 Least-squares solutions & computational complexity

When $Ax=b$ has no exact solution, minimize $\|Ax-b\|^2$. Normal equations:

$$A^TA\hat x=A^Tb\qquad\text{if full column rank: }\hat x=(A^TA)^{-1}A^Tb$$

Residual $e=b-A\hat x$; at the optimum, $A^Te=0$ — the error is perpendicular to the column space of $A$.

$$\text{General Gaussian elimination: }O(n^3)\qquad\text{Upper-triangular back-substitution: }O(n^2)$$
17 Eigenvalues and eigenvectors — the basics
$$Ax=\lambda x\qquad x\ne0$$
EigenvalueEffect
$\lambda\gt1$Stretch
$0\lt\lambda\lt1$Shrink
$\lambda\lt0$Reverse and scale
$\lambda=0$Crush to zero

17.1 Finding eigenvalues — from $Ax=\lambda x$, rearrange to $(A-\lambda I)x=0$; for a non-zero solution, $A-\lambda I$ must be singular:

$$\det(A-\lambda I)=0\qquad\text{(characteristic equation)}\qquad(A-\lambda I)x=0\ \text{(eigenvectors)}$$

17.2 Why $A-\lambda I$ is singular — $(A-\lambda I)x=0$ for non-zero $x$ means $A-\lambda I$ is never invertible, by definition.

18 Eigenvalue identities, and zero eigenvalue vs. zero eigenvector
$$\sum_i\lambda_i=\operatorname{tr}(A)\qquad\prod_i\lambda_i=\det(A)$$

For triangular/diagonal matrices, $\lambda_i=$ diagonal entries. The characteristic polynomial of an $n\times n$ matrix has degree $n$.

⚠️ Trap

Zero is allowed as an eigenvalue ($Ax=0$ for some non-zero $x$), but the zero vector is never allowed as an eigenvector.

19 Eigenvalues of powers, inverses & polynomials

If $Ax=\lambda x$, then $A^kx=\lambda^kx$:

$$\text{eigenvalues of }A^k=\lambda_i^k\qquad\text{eigenvalues of }A^{-1}=\frac1{\lambda_i}\ \text{(if invertible)}$$

If $B=p(A)$ for a polynomial $p$, and $Av=\lambda v$, then $Bv=p(\lambda)v$ — eigenvalues transform as $\lambda\to p(\lambda)$.

🎯 Example

$B=A^3-2A^2+A$. If $1$ is an eigenvalue of $A$: $1^3-2(1)^2+1=0$, so $B$ has eigenvalue zero and is singular.

20 Equal row-sum shortcut & the $2\times2$ characteristic shortcut

If every row of $A$ sums to $s$: $A\mathbf1=s\mathbf1$, so $s$ is automatically an eigenvalue with eigenvector $\mathbf1=[1,\ldots,1]^T$.

For a $2\times2$ matrix, the characteristic equation is always:

$$\lambda^2-\operatorname{tr}(A)\lambda+\det(A)=0\qquad\Delta=\operatorname{tr}(A)^2-4\det(A)$$
DiscriminantEigenvalues
$\Delta\gt0$Two distinct real
$\Delta=0$Repeated real
$\Delta\lt0$Complex-conjugate pair
21 Rotation eigenvalues & the outer-product shortcut

A planar rotation by $\theta$ has eigenvalues $\lambda=\cos\theta\pm i\sin\theta$ — a genuine rotation other than $0^\circ$ or $180^\circ$ has no real eigenvector. Real characteristic-polynomial coefficients only guarantee complex roots come in conjugate pairs, not that any root is real.

For non-zero $x$, $xx^T$ has rank $1$. If $x^Tx=1$, eigenvalues of $xx^T$ are $1,0,\ldots,0$, so eigenvalues of $I+xx^T$ are $2,1,\ldots,1$ — full rank, positive definite, invertible, with inverse eigenvalues $\tfrac12,1,\ldots,1$.

22 The matrix equation $A^3=A$

Eigenvalues satisfy $\lambda^3=\lambda\Rightarrow\lambda(\lambda-1)(\lambda+1)=0$, so $\lambda\in\{-1,0,1\}$. Also $N(A)=N(A^2)$, so $\operatorname{rank}(A)=\operatorname{rank}(A^2)$.

⚠️ Trap

Do not conclude $A$ is always singular or always invertible: $A=0$ is singular, $A=I$ is invertible, and both satisfy $A^3=A$.

23 Diagonalization
$$A=PDP^{-1}\qquad P=[v_1\ v_2\ \cdots\ v_n]\qquad D=\operatorname{diag}(\lambda_1,\ldots,\lambda_n)$$

$A$ is diagonalizable $\iff$ it has $n$ independent eigenvectors. $n$ distinct eigenvalues guarantee this; repeated eigenvalues don't automatically break it. Every real symmetric matrix can be orthogonally diagonalized: $A=Q\Lambda Q^T$.

$$A^k=PD^kP^{-1}\qquad D^k=\operatorname{diag}(\lambda_1^k,\ldots,\lambda_n^k)$$
⚠️ Diagonalizable $\ne$ invertible

Diagonalizable means enough independent eigenvectors; invertible means no eigenvalue equals zero. A diagonal matrix containing zero is diagonalizable but not invertible.

24 Orthogonal and orthonormal vectors
$$\text{Orthogonal: }u^Tv=0\qquad\text{Orthonormal: }u^Tv=0,\ \|u\|=\|v\|=1$$
🧠 Memory
  • Orthonormal = orthogonal + normalized. Every set of non-zero orthogonal vectors is independent, but independent vectors need not be orthogonal. Orthonormal bases are not unique.
25 Orthogonal matrices

A square matrix $Q$ is orthogonal if $Q^TQ=I$, so $Q^{-1}=Q^T$. Orthogonal matrices preserve $\|Qx\|=\|x\|$ and $(Qu)^T(Qv)=u^Tv$ — lengths, angles, and distances. $\det(Q)=\pm1$, and $|\lambda_i|=1$.

⚠️ GATE trap

Eigenvalues are not necessarily only $+1,-1$ — rotations may have complex eigenvalues of magnitude $1$.

If $\|Ax\|=\|x\|$ for every $x$, then $A^TA=I$: $A$ is orthogonal and full rank.

26 Projections
$$\operatorname{proj}_u(v)=\frac{u^Tv}{u^Tu}u\qquad\text{(unit }u\text{: }(u^Tv)u\text{)}\qquad e=v-\operatorname{proj}_u(v),\ e\perp u$$

26.1 Projection matrix — $P^2=P$ (idempotence). Orthogonal projection also has $P^T=P$. Eigenvalues are $0$ or $1$. If $P$ projects onto a $k$-dimensional subspace: $\operatorname{rank}(P)=\operatorname{tr}(P)=k$, $\operatorname{nullity}(P)=n-k$, $\det(P)=0$ unless $P=I$, and $P^m=P$ for every $m\ge1$.

26.2 Orthonormal-column projection — if $Q$ has orthonormal columns ($Q^TQ=I$), the projection onto $C(Q)$ is $P=QQ^T$. If $x_1,\ldots,x_k$ are orthonormal, $P=x_1x_1^T+\cdots+x_kx_k^T$, with eigenvalues $1$ ($k$ times) and $0$ ($n-k$ times).

26.3 General full-column-rank projection — if the columns of $A$ form a basis of the subspace:

$$P=A(A^TA)^{-1}A^T$$
27 The centering matrix
$$M=I_n-\frac1n\mathbf1\mathbf1^T\qquad Mx=x-\bar x\mathbf1\ \left(\bar x=\tfrac1n\mathbf1^Tx\right)$$
$$M^T=M\qquad M^2=M\qquad M\mathbf1=0$$

Eigenvalues: $0$ once, $1$ exactly $n-1$ times. So $\operatorname{rank}(M)=n-1$, $\operatorname{tr}(M)=n-1$, $\det(M)=0$. It projects onto $\mathbf1^\perp$, the zero-mean subspace.

28 Rayleigh quotient
$$R_A(x)=\frac{x^TAx}{x^Tx}\qquad\lambda_{\min}\le R_A(x)\le\lambda_{\max}$$

For unit vectors ($x^Tx=1$):

$$\max_{\|x\|=1}x^TAx=\lambda_{\max}(A)\qquad\min_{\|x\|=1}x^TAx=\lambda_{\min}(A)$$
🎯 Instant trigger

Whenever you see $\max_{x^Tx=1}x^TAx$, think: largest eigenvalue. For the centering matrix, the largest eigenvalue is $1$, so the maximum is $1$.

29 Positive-definite matrices & Gram matrices
$$\text{PSD: }x^TAx\ge0\ \forall x\qquad\text{PD: }x^TAx\gt0\ \forall x\ne0\qquad\text{(symmetric)}A\text{ PD}\iff\lambda_i\gt0\ \forall i$$

Consequences of positive definiteness: all eigenvalues positive; determinant positive; full rank; invertible; no zero singular value.

29.1 Gram matrix — if $A_{ij}=x_i^Tx_j$ and $X=[x_1\ \cdots\ x_n]$, then $A=X^TX$. For any $z$: $z^TAz=\|Xz\|^2\ge0$ — every Gram matrix is PSD. If the $x_i$ are independent, $Xz=0\Rightarrow z=0$, so $z^TAz\gt0$ for $z\ne0$: the Gram matrix is positive definite and invertible.

🎯 Instant trigger

Whenever you see $A_{ij}=x_i^Tx_j$, think: Gram matrix $X^TX$.

30 Gram–Schmidt

Converts independent vectors into an orthonormal basis. For $v_1,v_2$: $u_1=v_1$, $u_2=v_2-\operatorname{proj}_{u_1}(v_2)$, then normalize $q_1=u_1/\|u_1\|$, $q_2=u_2/\|u_2\|$. For more vectors, subtract projections onto all previous orthogonal directions. This leads to QR decomposition:

$$A=QR\qquad Q\text{ has orthonormal columns}$$
31 Singular Value Decomposition
$$A=U\Sigma V^T$$

Interpretation: $V^T$ rotates/reflects the input, $\Sigma$ stretches along perpendicular directions, $U$ rotates/reflects the result. Works for square, rectangular, singular, and rank-deficient matrices alike.

31.1 Singular values:

$$\sigma_i=\sqrt{\lambda_i(A^TA)}\ge0\qquad\operatorname{rank}(A)=\text{number of non-zero singular values}$$
EigenvaluesSingular values
Square matrices onlyAny matrix
May be negativeNever negative
May be complexAlways real
Direction preservedPrincipal stretch amount

For a symmetric PSD matrix, $\sigma_i=\lambda_i$.

31.2 Pseudoinverse through SVD — if $A=U\Sigma V^T$, then $A^+=V\Sigma^+U^T$: reciprocate each non-zero singular value, keep zeros as zero, transpose the rectangular diagonal matrix.

31.3 Rank-one shortcut — for non-zero $a$, $M=aa^T$ has eigenvalues/singular values $\|a\|^2,0,\ldots,0$. Example: $a=[1,2,3,4,5]^T\Rightarrow\|a\|^2=1+4+9+16+25=55$ — the singular-value sum of $aa^T$ is $55$.

32 PCA

PCA finds the directions containing maximum variance. Rows of $X$ are samples, columns are features.

Step 1 — centre: $X_c=X-\text{feature means}$. Step 2 — covariance: $C=\tfrac1nX_c^TX_c$ (population) or $\tfrac1{n-1}X_c^TX_c$ (sample) — use whichever convention the question specifies. Step 3 — eigendecompose: $Cv_i=\lambda_iv_i$, where $v_i$ is a principal direction and $\lambda_i$ is the variance along it. Step 4 — sort: $\lambda_1\ge\lambda_2\ge\cdots$; PC1 corresponds to $\lambda_1$, PC2 to $\lambda_2$, and so on.

32.1 Orthogonality — because covariance matrices are symmetric, their eigenvectors can be orthonormal: $v_i^Tv_j=0$ for $i\ne j$, so distinct principal components are always $90^\circ$ apart.

32.2 Maximum variance — for a unit direction $u$, variance along $u$ is $u^TCu$:

$$\max_{\|u\|=1}u^TCu=\lambda_{\max}(C)$$
🧠 Memory
  • Maximum PCA variance = largest covariance eigenvalue.

32.3 Explained variance ratio:

$$\text{EVR}_i=\frac{\lambda_i}{\lambda_1+\cdots+\lambda_d}\qquad\text{top }k\text{: }\frac{\lambda_1+\cdots+\lambda_k}{\lambda_1+\cdots+\lambda_d}$$

32.4 PCA and SVD — for centred data, $X_c=U\Sigma V^T$, and the columns of $V$ are the principal directions. Sample covariance: $\lambda_i=\dfrac{\sigma_i^2}{n-1}$; population covariance: $\lambda_i=\dfrac{\sigma_i^2}{n}$.

35 Exam attack strategy
  • Step 1 — identify the object. Is it about a vector, matrix, system, eigenvalue, projection, or covariance/PCA?
  • Step 2 — circle trigger words. Minimum norm, maximum variance, full rank, row sums, unit vector, projection, orthogonal, determinant, homogeneous.
  • Step 3 — write the smallest useful identity. Don't write every formula you know — e.g. projection → $P^2=P$; row sums → $A\mathbf1=s\mathbf1$; maximum quadratic form → largest eigenvalue; system solutions → rank table; subspace → check zero.
  • Step 4 — eliminate options using properties. GATE often expects property recognition over full calculation: a projection with zero eigenvalues can't be invertible; a positive-definite matrix can't have a zero singular value; a wide consistent system can't have a unique solution; a homogeneous system can't have no solution.
  • Step 5 — calculate only if still necessary. This saves time and reduces mistakes.

37 Final mental map

Remember this chain:

Vectors combine to create spaces. Matrices transform those spaces. Determinant detects collapse. Rank counts surviving directions. Nullity counts lost directions. Rank decides the number of solutions. Eigenvectors reveal stable directions. Diagonalization simplifies repeated transformations. Orthogonality gives independent perpendicular directions. Projections find shadows and best approximations. SVD explains every matrix using rotations and stretches. PCA uses eigenvectors or SVD to preserve maximum data variance.

The most important master rule:

$$\det(A)\ne0\iff A^{-1}\text{ exists}\iff\operatorname{rank}(A)=n\iff N(A)=\{0\}\iff0\text{ is not an eigenvalue}$$

Once this chain and the solution-count table become automatic, most GATE linear-algebra questions become recognition problems rather than long mathematical calculations.
