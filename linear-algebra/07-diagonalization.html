Diagonalization

Rewrite a matrix in its eigenvector world, where it acts as simple stretching. Hard jobs like high powers become one-liners.

2 Why this topic exists

In ordinary coordinates a matrix mixes directions together, which makes powers and behaviour hard to see. If we switch to the matrix's own eigenvector directions, it becomes a plain diagonal stretch. That's diagonalization — and it turns $A^{100}$ into raising numbers to a power.

3 Game / visual imagination
🎮 Imagine it

$P^{-1}$ rotates you into the eigenvector world, $D$ just stretches each axis by its eigenvalue, and $P$ rotates you back. Hard work in the normal world becomes easy work in the right world: $A=PDP^{-1}$.

4 Basic definitions
  • Diagonalizable: $A$ can be written $A=PDP^{-1}$, with $D$ diagonal.
  • $P$: columns are the eigenvectors. $D$: diagonal entries are the matching eigenvalues.
  • Defective: a matrix that cannot be diagonalized (too few independent eigenvectors).
  • Algebraic vs geometric multiplicity: how many times $\lambda$ repeats vs how many independent eigenvectors it has; diagonalizable needs them equal for every $\lambda$.
📖 Study in Depth — diagonalization & Cayley–Hamilton from zero

Diagonalization rewrites a matrix in its own eigenvector coordinates as A = PDP⁻¹, turning a tangled transformation into simple independent stretches — which makes powers, inverses and PCA easy. Whether it is possible comes down to having enough independent eigenvectors (GM = AM). Cayley–Hamilton is the companion shortcut. Open each card in order.

1. What is diagonalization?

A square matrix A is diagonalizable if it can be written

A = P D P⁻¹

D = diagonal matrix of eigenvalues
P = matrix whose columns are the matching eigenvectors

If Av₁=λ₁v₁, Av₂=λ₂v₂, … then P = [v₁ v₂ …] and D = diag(λ₁, λ₂, …).

2. Why do we need it? (easy powers)

Diagonal matrices are trivial to power:

D = [2 0; 0 3]  →  D¹⁰ = [2¹⁰ 0; 0 3¹⁰] = [1024 0; 0 59049]

Diagonalization gives the same ease for any matrix: Aᵏ = P Dᵏ P⁻¹.

3. Game imagination

In normal x–y coordinates a matrix looks tangled. Rotate to coordinates aligned with the eigenvectors, and it becomes pure independent stretching:

direction 1 stretched by λ₁
direction 2 stretched by λ₂   (no mixing, no twisting)

Diagonalization = choosing the coordinate system where A is simplest.

4. Meaning of P, D and P⁻¹ 
A = P D P⁻¹

P⁻¹ → convert into eigenvector coordinates
D   → scale each eigen-direction by its eigenvalue
P   → convert back to normal coordinates

Read right to left: go into the eigenbasis, scale, come back.

5. Why D is diagonal (the derivation) 
P = [v₁ v₂]
AP = [Av₁ Av₂] = [λ₁v₁ λ₂v₂] = P·[λ₁ 0; 0 λ₂] = PD

so  AP = PD   ⇒   A = P D P⁻¹

That single line is the whole engine of diagonalization.

6. When can a matrix be diagonalized? 
A (n×n) is diagonalizable  ⟺  A has n linearly independent eigenvectors

Because P must be invertible, and P's columns are the eigenvectors — they must be independent for P⁻¹ to exist.

7. Rule 1: distinct eigenvalues ⇒ diagonalizable 
3×3 with eigenvalues 1, 2, 5  → all distinct → diagonalizable

Eigenvectors for distinct eigenvalues are automatically independent, so n distinct eigenvalues give n independent eigenvectors.

8. Rule 2: real symmetric ⇒ orthogonally diagonalizable

If A = Aᵀ (real), then A is always diagonalizable, and even orthogonally:

A = Q D Qᵀ      (Q orthogonal: Q⁻¹ = Qᵀ)

This is the spectral theorem — the backbone of PCA.

9. Algebraic multiplicity (AM)

AM = how many times an eigenvalue is a root of the characteristic polynomial.

p(λ) = (λ − 2)²(λ − 5)   →   AM(2) = 2,  AM(5) = 1
10. Geometric multiplicity (GM)

GM = number of independent eigenvectors for that eigenvalue = dimension of its eigenspace.

GM(λ) = dim Null(A − λI) = n − rank(A − λI)
11. The AM / GM rule 
1 ≤ GM ≤ AM   (every eigenvalue)

A is diagonalizable  ⟺  GM = AM for EVERY eigenvalue
                     ⟺  total independent eigenvectors = n
12. Repeated eigenvalue — must check GM 
char poly = (λ − 2)²  →  λ = 2, AM = 2

GM = 2 → 2 independent eigenvectors → diagonalizable
GM = 1 → only 1 eigenvector        → NOT diagonalizable

A repeated eigenvalue does not automatically block diagonalization — check GM.

13. Example: identity (repeated but fine) 
I = [1 0; 0 1]   char poly (λ−1)²,  λ = 1, AM = 2
Ix = x for every x → eigenspace is all of ℝ² → GM = 2

GM = AM = 2, so the identity is diagonalizable (it already is diagonal).

14. Example: a defective matrix 
A = [1 1; 0 1]   λ = 1, AM = 2
A − I = [0 1; 0 0]  → x₂ = 0 → eigenvectors x₁·[1;0] only → GM = 1

GM (1) < AM (2) → not diagonalizable. Such matrices are called defective.

15. Full worked diagonalization 
A = [4 1; 2 3]   →  λ = 5 (v = [1;1]),  λ = 2 (v = [1;−2])

P = [1  1     D = [5  0
     1 −2]         0  2]

Then A = P D P⁻¹.

16. Order-matching warning 
P = [v₁ v₂]  with v₁↔λ₁, v₂↔λ₂   ⇒   D = [λ₁ 0; 0 λ₂]

The eigenvector order in P must match the eigenvalue order in D — mismatching gives a wrong result.

17. Computing P⁻¹ (2×2) 
P = [1  1;  1 −2]   det(P) = 1(−2) − 1(1) = −3

P⁻¹ = (1/−3)·[−2 −1; −1  1] = [ 2/3  1/3
                                1/3 −1/3]
18. Why diagonalization makes powers easy 
A² = (PDP⁻¹)(PDP⁻¹) = PD(P⁻¹P)DP⁻¹ = P D² P⁻¹

⇒  Aᵏ = P Dᵏ P⁻¹

The inner P⁻¹P = I collapses everything to a clean diagonal power.

19. Computing Dᵏ 
D = diag(λ₁, λ₂, λ₃)  →  Dᵏ = diag(λ₁ᵏ, λ₂ᵏ, λ₃ᵏ)
D = [5 0; 0 2]  →  D¹⁰ = [5¹⁰ 0; 0 2¹⁰]

Just raise each diagonal entry to the power.

20. Powers — diagonal vs non-diagonal 
already diagonal:  [2 0; 0 3]¹⁰ = [1024 0; 0 59049]

non-diagonal A = PDP⁻¹:  A¹⁰ = P·[5¹⁰ 0; 0 2¹⁰]·P⁻¹

GATE loves high powers (A⁵, A¹⁰, A¹⁰⁰) — diagonalization (or Cayley–Hamilton) is how you do them.

21. The transformation meaning 
any x = c₁v₁ + c₂v₂
Ax = c₁·Av₁ + c₂·Av₂ = c₁λ₁v₁ + c₂λ₂v₂

So A simply scales each eigen-direction independently — that is the deep meaning of being diagonalizable.

22. Symmetric & orthogonal diagonalization 
real symmetric A = A ᵀ  →  A = Q D Qᵀ   (QᵀQ = I)

Q columns = orthonormal eigenvectors,  D = eigenvalues

Called orthogonal diagonalization / spectral decomposition; the inverse is free (Q⁻¹ = Qᵀ).

23. PCA connection 
covariance matrix Σ (symmetric)  →  Σ = Q D Qᵀ
Q = principal directions ,  D = variances

Big eigenvalue = important direction (much spread); small = less important. (Full PCA in the SVD & PCA topic.)

24. Diagonalizable ≠ invertible 
A = [1 0; 0 0]   diagonal → diagonalizable
but det(A) = 0 (eigenvalue 0) → NOT invertible

Diagonalizable = enough independent eigenvectors. Invertible = all eigenvalues non-zero. Different things!

25. Diagonalizable vs distinct eigenvalues 
distinct eigenvalues  ⇒  diagonalizable     (TRUE)
diagonalizable        ⇒  distinct eigenvalues (FALSE — e.g. the identity)

The implication only goes one way.

26. Cayley–Hamilton theorem

Every square matrix satisfies its own characteristic equation.

p(λ) = λ² − 5λ + 4   ⇒   p(A) = A² − 5A + 4I = 0

Trap: the constant becomes 4I, not 4 — replace 1 with I.

27. Cayley–Hamilton — worked example 
A = [3 1; 2 2]   tr = 5, det = 4
p(λ) = λ² − 5λ + 4   ⇒   A² − 5A + 4I = 0

⇒  A² = 5A − 4I

This rewrites A² in terms of A and I — the key to reducing powers.

28. Cayley–Hamilton to find A⁻¹ 
A² − 5A + 4I = 0  ⇒  A(5I − A) = 4I  ⇒  A⁻¹ = (5I − A)/4

General 2×2: if λ² + aλ + b = 0 with b ≠ 0, then A⁻¹ = −(A + aI)/b.

29. Cayley–Hamilton for higher powers 
A² = 5A − 4I
A³ = A·A² = A(5A − 4I) = 5A² − 4A = 5(5A − 4I) − 4A = 21A − 20I

Any power of a 2×2 reduces to the form αA + βI — no repeated multiplication needed.

30. Diagonalization vs Cayley–Hamilton 
Diagonalization:  best when eigenvectors are easy; gives Aᵏ = PDᵏP⁻¹
Cayley–Hamilton:  best when you only need the char. polynomial,
                  to reduce powers or find A⁻¹ (eigenvectors not needed)
31. Matrix exponential (preview) 
A = PDP⁻¹  ⇒  e^A = P e^D P⁻¹ ,   e^D = diag(e^{λ₁}, e^{λ₂}, …)

Used for dynamical systems and differential equations; appears rarely in GATE DA but the idea reuses diagonalization.

32. trace, det & similar matrices 
A = PDP⁻¹  ⇒  trace(A) = trace(D) = Σλ ,  det(A) = det(D) = Πλ

Similar matrices (B = P⁻¹AP) share eigenvalues, trace, determinant, rank and characteristic polynomial. Diagonalization = A is similar to a diagonal matrix.

33. How to check diagonalizability (steps)
  1. Find eigenvalues from det(A − λI) = 0.
  2. All eigenvalues distinct? → diagonalizable, stop.
  3. Repeated λ? compute GM = nullity(A − λI) for each.
  4. GM = AM for all → diagonalizable; otherwise not.

Shortcut: real symmetric → diagonalizable automatically.

34. Common facts & traps
  • n distinct eigenvalues ⇒ diagonalizable. True.
  • Diagonalizable ⇒ distinct eigenvalues? False (identity).
  • Real symmetric ⇒ diagonalizable (orthogonally). True.
  • Every matrix is diagonalizable? False (defective ones).
  • Diagonalizable ⇒ invertible? False (zero eigenvalue allowed).
  • In A = PDP⁻¹, P holds eigenvectors, D holds eigenvalues.
  • Cayley–Hamilton: constant term becomes ·I (e.g. A²−5A+4I=0).
35. Practice problems

Q1. 3×3 with eigenvalues 1, 2, 3 — diagonalizable?

distinct → yes

Q2. 2×2, (λ−4)², GM(4)=1 — diagonalizable?

GM 1 < AM 2 → no

Q3. Identity I₂ — diagonalizable?

GM = AM = 2 → yes

Q4. A = PDP⁻¹, D = [2 0; 0 5] — A¹⁰?

A¹⁰ = P·[2¹⁰ 0; 0 5¹⁰]·P⁻¹

Q5. p(λ)=λ²−7λ+10 — express A²?

A² − 7A + 10I = 0 → A² = 7A − 10I

Q6. A²−5A+6I=0 — find A⁻¹?

A(5I−A)=6I → A⁻¹ = (5I − A)/6
36. Cheatsheet
  • A = PDP⁻¹: P = eigenvectors (order matters!), D = eigenvalues; Aᵏ = PDᵏP⁻¹.
  • Diagonalizable ⟺ n independent eigenvectors ⟺ GM = AM for every λ.
  • n distinct eigenvalues ⇒ diagonalizable; real symmetric ⇒ A = QDQᵀ.
  • Diagonalizable ≠ invertible; trace = Σλ, det = Πλ.
  • Cayley–Hamilton: p(A) = 0 (constant → I) → reduce powers & get A⁻¹.
37. Best mental model 
A = complicated machine
eigenvectors = its natural directions
eigenvalues = stretch factors there
diagonalization = switch to those directions, where A = D (simple)

A = PDP⁻¹ = go to the eigenbasis, scale, come back; Aᵏ = PDᵏP⁻¹ = a hard power becomes an easy diagonal power.

6 Main formulas
$$ A = PDP^{-1},\qquad D=\operatorname{diag}(\lambda_1,\dots,\lambda_n) $$
$$ A^{k} = P D^{k} P^{-1},\qquad D^{k}=\operatorname{diag}(\lambda_1^{k},\dots,\lambda_n^{k}) $$
7 Step-by-step method
  1. Find the eigenvalues from $\det(A-\lambda I)=0$.
  2. Find an eigenvector for each eigenvalue.
  3. Check you have $n$ independent eigenvectors; if not, $A$ is not diagonalizable.
  4. Put the eigenvectors as columns of $P$ and the eigenvalues on the diagonal of $D$.
  5. Then $A=PDP^{-1}$; for powers use $A^k=PD^kP^{-1}$.
8 Worked example
✦ Powers via diagonalization

$A=\begin{bmatrix}2&0\\0&3\end{bmatrix}$ is already diagonal, so $A^{10}=\begin{bmatrix}2^{10}&0\\0&3^{10}\end{bmatrix}=\begin{bmatrix}1024&0\\0&59049\end{bmatrix}$.

For a general diagonalizable $A$: change to the eigenbasis ($P^{-1}$), raise the eigenvalues to the power ($D^k$), change back ($P$).

9 GATE cheat sheet
⚡ High-value facts
  • $n$ distinct eigenvalues ⇒ automatically diagonalizable.
  • Every symmetric matrix is diagonalizable — orthogonally: $A=QDQ^{\mathsf T}$.
  • Diagonalizable $\iff$ $n$ independent eigenvectors (geometric = algebraic multiplicity for each $\lambda$).
  • $A^k=PD^kP^{-1}$ — the reason diagonalization is worth it.
10 Common traps
✕ Watch out

Trap: "Every matrix is diagonalizable." Reality: $\begin{bmatrix}1&1\\0&1\end{bmatrix}$ has $\lambda=1$ twice but only one eigenvector — defective.

Trap: "Repeated eigenvalues ⇒ not diagonalizable." Reality: fine if enough independent eigenvectors (e.g. $I$).

Trap: "$D$ can hold eigenvalues in any order." Reality: order of $D$ must match the column order of $P$.

12 Practice problems

Q1. A $3\times3$ matrix has eigenvalues $2,4,7$ (all distinct). Diagonalizable?

Yes — distinct eigenvalues guarantee it.

Q2. If $A=PDP^{-1}$ with $D=\operatorname{diag}(1,2)$, what is $A^5$ in factored form?

$A^5=P\operatorname{diag}(1^5,2^5)P^{-1}=P\operatorname{diag}(1,32)P^{-1}$.
