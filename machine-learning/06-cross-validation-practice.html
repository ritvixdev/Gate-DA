Cross-Validation

8 questions. Tap any option — you'll see why it's right or wrong, and the correct one lights up.

Q1. In 10-fold CV, the model is trained…
Once is a single split, not CV.
Not twice.
✅ Correct — once per fold, with each fold held out in turn.
That's $k^2$, not $k$.
Q2. Train acc 99%, validation 70% indicates…
Underfitting shows both low.
✅ Correct — memorized training, poor generalization.
A big gap is the opposite of good.
The gap itself signals overfitting.
Q3. LOOCV uses $k=$…
That's a single hold-out.
That's 5-fold.
✅ Correct — leave-one-out means $k=n$ folds.
Not the LOOCV definition.
Q4. You should choose hyperparameters using…
Tuning on test leaks information.
Training accuracy rewards overfitting.
✅ Correct — use CV / a validation set, keep the test set untouched.
Not a method.
Q5. Both train and validation accuracy low ⇒…
Overfitting has high train, low validation.
✅ Correct — model too simple to capture the pattern.
Both low is the opposite of perfect.
Leakage usually inflates scores.
Q6. In k-fold CV, each point is used for validation…
Every point is validated.
✅ Correct — once (and used for training in the other folds).
It's validated once, trained on $k-1$ times.
Once for validation.
Q7. The CV score is the…
Using the best is optimistic.
Using the worst is pessimistic.
✅ Correct — average the validation scores across folds.
Not the training score.
Q8. CV is commonly used to choose…
Not for the seed.
✅ Correct — CV tunes hyperparameters reliably.
The dataset is given.
Labels come with the data.
