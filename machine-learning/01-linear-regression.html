Linear Regression

Draw the best straight line through a cloud of points тАФ then use it to predict. It's the simplest model in ML, and the gateway to everything else.

2 Why this topic exists

We often want to predict a number тАФ a price, a score тАФ from some inputs. Linear regression assumes the output is roughly a weighted sum of the inputs and finds the best weights, giving a simple, interpretable predictor and a baseline every ML course starts with.

3 Game / visual imagination
ЁЯОо Imagine it

Scatter your data as dots. Simple regression lays a single straight line through the cloud so it's as close to the dots as possible. Multiple regression (two inputs) tilts a flat plane through 3-D points; with more inputs it's a hyperplane you can't draw but the idea is the same.

4 Basic definitions
  • Feature / predictor $x$: an input. Target $y$: the number we predict.
  • Simple regression: one feature. Multiple regression: several features $x_1,\dots,x_p$.
  • Coefficients (weights) $\beta_j$: how much each feature contributes. Intercept $\beta_0$: the baseline value.
  • Prediction $\hat{y}$: the model's output. Residual: the error $y-\hat{y}$ for a data point.
ЁЯУЦ Study in Depth тАФ supervised learning, predictive models, linear regression & error metrics
1. What supervised learning is

The model learns from labeled data тАФ examples where the answer is already given:

study hours = 5  тЖТ marks = 75
email text       тЖТ spam / not spam
house = 1000 sqft тЖТ price = тВ╣50 lakh

After seeing many such examples it learns the pattern, then predicts the answer for a new input. Linear regression (this topic) is the simplest supervised algorithm.

2. Labeled data = features (X) + target (Y) 
X = inputs / features   (height, weight, age, pixels, text тАж)
Y = output / label / target   (price, spam?, cat/dog тАж)

Every training row carries both: the question (X) and its correct answer (Y). No labels тЗТ it isn't supervised learning.

3. TeacherтАУstudent mental model

"Supervised" = there is a teacher. A teacher shows a child many labelled pictures (cat, dog, car); the child learns, then names a new picture correctly.

training data = teacher's examples ┬╖ labels = correct answers
model = student ┬╖ algorithm = the learning process ┬╖ prediction = the child's answer
4. The goal: learn a function f(X) тЙИ Y 
X  тЖТ  f (model)  тЖТ  ┼╢  (prediction)
goal: f(X) тЙИ Y  on new, unseen data
f(study hours = 5) тЖТ 75 marks

Training picks the function that best reproduces the known answers and generalizes beyond them.

5. The supervised-learning workflow 
1. collect labeled data
2. split into training + test sets
3. choose an algorithm
4. train the model
5. evaluate on the test set
6. predict on new data

e.g. past house data тЖТ linear regression тЖТ price model тЖТ predict a new house. (Why the train/test split matters тАФ generalization, overfitting тАФ is covered in the Cross-Validation topic.)

6. Regression vs classification

Linear regression is the canonical regression method; logistic regression (later topic) handles classification.

7. Error & the loss function

During training the model predicts, and we compare with the truth:

actual = 80, predicted = 75 тЖТ error = 5

A loss function scores the total error; training tries to minimize it.

regression loss: MSE, MAE   ┬╖   classification loss: cross-entropy
8. Training = optimization

Training is an optimization problem: find the parameters that minimize the loss.

linear model: ┼╖ = w┬╖x + b      (w = weight/slope, b = bias/intercept)
methods: least squares (closed form) ┬╖ gradient descent (iterative)

So "learning" literally means searching parameter space for the lowest-loss model.

10. Why machine learning at all?

Use ML when rules are hard to hand-write but data is plentiful. You can't easily code "if ears triangular and whiskersтАж then cat", but a model can learn it from thousands of labelled images.

complex pattern + lots of labelled data тЖТ let the model learn the rule
11. Connections тАФ linear algebra & probability

Data is stored as matrices/vectors: X = feature matrix (rows = examples, columns = features), y = target vector. So least squares, projections and matrix multiply all reappear here.

Classifiers often output probabilities: P(spam | email) = 0.92 тАФ logistic regression, Naive Bayes and neural nets all do this.

18. Simple linear regression тАФ ┼╖ = ╬▓тВА + ╬▓тВБx

Assume an approximately straight-line relation between one feature and the target:

┼╖ = ╬▓тВА + ╬▓тВБx        (╬▓тВА intercept, ╬▓тВБ slope)   also written ┼╖ = b + wx

19. What slope & intercept mean 
slope ╬▓тВБ     = change in ┼╖ per +1 in x   (rate of change)

20. Multiple linear regression

Several features, one weight each:

┼╖ = ╬▓тВА + ╬▓тВБxтВБ + ╬▓тВВxтВВ + тАж + ╬▓тВЩxтВЩ

Categorical inputs are encoded first (smoker: yes=1, no=0).

22. Scatter plot тАФ reading the relationship

Plot each example as a point (x vs y). The shape reveals the relationship:

23. Many lines тАФ which is best?

Infinitely many lines fit a scatter; the best one has the smallest overall error.

data (1,30)(2,40)(3,50)
A: ┼╖=20+10x тЖТ 30,40,50  errors 0,0,0
B: ┼╖=25+5x  тЖТ 30,35,40  errors 0,5,10  тЖТ  A is better
24. Why square the errors?

Raw residuals cancel: +5 + (тИТ5) = 0 hides two real mistakes. Squaring fixes both problems:

(+5)┬▓ = 25 , (тИТ5)┬▓ = 25 тЖТ both count; large errors are punished more

So regression minimizes the sum of squared residuals, not the raw sum.

25. RSS тАФ residual sum of squares 
RSS = ╬г (yс╡в тИТ ┼╖с╡в)┬▓     (total squared prediction error; lower = better fit)
y=10,20,30 ; ┼╖=12,18,33 тЖТ errors тИТ2,2,тИТ3 тЖТ squares 4,4,9 тЖТ RSS = 17

Least squares (next topic) chooses the parameters that minimize RSS.

26. MSE тАФ mean squared error 
MSE = RSS / n = (1/n) ╬г (yс╡в тИТ ┼╖с╡в)┬▓
RSS=17, n=3 тЖТ MSE тЙИ 5.67   (units = target┬▓, harder to read)
27. RMSE тАФ root mean squared error 
RMSE = тИЪMSE
MSE=5.67 тЖТ RMSE тЙИ 2.38  тЖТ "typical error тЙИ 2.38 units"

RMSE is in the same unit as the target (тВ╣, marksтАж), so it's the readable one. MSE is in squared units.

28. Normalized RMSE

RMSE depends on scale, so normalize it to compare across datasets:

NRMSE = RMSE / (y_max тИТ y_min)      (or RMSE / mean(y))
RMSE=5, range 0тАУ100 тЖТ NRMSE = 0.05 = 5%
29. TSS & R┬▓ тАФ variance explained 
TSS = ╬г (yс╡в тИТ ╚│)┬▓      (total variation around the mean)
R┬▓  = 1 тИТ RSS/TSS      (fraction of variance the model explains)
RSS=20, TSS=100 тЖТ R┬▓ = 1 тИТ 0.2 = 0.8  тЖТ explains 80%
  • R┬▓ = 1 perfect ┬╖ R┬▓ = 0 no better than predicting ╚│ ┬╖ R┬▓ < 0 worse than the mean

6 Main formulas
$$ \textbf{Simple: } \hat{y}=\beta_0+\beta_1 x \qquad \textbf{Multiple: } \hat{y}=\beta_0+\beta_1 x_1+\dots+\beta_p x_p $$

The least-squares slope and intercept (simple regression):

$$ \beta_1=\frac{\sum (x_i-\bar{x})(y_i-\bar{y})}{\sum (x_i-\bar{x})^2}, \qquad \beta_0=\bar{y}-\beta_1\bar{x} $$

Error & evaluation metrics:

$$ \text{RSS}=\sum (y_i-\hat{y}_i)^2,\quad \text{MSE}=\tfrac{\text{RSS}}{n},\quad \text{RMSE}=\sqrt{\text{MSE}},\quad \text{TSS}=\sum (y_i-\bar{y})^2,\quad R^2=1-\tfrac{\text{RSS}}{\text{TSS}} $$

RMSE is in the target's own units; $R^2\in(-\infty,1]$ ($1$ perfect, $0$ = as good as the mean, $<0$ worse).

7 Step-by-step method
  1. Compute the means $\bar{x}$ and $\bar{y}$.
  2. Slope $\beta_1=\dfrac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sum(x_i-\bar{x})^2}$.
  3. Intercept $\beta_0=\bar{y}-\beta_1\bar{x}$.
  4. Predict new points with $\hat{y}=\beta_0+\beta_1 x$.
8 Worked example
тЬж Fit a line through (1,2), (2,4), (3,5), (4,4), (5,5)

$\bar{x}=3,\ \bar{y}=4$.   $\sum(x-\bar{x})(y-\bar{y})=(-2)(-2)+(-1)(0)+0+1\cdot0+2\cdot1=6$.

$\sum(x-\bar{x})^2=4+1+0+1+4=10$. So $\beta_1=6/10=0.6$, $\beta_0=4-0.6(3)=2.2$.

Line: $\hat{y}=2.2+0.6x$. Predict $x=6$: $\hat{y}=5.8$.

9 GATE cheat sheet
тЪб High-value facts
  • The least-squares line always passes through $(\bar{x},\bar{y})$.
  • Slope $\beta_1=\operatorname{Cov}(x,y)/\operatorname{Var}(x)=r\cdot s_y/s_x$.
  • $R^2$ = fraction of variance explained (1 = perfect fit, 0 = no better than the mean).
  • Multiple regression in matrix form: $\hat{y}=X\beta$; "linear" means linear in the coefficients.
  • Adding features never increases training error (but can overfit).
10 Common traps
тЬХ Watch out

Trap: "Correlation implies causation." Reality: a good fit doesn't prove $x$ causes $y$.

Trap: "Predict far outside the data range." Reality: extrapolation is unreliable.

Trap: "Linear means a straight feature relationship." Reality: it means linear in $\beta$; you can use $x^2$ as a feature and still call it linear regression.
