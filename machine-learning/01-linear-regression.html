Linear Regression

Draw the best straight line through a cloud of points — then use it to predict. It's the simplest model in ML, and the gateway to everything else.

2 Why this topic exists

We often want to predict a number — a price, a score — from some inputs. Linear regression assumes the output is roughly a weighted sum of the inputs and finds the best weights, giving a simple, interpretable predictor and a baseline every ML course starts with.

3 Game / visual imagination
🎮 Imagine it

Scatter your data as dots. Simple regression lays a single straight line through the cloud so it's as close to the dots as possible. Multiple regression (two inputs) tilts a flat plane through 3-D points; with more inputs it's a hyperplane you can't draw but the idea is the same.

4 Basic definitions
  • Feature / predictor $x$: an input. Target $y$: the number we predict.
  • Simple regression: one feature. Multiple regression: several features $x_1,\dots,x_p$.
  • Coefficients (weights) $\beta_j$: how much each feature contributes. Intercept $\beta_0$: the baseline value.
  • Prediction $\hat{y}$: the model's output. Residual: the error $y-\hat{y}$ for a data point.

6 Main formulas
$$ \textbf{Simple: } \hat{y}=\beta_0+\beta_1 x \qquad \textbf{Multiple: } \hat{y}=\beta_0+\beta_1 x_1+\dots+\beta_p x_p $$

The least-squares slope and intercept (simple regression):

$$ \beta_1=\frac{\sum (x_i-\bar{x})(y_i-\bar{y})}{\sum (x_i-\bar{x})^2}, \qquad \beta_0=\bar{y}-\beta_1\bar{x} $$
7 Step-by-step method
  1. Compute the means $\bar{x}$ and $\bar{y}$.
  2. Slope $\beta_1=\dfrac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sum(x_i-\bar{x})^2}$.
  3. Intercept $\beta_0=\bar{y}-\beta_1\bar{x}$.
  4. Predict new points with $\hat{y}=\beta_0+\beta_1 x$.
8 Worked example
✦ Fit a line through (1,2), (2,4), (3,5), (4,4), (5,5)

$\bar{x}=3,\ \bar{y}=4$.   $\sum(x-\bar{x})(y-\bar{y})=(-2)(-2)+(-1)(0)+0+1\cdot0+2\cdot1=6$.

$\sum(x-\bar{x})^2=4+1+0+1+4=10$. So $\beta_1=6/10=0.6$, $\beta_0=4-0.6(3)=2.2$.

Line: $\hat{y}=2.2+0.6x$. Predict $x=6$: $\hat{y}=5.8$.

9 GATE cheat sheet
⚡ High-value facts
  • The least-squares line always passes through $(\bar{x},\bar{y})$.
  • Slope $\beta_1=\operatorname{Cov}(x,y)/\operatorname{Var}(x)=r\cdot s_y/s_x$.
  • $R^2$ = fraction of variance explained (1 = perfect fit, 0 = no better than the mean).
  • Multiple regression in matrix form: $\hat{y}=X\beta$; "linear" means linear in the coefficients.
  • Adding features never increases training error (but can overfit).
10 Common traps
✕ Watch out

Trap: "Correlation implies causation." Reality: a good fit doesn't prove $x$ causes $y$.

Trap: "Predict far outside the data range." Reality: extrapolation is unreliable.

Trap: "Linear means a straight feature relationship." Reality: it means linear in $\beta$; you can use $x^2$ as a feature and still call it linear regression.

11 Connections with other topics
  • Least-squares: the method that finds these coefficients (next topic).
  • Optimization: fitting = minimizing the sum of squared residuals.
  • Linear algebra: multiple regression solves the normal equations $X^{\mathsf T}X\beta=X^{\mathsf T}y$.
  • Covariance: the slope is $\operatorname{Cov}(x,y)/\operatorname{Var}(x)$.
12 Practice problems

Q1. The least-squares line always passes through which point?

$(\bar{x},\bar{y})$ — the means.

Q2. $\operatorname{Cov}(x,y)=6$, $\operatorname{Var}(x)=10$. Slope $\beta_1$?

$\beta_1=6/10=0.6$.

Q3. If $\bar{x}=3,\bar{y}=4,\beta_1=0.6$, find the intercept.

$\beta_0=\bar{y}-\beta_1\bar{x}=4-1.8=2.2$.

Quick MCQ

In linear regression, "linear" means the model is linear in the…
You can transform features (e.g. $x^2$) and it's still linear regression.
✅ Correct — linear in the $\beta$'s; that's what makes least-squares work.
The target is what we predict, not the linearity condition.
Residuals are the errors, not the linearity.

  • $\beta_0=\bar{y}-\beta_1\bar{x}$.
  • $R^2$ = variance explained; linear means linear in $\beta$.
