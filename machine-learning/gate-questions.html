GATE DA 2024 k-means clustering
Q1. Euclidean-distance $k$-means with $k=3$ is run on 100 points. If $\begin{bmatrix}1\\1\end{bmatrix}$ and $\begin{bmatrix}-1\\1\end{bmatrix}$ are both in cluster 3, which ONE of the following points is necessarily also in cluster 3?
$(0,0)$ need not be closest to centroid 3 — it isn't forced.
$(0,2)$ lies outside the segment joining the two given points.
$(2,0)$ is not forced into cluster 3.
✓ Correct — a $k$-means cluster region is convex. The midpoint $\begin{bmatrix}0\\1\end{bmatrix}$ of two points in cluster 3 lies in that convex region, so it must also belong to cluster 3.
