Q1. Logistic regression is used for…
Despite the name, it classifies.
✅ Correct — it predicts class probabilities.
Clustering is unsupervised (k-means).

Q2. $\sigma(0)=?$
$\sigma(z)\to0$ as $z\to-\infty$, not at 0.
$\sigma(z)\to1$ as $z\to+\infty$.
✅ Correct — $1/(1+1)=0.5$.
Sigmoid is bounded in $(0,1)$.
Q3. The decision boundary is where…
That's deep inside one class.
The boundary is at $z=0$.
✅ Correct — $z=0$ gives $\hat{p}=0.5$.
Not tied to $x=0$.
Q4. Logistic regression minimizes…
That's linear regression.
Inertia is for k-means.
✅ Correct — it minimizes log-loss.
Unrelated.
Q5. The output of logistic regression is…
That's $z$ before the sigmoid.
That's k-means.
✅ Correct — the sigmoid gives a probability.
Not a distance.
Q6. The model assumes the log-odds are…
It's linear, not quadratic.
✅ Correct — $\log\frac{p}{1-p}=z=\beta_0+\beta^{\mathsf T}x$.
It varies with $x$.
It's a fitted linear function.
Q7. If $z=2$, the prediction (threshold 0.5) is…
$z\gt0$ ⇒ $\hat{p}\gt0.5$.
✅ Correct — $\sigma(2)\approx0.88\ge0.5$ → class 1.
It's well-defined.
Not a tie.
Q8. The decision boundary in raw features is…
Not a circle for plain logistic regression.
It's straight where $z=0$.
✅ Correct — $z=0$ is a linear boundary.
There is one, at $z=0$.
