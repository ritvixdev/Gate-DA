Least-Squares

8 questions. Tap any option — you'll see why it's right or wrong, and the correct one lights up.

Q1. Least-squares minimizes…
That's least absolute deviations, a different method.
✅ Correct — it minimizes $\sum(y_i-\hat{y}_i)^2$.
That's the minimax criterion.
Unrelated to the objective.
Q2. The normal-equations solution is…
Dimensions don't match.
$X$ isn't square / invertible in general.
✅ Correct — solve $X^{\mathsf T}X\beta=X^{\mathsf T}y$.
Missing the $(X^{\mathsf T}X)^{-1}$ factor.
Q3. Why is the least-squares solution unique?
The cost is quadratic (convex), not linear.
It's about the cost shape, not the data count.
✅ Correct — a convex bowl has a single global minimum.
It is unique when $X^{\mathsf T}X$ is invertible.
Q4. OLS uses which residual distance?
Perpendicular distance is total least squares, not OLS.
✅ Correct — ordinary least squares uses vertical residuals in $y$.
No — it's the error in $y$, not $x$.
Not a thing here.
Q5. $(X^{\mathsf T}X)^{-1}$ fails to exist when…
Noise doesn't make it singular.
More data usually helps.
✅ Correct — multicollinearity makes $X^{\mathsf T}X$ singular.
It can fail (collinear features).
Q6. At the OLS optimum, the residuals…
They have mixed signs.
Only for a perfect fit.
✅ Correct — with an intercept, the residuals sum to zero.
They're minimized (squared), not maximized.
Q7. Gradient descent on the SSE reaches…
Same convex problem, same optimum.
✅ Correct — the SSE is convex, so GD finds the global least-squares solution.
No local traps in a convex bowl.
It converges with a suitable rate.
Q8. SSE is also called…
The gradient is its derivative.
Related but not the same.
✅ Correct — SSE = RSS = $\sum e_i^2$.
Unrelated.
