PMF, PDF &
Distributions

A distribution is the full "menu" of probabilities for a random variable. Discrete ones use a PMF; continuous ones use a PDF.

2 Why this topic exists

Once a random variable has numbers, we need to know how the probability is spread across those numbers. That spread is the distribution — and a handful of named distributions (Binomial, Poisson, Normal…) describe almost everything you'll meet in GATE DA and ML.

3 Game / visual imagination
🎮 Imagine it

PMF = a bar chart: each bar's height is the probability of that exact value (bars sum to 1). CDF = a progress bar: how much probability has piled up from the left up to $x$ — it only rises, from 0 to 1. Play below — switch between a Binomial PMF and a Normal PDF.

4 Basic definitions
  • PMF $p(x)=P(X=x)$: for discrete $X$; each $p(x)\ge0$ and $\sum p(x)=1$.
  • PDF $f(x)$: for continuous $X$; $f(x)\ge0$ and $\int f(x)\,dx=1$. Probability is area: $P(a\le X\le b)=\int_a^b f(x)\,dx$.
  • CDF $F(x)=P(X\le x)$: works for both; non-decreasing, rising from 0 to 1.
  • Density is not probability: for continuous $X$, $P(X=x)=0$ — only intervals carry probability.
  • CDF & PDF link: $f(x)=F'(x)$ (slope of the CDF); $F$ is the running total of $f$.
  • Named distributions: Bernoulli, Binomial, Poisson, Uniform, Normal, Exponential (see the table below).

1. The probability mass function (PMF)

For a discrete random variable, the PMF gives the probability of each exact value:

pₓ(x) = P(X = x)        (also written p(x))
fair die: pₓ(1)=…=pₓ(6)=1/6 ;  pₓ(7)=0 (impossible)

Valid PMF: pₓ(x) ≥ 0 and Σ pₓ(x) = 1. (It is the discrete partner of the continuous PDF below.)

2. PMF examples: Bernoulli

A single yes/no trial gives a Bernoulli random variable.

biased coin, P(H)=0.7: X = 1 if Head, 0 if Tail
pₓ(1) = 0.7 ,  pₓ(0) = 0.3   (sum = 1)
3. PMF via combinations (binomial preview)

Three fair coins, X = number of heads. Count outcomes with combinations:

P(X = x) = 3Cx / 2³
P(X=0)=1/8, P(X=1)=3/8, P(X=2)=3/8, P(X=3)=1/8   (sum = 1)

This nCx / 2ⁿ pattern is the start of the binomial distribution.

4. PMF with an unknown constant

GATE often gives a PMF with a constant — pin it down with Σ pₓ(x) = 1.

pₓ(x) = k·x for x = 1,2,3,4
k(1+2+3+4) = 1 → 10k = 1 → k = 1/10

Then read off pₓ(1)=1/10, …, pₓ(4)=4/10. (Once you have the PMF, expectation/variance use the formulas from Random Variables.)

5. Why continuous random variables exist

Some quantities can't be counted one by one — height, weight, time, temperature, speed. Between any two values lie infinitely many more (170, 170.1, 170.12, …).

So we can't make a table of P(X = each value). Instead we ask for probability over an interval:

P(160 ≤ X ≤ 170) ,  P(X > 180)
6. Game imagination

A random speed boost can be 1.1×, 1.111×, 1.2×, 1.35× — any decimal in a range.

asking P(boost = exactly 1.234567×) is useless
asking P(boost between 1.2× and 1.5×) is meaningful

Memory: discrete = probability on points; continuous = probability over intervals (area).

7. Continuous random variable — definition

A continuous random variable takes any value in an interval (infinitely many decimal values).

X = height of a student ∈ [150, 200] cm
T = run time of a program ∈ (0, ∞) sec

Probability is assigned to ranges, written P(a ≤ X ≤ b).

8. Why P(X = a) = 0 for continuous

With infinitely many possible values, the chance of landing on one exact value is zero:

P(X = exactly 170.000…) = 0     (a single point has zero width)

But an interval has positive probability: P(160 ≤ X ≤ 170) > 0. Probability lives in intervals, not points.

9. The probability density function (PDF)

A continuous RV is described by a PDF f(x). Crucially:

f(x) is NOT a probability — it is a DENSITY.
Probability = AREA under f(x).
P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx     (area under the curve from a to b)
10. Properties of a valid PDF 
1.  f(x) ≥ 0  for all x            (density can't be negative)
2.  ∫₋∞^∞ f(x) dx = 1               (total area = 1)

These mirror the discrete rules p(x) ≥ 0 and Σ p(x) = 1 — sum becomes integral.

12. A PDF value can exceed 1 (trap)

Probability can never exceed 1 — but a density can, because probability is area, not height.

f(x) = 2 on [0, 0.5]:  area = base × height = 0.5 × 2 = 1  ✓ valid
here f(x) = 2 > 1, yet it is a perfectly good PDF
13. Probability over an interval (rectangle area)

For a flat (uniform) PDF, area is just base × height.

f(x) = 1/10 on [0,10]:  P(2 ≤ X ≤ 5) = (5−2) × 1/10 = 3/10 = 0.3
14. Uniform(0,1) 
f(x) = 1 on [0,1], else 0   (total area 1×1 = 1 ✓)
P(0.2 ≤ X ≤ 0.7) = (0.7−0.2) × 1 = 0.5
P(X = 0.5) = 0   (a point)
15. ≤ vs < doesn't matter (continuous)

Because endpoints have probability 0:

P(a ≤ X ≤ b) = P(a < X < b) = P(a < X ≤ b) = P(a ≤ X < b)

(For discrete variables the endpoints do matter — not here.)

16. Finding the constant in a PDF

GATE often gives a PDF with an unknown constant; use total area = 1.

f(x) = c·x on [0,2]:  ∫₀² c x dx = c[x²/2]₀² = 2c = 1  →  c = 1/2

then P(0 ≤ X ≤ 1) = ∫₀¹ (x/2) dx = (1/2)[x²/2]₀¹ = 1/4
17. The cumulative distribution function (CDF)

The CDF is the running total of probability:

F(x) = P(X ≤ x) = ∫₋∞ˣ f(t) dt     (area under the PDF up to x)

It works for discrete and continuous variables alike.

18. PDF ↔ CDF relationship 
F(x) = ∫₋∞ˣ f(t) dt        (PDF → integrate → CDF)
f(x) = d/dx F(x)           (CDF → differentiate → PDF)

The CDF accumulates area; the PDF is its slope.

19. CDF properties
  • 0 ≤ F(x) ≤ 1
  • F is non-decreasing (never goes down)
  • F(−∞) = 0,   F(+∞) = 1
  • P(a < X ≤ b) = F(b) − F(a) (and for continuous, P(a ≤ X ≤ b) = F(b) − F(a))
20. Discrete CDF — the staircase

For a discrete variable the CDF adds up PMF values up to x:

F(x) = P(X ≤ x) = Σ p(t)   for all t ≤ x

Fair die — accumulate 1/6 at a time:

F(x) = 0      x < 1
     = 1/6    1 ≤ x < 2
     = 2/6    2 ≤ x < 3
     = 3/6    3 ≤ x < 4
     = 4/6    4 ≤ x < 5
     = 5/6    5 ≤ x < 6
     = 1      x ≥ 6

The graph is a staircase: flat between values, jumping up at each value it can take.

21. Recover the PMF from a discrete CDF

Each jump in the staircase is the probability at that point:

P(X = k) = F(k) − F(k−1)     (the jump size)
die:  P(X = 3) = F(3) − F(2) = 3/6 − 2/6 = 1/6 ✓

So PMF height = size of the CDF step. (Contrast: for continuous X the CDF has no jumps, so P(X=a)=0.)

22. Discrete interval rule — mind the a−1

For integer-valued X you must drop to a−1, not a:

P(a ≤ X ≤ b) = F(b) − F(a−1)
die:  P(3 ≤ X ≤ 5) = F(5) − F(2) = 5/6 − 2/6 = 3/6 = 1/2

Why F(2) and not F(3)? We must keep the value 3, so we subtract everything up to 2.

Trap: continuous uses F(b)−F(a); discrete integer uses F(b)−F(a−1). Mixing them is a classic GATE mistake.

23. CDF example — Uniform(0,1) 
f(x) = 1 on [0,1]

F(x) = 0       x < 0
     = x       0 ≤ x ≤ 1
     = 1       x > 1

P(0.2 ≤ X ≤ 0.7) = F(0.7) − F(0.2) = 0.5
24. CDF example — from f(x)=x/2 
f(x) = x/2 on [0,2]

F(x) = ∫₀ˣ t/2 dt = x²/4   on [0,2]   (F(2)=1 ✓)

P(1 ≤ X ≤ 2) = F(2) − F(1) = 1 − 1/4 = 3/4
25. Expectation of a continuous RV 
E[X] = ∫₋∞^∞ x · f(x) dx

Same idea as the discrete weighted average Σ x·p(x) — the sum just becomes an integral.

26. Variance of a continuous RV 
E[X²] = ∫ x² f(x) dx
Var(X) = E[X²] − (E[X])²
SD(X)  = √Var(X)

Identical structure to the discrete case, computed with integrals.

27. The continuous Uniform(a,b) 
f(x) = 1/(b−a) on [a,b]
E[X]  = (a + b)/2
Var(X) = (b − a)² / 12
Uniform(0,1): E[X] = 1/2 ,  Var = 1/12
28. Uniform(0,10) — worked 
f(x) = 1/10 on [0,10]
P(3 ≤ X ≤ 7) = (7−3) × 1/10 = 0.4
E[X] = (0+10)/2 = 5
Var(X) = 10²/12 = 25/3
29. Probability via the graph's area

Always read continuous probability as area — match the shape:

rectangle → base × height
triangle  → ½ × base × height
curve     → integral
triangle on [0,2], peak height 1:  area = ½ × 2 × 1 = 1 → valid PDF
30. PDF height is not probability 
f(2) = 0.4  does NOT mean P(X = 2) = 0.4
for continuous X,  P(X = 2) = 0

f(2)=0.4 is the density at 2; probability comes from the area around it, e.g. P(1.9 ≤ X ≤ 2.1) = ∫ f(x) dx.

31. Probability distribution & its two types

A distribution is the rule that says which values a random variable can take and how likely each is.

Discrete   → countable values   → described by a PMF   (Bernoulli, Binomial, Poisson, Geometric)
Continuous → interval of values  → described by a PDF   (Uniform, Exponential, Normal)

The PMF/PDF/CDF tools above are the language; the named distributions below are the common sentences you meet in GATE DA and ML.

32. Bernoulli distribution — one yes/no trial

A single trial with two outcomes: success (1) or failure (0), P(success)=p, q=1−p.

PMF:  P(X=x) = pˣ q¹⁻ˣ ,  x ∈ {0,1}     (x=1 → p ;  x=0 → q)
E(X) = p        Var(X) = pq        SD = √(pq)
p=0.7 → E(X)=0.7 , Var(X)=0.7×0.3=0.21

It is the building block: one Bernoulli trial repeated independently becomes the Binomial.

33. Binomial distribution — counting successes in n trials

Count successes in n independent trials, each with the same p. Written X ~ Binomial(n, p).

Needs: fixed n · two outcomes · independent trials · constant p
PMF:  P(X=r) = ⁿCr · pʳ · qⁿ⁻ʳ

Why: pʳqⁿ⁻ʳ is one arrangement; ⁿCr counts which r of the n trials succeed.

3 fair coins, X = #heads:  P(X=2) = ³C₂(½)²(½)¹ = 3·¼·½ = 3/8
34. Binomial mean & variance 
E(X) = np        Var(X) = npq        SD = √(npq)
n=10, p=0.2, q=0.8 → mean = 2 , variance = 1.6 , SD = √1.6 ≈ 1.265

Used for defect counts, hits-out-of-attempts, responders to a treatment, customers who buy — any fixed number of independent yes/no trials.

35. Poisson distribution — counting rare events

Counts events in a fixed interval of time/space when they are rare, at average rate m (often written λ).

PMF:  P(X=r) = e⁻ᵐ · mʳ / r!
E(X) = m        Var(X) = m        (mean = variance — the Poisson signature)
m=2 calls/min:  P(X=3) = e⁻²·2³/3! = (4/3)e⁻² ≈ 1.333×0.1353 ≈ 0.180

Examples: calls per hour, accidents per day, defects per metre, requests per second.

36. Binomial → Poisson approximation

When trials are many and successes rare, the Binomial collapses to a Poisson:

n large , p small , m = np moderate   ⇒   Binomial(n,p) ≈ Poisson(m=np)

This turns an awkward ⁿCr pʳ qⁿ⁻ʳ with huge n into a quick e⁻ᵐmʳ/r! — the go-to move for rare-event GATE problems.

37. Exponential distribution — waiting time GATE extra

The waiting time between events of a Poisson process, with rate λ>0.

PDF:  f(x) = λe^(−λx) , x > 0
CDF:  F(x) = P(X ≤ x) = 1 − e^(−λx)
Survival:   P(X > x) = e^(−λx)
E(X) = 1/λ        Var(X) = 1/λ²        SD = 1/λ
λ=2 per min:  P(next call within 1 min) = 1 − e⁻² ≈ 0.865

Pair it up: Poisson counts how many events; Exponential measures how long until the next one.

38. Memoryless property (exponential) GATE extra

The exponential forgets how long you've already waited:

P(X > s + t | X > s) = P(X > t)

Already waited 10 min for a bus? The chance of 5 more minutes is the same as a fresh 5-minute wait. The exponential is the only continuous distribution with this property — a favourite GATE one-liner.

39. Normal distribution — the bell curve

Symmetric, bell-shaped, centred at μ. Written X ~ N(μ, σ²).

  • symmetric about μ → Mean = Median = Mode
  • total area = 1; tails extend forever but thin out
  • P(X < μ) = P(X > μ) = 0.5

Models measurement errors, heights, noise, ML residuals, and (via the CLT) sample means — which is why it is everywhere in statistics.

40. Standard normal & the Z-score

The standard normal Z ~ N(0,1) has mean 0, SD 1. Any normal value converts to it:

Z = (X − μ) / σ      (how many SDs the value sits from the mean)
Z > 0 above mean · Z < 0 below mean · Z = 0 at the mean
μ=50, σ=10, X=70 → Z = +2  ;  X=35 → Z = −1.5
Empirical (68–95–99.7) rule:  within 1σ ≈ 68% · 2σ ≈ 95% · 3σ ≈ 99.7%
Z-table:  P(Z≤1)≈0.8413 , P(Z≤2)≈0.9772 , P(Z≤−1)≈0.1587

Trap: the denominator is σ (standard deviation), not σ².

41. Linear transforms of a normal stay normal

Scaling and shifting a normal variable keeps it normal — the linear filtering property:

X ~ N(μ, σ²)  ⇒  aX + b ~ N(aμ + b, a²σ²)
X ~ N(10, 4), Y = 3X + 5 → mean 3(10)+5=35, var 3²(4)=36 → Y ~ N(35, 36)

This is also why the CLT's sample mean (a scaled sum of variables) ends up normal.

42. Moments — what each one describes 
1st moment           → mean      (centre)
2nd central moment   → variance  (spread)
3rd standardized     → skewness  (asymmetry)
4th standardized     → kurtosis  (tail heaviness / peakedness)

For the normal distribution skewness = 0 (perfectly symmetric); kurtosis is the reference baseline that other distributions are compared against.

43. The 68–95–99.7 tail split GATE extra

Turn the empirical rule into one-tail probabilities by splitting the leftover area in two:

within ±1σ = 68%  → outside 32%  → each tail 16%
within ±2σ = 95%  → outside 5%   → each tail 2.5%
within ±3σ = 99.7% → outside 0.3% → each tail 0.15%
μ=100, σ=15: P(X > 130) ≈ 2.5%  (130 = μ+2σ)

Lets you answer many normal questions without the full Z-table.

44. Gaussian = Normal; the Gaussian process (intro) GATE extra

Gaussian distribution = normal = bell curve — just another name, X ~ N(μ, σ²).

A Gaussian process goes one level up: a distribution over functions, not numbers. At any finite set of input points the outputs are jointly multivariate normal, so it predicts a whole curve plus its uncertainty.

Gaussian distribution → random number
Gaussian process      → random function (prediction + uncertainty)

Used in Bayesian ML, regression, time series and spatial prediction.

46. Common GATE traps
  • PDF value = probability? No — area is probability; the value is density.
  • P(X=a) > 0 for continuous? No — it's 0.
  • PDF can't exceed 1? Wrong — only the total area must be 1.
  • P(a≤X≤b) ≠ P(a<X<b)? Equal for continuous (endpoints carry 0).
  • CDF can decrease? No — it's non-decreasing.
  • Continuous probability uses summation? No — integration.
  • Bernoulli = Binomial? No — Bernoulli is one trial; Binomial is n of them.
  • Binomial with changing p? No — p must stay constant across trials.
  • Poisson mean ≠ variance? No — both equal m.
  • Exponential counts events? No — Poisson counts; Exponential is the waiting time.
  • "Normal" always means N(0,1)? No — that's the standard normal; general is N(μ,σ²).
  • Z-score divides by variance? No — by σ (the SD), not σ².
47. Practice problems

Q1. Valid PDF? f(x)=2 on [0,0.5]

area = 0.5×2 = 1, f≥0 → valid

Q2. f(x)=1/5 on [0,5] — P(1≤X≤3)?

(3−1)×1/5 = 2/5

Q3. f(x)=c·x on [0,3] — find c.

c[x²/2]₀³ = 9c/2 = 1 → c = 2/9

Q4. Uniform(0,10) — P(X>7)?

(10−7)×1/10 = 0.3

Q5. F(x)=x²/4 on [0,2] — find the PDF.

f(x) = F'(x) = x/2

Q6. Bernoulli p=0.6 — mean and variance?

E=p=0.6 ,  Var=pq=0.6×0.4=0.24

Q7. Binomial(n=10, p=0.2) — mean and variance?

np=2 ,  npq=10×0.2×0.8=1.6

Q8. Poisson m=2 — P(X=3)?

e⁻²·2³/3! = (4/3)e⁻² ≈ 0.180

Q9. Exponential λ=2 — P(X>1)?

e^(−λx) = e⁻² ≈ 0.135

Q10. N(μ=50, σ=10) — Z-score of X=65?

Z = (65−50)/10 = 1.5
48. Cheatsheet & mental model
  • PDF f(x): f≥0, ∫f=1; P(a≤X≤b)=∫ₐᵇ f dx; P(X=a)=0.
  • CDF F(x)=P(X≤x)=∫₋∞ˣ f; f=F'; P(a≤X≤b)=F(b)−F(a).
  • Continuous: E[X]=∫x f dx, Var=E[X²]−(E[X])².
  • Uniform(a,b): f=1/(b−a), E=(a+b)/2, Var=(b−a)²/12.
  • Bernoulli: E=p, Var=pq. Binomial: P=ⁿCr pʳqⁿ⁻ʳ, E=np, Var=npq.
  • Poisson: P=e⁻ᵐmʳ/r!, E=Var=m. Exponential: f=λe⁻ᵏˣ, E=1/λ, Var=1/λ², memoryless.
  • Normal N(μ,σ²); standardise Z=(X−μ)/σ; 68–95–99.7 within 1/2/3 σ.

Mental model: discrete = coins on exact points; continuous = probability "paint" spread along a line. PDF = the landscape, probability = area under it, CDF = area collected from the left. Distributions: Bernoulli = one trial · Binomial = repeated trials · Poisson = rare counts · Exponential = waiting time · Normal = bell curve · Z = standardise.

6 Main formulas & key distributions

PMF (discrete): $p(x)=P(X=x)$,   $p(x)\ge 0$,   $\sum_x p(x)=1$.

PDF (continuous): $f(x)\ge 0$,   $\int_{-\infty}^{\infty} f(x)\,dx=1$,   $P(a\le X\le b)=\int_a^b f(x)\,dx$.

CDF: $F(x)=P(X\le x)$;   discrete $F(x)=\sum_{t\le x}p(t)$,   continuous $F(x)=\int_{-\infty}^{x} f(t)\,dt$ and $f(x)=F'(x)$.

Interval from CDF: continuous $P(a\le X\le b)=F(b)-F(a)$;   discrete integer $P(a\le X\le b)=F(b)-F(a-1)$.

7 Step-by-step: use a distribution
  1. Identify the situation: fixed trials & success prob → Binomial; rare counts in an interval → Poisson; "bell-shaped" measurement → Normal.
  2. Read off the parameters ($n,p$ / $\lambda$ / $\mu,\sigma$).
  3. Plug into the PMF/PDF for a specific probability, or use mean/variance for summaries.
  4. For Normal, standardize: $Z=(X-\mu)/\sigma$, then use the $Z$-table.
8 Worked example
✦ Binomial

10 fair coin flips, $P(\text{exactly 3 heads})=\binom{10}{3}(0.5)^3(0.5)^7=120\cdot(0.5)^{10}\approx0.117$.

Mean $=np=10(0.5)=5$; variance $=np(1-p)=2.5$.

9 GATE cheat sheet
⚡ High-value facts
  • Binomial: mean $np$, variance $np(1-p)$. Poisson: mean = variance = $\lambda$.
  • Binomial $\to$ Poisson when $n$ large, $p$ small ($\lambda=np$).
  • Normal: ~68% within $\mu\pm\sigma$, ~95% within $\mu\pm2\sigma$, ~99.7% within $\mu\pm3\sigma$.
  • Standardize Normal with $Z=(X-\mu)/\sigma$ to use tables.
  • For continuous $X$, $P(X=x)=0$; height of a PDF can exceed 1 (it's a density, not a probability).
  • Find an unknown constant by forcing total $=1$: set $\sum p(x)=1$ (discrete) or $\int f(x)\,dx=1$ (continuous).
  • Recover the PMF from a CDF: $P(X=k)=F(k)-F(k-1)$ — the jump size of the staircase.
  • Interval probability: continuous $F(b)-F(a)$; integer discrete $F(b)-F(a-1)$ (keep the lower endpoint).
  • CDF is always non-decreasing and runs $0\to1$; for continuous $X$, $\le$ and $<$ give the same probability.
  • Exponential: $F(x)=1-e^{-\lambda x}$, survival $P(X>x)=e^{-\lambda x}$, mean $1/\lambda$, variance $1/\lambda^2$.
  • Memoryless (only the exponential): $P(X>s+t\mid X>s)=P(X>t)$ — past waiting doesn't matter.
  • $Z$-table anchors: $P(Z\le1)\approx0.8413$, $P(Z\le2)\approx0.9772$, $P(Z\le-1)\approx0.1587$.
  • Poisson counts events; Exponential is the waiting time between them (same process, rate $\lambda$).
  • Linear transform of a normal stays normal: $X\sim N(\mu,\sigma^2)\Rightarrow aX+b\sim N(a\mu+b,\,a^2\sigma^2)$.
  • Tail split: outside $\pm1\sigma$ each tail $16\%$; outside $\pm2\sigma$ each tail $2.5\%$; outside $\pm3\sigma$ each tail $0.15\%$.
10 Common traps
✕ Watch out

Trap: "$f(x)$ is a probability." Reality: a PDF value is a density; probability is the area under it.

Trap: "Binomial variance is $np$." Reality: $np(1-p)$; the mean is $np$.

Trap: "Poisson mean and variance differ." Reality: both equal $\lambda$.

Trap: "$P(X=k)=F(k)$ for discrete $X$." Reality: $P(X=k)=F(k)-F(k-1)$.

Trap: "The CDF can decrease." Reality: it is non-decreasing — it only rises or stays flat.

Trap: "Discrete $P(a\le X\le b)=F(b)-F(a)$." Reality: for integers it is $F(b)-F(a-1)$, else you drop the value $a$.

Trap: "Bernoulli and Binomial are the same." Reality: Bernoulli is one trial; Binomial is $n$ independent Bernoulli trials.

Trap: "Exponential counts events." Reality: Poisson counts events; the Exponential is the waiting time between them.

Trap: "$Z=(X-\mu)/\sigma^2$." Reality: divide by $\sigma$ (the SD), not the variance.

Trap: "The 68–95–99.7 rule works for any data." Reality: only for an (approximately) normal distribution.

11 Connections with other topics
  • Random variables: the PMF/PDF is exactly what you sum/integrate for $E[X]$ and variance.
  • CLT: sums of many variables become Normal regardless of the original distribution.
  • Hypothesis testing: z/t/chi-square/F are all specific distributions of test statistics.
12 Practice problems

Q1. Binomial(20, 0.5): mean and variance?

Mean $=np=10$; variance $=np(1-p)=5$.

Q2. Poisson(λ=3): mean and variance?

Both equal $\lambda=3$.

Q3. For Normal, roughly what % lies within μ±2σ?

About 95%.

Q4. $p(x)=kx$ for $x=1,2,3,4$. Find $k$.

$k(1{+}2{+}3{+}4)=1\Rightarrow 10k=1\Rightarrow k=1/10$.

Q5. $f(x)=1/5$ on $[0,5]$. Find $P(1\le X\le 3)$.

Area $=(3-1)\times\tfrac15=\tfrac25$.

Q6. Fair die, $F$ its CDF. Find $P(3\le X\le 5)$ using the CDF.

$F(5)-F(2)=\tfrac56-\tfrac26=\tfrac12$ (integer rule uses $a-1=2$).

Quick MCQ

For a continuous random variable $X$, what is $P(X=2)$?
$f(2)$ is the density, not the probability of an exact point.
✅ Correct — for continuous $X$, any single point has probability $0$; only intervals count.
Probabilities of single points aren't 1.
$F(2)=P(X\le2)$, not $P(X=2)$.
13 Final memory box
🧠 Remember
  • PMF: bars, $\sum=1$. PDF: curve, area $=1$; height isn't probability.
  • CDF $F(x)=P(X\le x)$, non-decreasing, rises 0→1; PMF $=$ jumps: $P(X{=}k)=F(k)-F(k{-}1)$.
  • Interval: continuous $F(b)-F(a)$; integer discrete $F(b)-F(a-1)$.
  • Find unknown constant by setting the total to 1 ($\sum p=1$ or $\int f=1$).
  • Bernoulli: $p,\ pq$. Binomial: $np,\ npq$. Poisson: $\lambda,\lambda$. Normal: $\mu,\sigma^2$.
  • Exponential: mean $1/\lambda$, var $1/\lambda^2$, $P(X>x)=e^{-\lambda x}$, memoryless.
  • Normal 68–95–99.7 rule; standardize with $Z=(X-\mu)/\sigma$.
  • One trial → Bernoulli · n trials → Binomial · rare counts → Poisson · waiting → Exponential · bell → Normal.
