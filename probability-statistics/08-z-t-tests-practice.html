z-test & t-test

8 questions. Tap any option — you'll see why it's right or wrong, and the correct one lights up.

Q1. Use a t-test (not z) when…
Large samples lean toward z.
Known $\sigma$ → z-test.
✅ Correct — unknown $\sigma$ (use $s$), especially small $n$.
Categorical data uses chi-square.
Q2. $\bar x=52,\ \mu_0=50,\ \sigma=8,\ n=64$. $z=?$
SE $=8/\sqrt{64}=1$, so $z=2$.
✅ Correct — $z=(52-50)/(8/8)=2$.
Don't forget to divide by the SE.
Recompute the standard error: it's 1.
Q3. At $\alpha=0.05$ two-tailed, $z=2$ is…
$|2|\gt1.96$, so it is significant.
✅ Correct — beyond the critical $1.96$, reject $H_0$.
Perfectly possible value.
Tests never "prove" $H_0$.
Q4. The p-value is…
Common misread — it's not $P(H_0)$.
$\alpha$ is the threshold you choose, not the p-value.
✅ Correct — probability of data this extreme assuming $H_0$.
Effect size is a separate measure.
Q5. Two-tailed critical $z$ at $\alpha=0.05$ is…
$1.64$ is the one-tailed value.
✅ Correct — $1.96$ for two-tailed 5%.
$2.58$ is for $\alpha=0.01$.
Far too small.
Q6. "Fail to reject $H_0$" means…
We never prove $H_0$.
✅ Correct — the data wasn't surprising enough to reject $H_0$.
That would be rejecting $H_0$.
The test worked fine; the result is just "no evidence".
Q7. A Type I error is…
That's a Type II error ($\beta$).
It's a statistical, not arithmetic, error.
✅ Correct — a false alarm; its rate is $\alpha$.
Sample size isn't the definition.
Q8. $\sigma$ unknown, $n=12$. Which test & df?
Unknown $\sigma$ → t, not z.
It's a t-test, and df $=n-1$.
✅ Correct — t-test with $n-1=11$ degrees of freedom.
df is $n-1=11$, not $n$.
