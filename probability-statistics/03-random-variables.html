Random Variables

A random variable turns messy outcomes into numbers — so we can average them, measure their spread, and do real math with chance.

2 Why this topic exists

"Heads/tails" or "red/blue" can't be averaged. A random variable attaches a number to each outcome (heads → 1, tails → 0), so we can talk about its average (expectation) and its spread (variance) — the two numbers that summarize almost any uncertain quantity.

3 Game / visual imagination
🎮 Imagine it

A random variable $X$ is a labeling machine: each outcome walks in, $X$ stamps a number on it. Expectation $E[X]$ = the balance point of all those numbers weighted by their probability. Variance = how far the numbers spread from that balance point.

4 Basic definitions

Random variable $X$: a rule assigning a real number to each outcome.

Discrete: takes countable values (die face, # of heads). Continuous: takes any value in a range (height, time).

Expectation (mean) $E[X]$: the probability-weighted average. Variance $\operatorname{Var}(X)$: average squared distance from the mean. Standard deviation: $\sqrt{\operatorname{Var}(X)}$.

6 Main formulas
$$ E[X]=\sum_x x\,p(x)\quad\text{(discrete)}\qquad E[X]=\int x\,f(x)\,dx\quad\text{(continuous)} $$
$$ \operatorname{Var}(X)=E[(X-\mu)^2]=E[X^2]-\big(E[X]\big)^2 $$

Linearity: $E[aX+b]=aE[X]+b$, and $\operatorname{Var}(aX+b)=a^2\operatorname{Var}(X)$.

7 Step-by-step: mean & variance (discrete)
  1. List the values $x$ and their probabilities $p(x)$.
  2. $E[X]=\sum x\,p(x)$.
  3. $E[X^2]=\sum x^2\,p(x)$.
  4. $\operatorname{Var}(X)=E[X^2]-(E[X])^2$; std $=\sqrt{\operatorname{Var}}$.
8 Worked example
✦ A fair die

$E[X]=\tfrac{1}{6}(1+2+3+4+5+6)=3.5$.

$E[X^2]=\tfrac{1}{6}(1+4+9+16+25+36)=\tfrac{91}{6}\approx15.17$.

$\operatorname{Var}(X)=15.17-3.5^2=15.17-12.25=2.92$.

9 GATE shortcuts
⚡ High-value facts
  • Computing formula: $\operatorname{Var}(X)=E[X^2]-(E[X])^2$ (faster than the definition).
  • $E[aX+b]=aE[X]+b$; $\operatorname{Var}(aX+b)=a^2\operatorname{Var}(X)$ (the $+b$ shift doesn't change variance).
  • $E[X+Y]=E[X]+E[Y]$ always; $\operatorname{Var}(X+Y)=\operatorname{Var}(X)+\operatorname{Var}(Y)$ only if independent.
  • Variance is never negative; std has the same units as $X$.
10 Common traps
✕ Watch out

Trap: "$\operatorname{Var}(aX)=a\operatorname{Var}(X)$." Reality: it's $a^2\operatorname{Var}(X)$.

Trap: "$\operatorname{Var}(X+Y)=\operatorname{Var}(X)+\operatorname{Var}(Y)$ always." Reality: only when independent (else add $2\operatorname{Cov}$).

Trap: "$E[X^2]=(E[X])^2$." Reality: equal only if variance is 0; in general $E[X^2]\ge(E[X])^2$.

11 Connections with other topics
  • PMF/PDF: the distribution that feeds the expectation/variance sums.
  • Joint & covariance: $\operatorname{Var}(X+Y)$ needs covariance when dependent.
  • CLT: sums of random variables become normal; mean and variance set its shape.
  • Estimation: sample mean estimates $E[X]$; sample variance estimates $\operatorname{Var}(X)$.
12 Practice problems

Q1. $X$: $P(X=0)=0.5,\ P(X=2)=0.5$. Find $E[X]$.

$0(0.5)+2(0.5)=1$.

Q2. Same $X$. Find $\operatorname{Var}(X)$.

$E[X^2]=0+4(0.5)=2$; $\operatorname{Var}=2-1^2=1$.

Q3. If $\operatorname{Var}(X)=4$, find $\operatorname{Var}(3X+5)$.

$3^2\cdot4=36$ (the $+5$ doesn't matter).

Quick MCQ

$\operatorname{Var}(2X-3)=?$ given $\operatorname{Var}(X)=5$.
The $-3$ shift doesn't change variance; scale by $a^2$.
That's $2\cdot5$; variance scales by $a^2=4$.
✅ Correct — $\operatorname{Var}(2X-3)=2^2\cdot5=20$.
Don't subtract anything — the constant shift drops out.
13 Final memory box
🧠 Remember
  • Random variable = number attached to each outcome (discrete or continuous).
  • $E[X]=\sum x\,p(x)$; $\operatorname{Var}(X)=E[X^2]-(E[X])^2$.
  • $E[aX+b]=aE[X]+b$; $\operatorname{Var}(aX+b)=a^2\operatorname{Var}(X)$.
  • $E[X+Y]=E[X]+E[Y]$ always; variances add only if independent.
