Random Variables

A random variable turns messy outcomes into numbers — so we can average them, measure their spread, and do real math with chance.

2 Why this topic exists

"Heads/tails" or "red/blue" can't be averaged. A random variable attaches a number to each outcome (heads → 1, tails → 0), so we can talk about its average (expectation) and its spread (variance) — the two numbers that summarize almost any uncertain quantity.

3 Game / visual imagination
🎮 Imagine it

A random variable $X$ is a labeling machine: each outcome walks in, $X$ stamps a number on it. Expectation $E[X]$ = the balance point of all those numbers weighted by their probability. Variance = how far the numbers spread from that balance point.

🧪 PMF & Expectation E[X] is the balance point of the bars
4 Basic definitions
  • Random variable $X$: a rule assigning a real number to each outcome of an experiment.
  • Discrete: takes countable values (die face, number of heads).
  • Continuous: takes any value in a range (height, waiting time).
  • Support: the set of values $X$ can actually take.
  • Expectation (mean) $E[X]$: the long-run, probability-weighted average.
  • Variance $\operatorname{Var}(X)$: the average squared distance from the mean — the spread.
  • Standard deviation: $\sqrt{\operatorname{Var}(X)}$ — spread in the original units.
  • Function of $X$: $g(X)$ is itself a random variable, with its own expectation $E[g(X)]$.
📖 Study in Depth — random variables from zero
1. What is a random variable?

A random variable assigns a number to the outcome of a random experiment — a function from outcomes to numbers.

roll a die, X = the number shown  →  X ∈ {1,2,3,4,5,6}

Turning outcomes into numbers is what lets us do maths (averages, spread, distributions).

2. Why we need random variables

Outcomes are not always numbers — a coin gives H/T. Define a random variable to make them numeric:

X = 1 if Head, X = 0 if Tail
P(X=1) = P(H) = 1/2 ,  P(X=0) = P(T) = 1/2

Now expectation, variance, distributions and ML models can all operate on X.

3. Random variable from a die 
X = outcome of a fair die,  P(X=k) = 1/6 for k = 1..6

An event becomes a statement about X:

"X is even" = {2,4,6} → P = P(X=2)+P(X=4)+P(X=6) = 3/6 = 1/2
4. Number of heads in two coins

Toss two coins, let X = number of heads. Map each outcome to a value:

HH→2 , HT→1 , TH→1 , TT→0
XoutcomesP(X)
0TT1/4
1HT, TH1/2
2HH1/4

This value-with-probability listing is the start of a probability distribution.

5. Random variable vs event 
event           = a SET of outcomes        (e.g. "head" = {H})
random variable = a NUMBER assigned to each outcome (X = 1 if head, 0 if tail)

Trap: an event and a random variable are not the same — but events can be written as conditions on a random variable, like {X = 1}.

6. Discrete vs continuous random variables 
Discrete   = countable values (die: 1–6, #heads, #defects, #arrivals)
Continuous = any value in an interval (height, weight, time, temperature)

Continuous variables take infinitely many values (170.1, 170.12, 170.123, …).

Preview: discrete variables are described by a PMF, continuous ones by a PDF (next topics).

7. Probability distribution & its rules

A probability distribution lists every value of X with its probability (the die and two-coin tables above are distributions).

two valid rules for a discrete distribution:
1.  P(X = x) ≥ 0   for every value
2.  Σ P(X = x) = 1  (all probabilities add to 1)
8. Expectation — a preview

The expected value is the long-run average of a random variable:

E[X] = Σ x · P(X = x)
fair die: E[X] = (1+2+3+4+5+6)/6 = 21/6 = 3.5

The die never shows 3.5 — it's the average over many rolls. (Variance and full expectation come later.)

9. Properties of expectation (linearity) 
E[aX + b] = a·E[X] + b
E[X + Y]  = E[X] + E[Y]
E[X]=10  →  E[2X+5] = 2·10 + 5 = 25

Key GATE fact: linearity of expectation holds even when X and Y are dependent — no independence needed.

10. Variance & standard deviation

Expectation gives the average; variance measures the spread around it.

Var(X) = E[(X − μ)²] = E[X²] − (E[X])²      (the second form is easier)
SD(X)  = σ = √Var(X)

We square the deviations so positives and negatives don't cancel; the square root then returns spread to the original units (variance is in units², SD in units).

11. Variance — worked examples

Fair die:

E[X] = 3.5 ,  E[X²] = (1+4+9+16+25+36)/6 = 91/6
Var = 91/6 − (7/2)² = 91/6 − 49/4 = 35/12 ≈ 2.917 ,  SD ≈ 1.708

Two coins, X = number of heads:

E[X] = 1 ,  E[X²] = 0²(1/4)+1²(1/2)+2²(1/4) = 3/2
Var = 3/2 − 1² = 1/2 ,  SD ≈ 0.707

Remember: fair die → E[X]=3.5, Var=35/12.

12. Properties of variance 
Var(aX + b) = a²·Var(X)        (adding b shifts, doesn't spread;
SD(aX + b)  = |a|·SD(X)         multiplying by a scales spread by a²)
Var(X + Y)  = Var(X) + Var(Y)   ONLY if X, Y are independent
Var(X)=4  →  Var(3X+10) = 9·4 = 36
13. Geometric RV: failures before first success GATE extra

Repeat a trial until the first success (probability p, failure q = 1−p). Let X = number of failures before that success:

X ∈ {0,1,2,…} ,  P(X=k) = qᵏ·p      (k failures, then a success)
E[X] = q/p ,  Var(X) = q/p²
coin until first Head (p=1/2): P(X=0)=1/2, P(X=1)=1/4, …; E[X]=1
p=0.3: P(X=2)=0.7²·0.3=0.147, E[X]=0.7/0.3=7/3
14. Trials until success vs failures before (the trap) GATE extra

Two versions of the geometric variable — don't mix them up:

X = failures before success: values 0,1,2,…  P=qᵏp   E[X]=q/p
Y = trials  until  success:  values 1,2,3,…  P=qᵏ⁻¹p E[Y]=1/p
relation:  Y = X + 1   (trials = failures + the success trial)

Both have Var = q/p². Failures start at 0; trials start at 1.

15. Common traps (variance & geometric)
  • Var = E[X²] − E[X]? No — it's E[X²] − (E[X])² (square the mean).
  • SD = Var²? No — SD = √Var.
  • Var(aX+b) = a·Var(X) + b? No — it's a²·Var(X).
  • Failures-before-success starts at 1? No — at 0 (immediate success = 0 failures).
  • Expectation must be an achievable value? No — a die's E[X]=3.5.
  • E[X+Y] needs independence? No — linearity always holds; only Var(X+Y)=Var(X)+Var(Y) needs it.

17. Mental model
  • Outcome = the raw result; random variable = a score assigned to that result.
  • Coin → X = 1/0; die → X = number shown; two coins → X = number of heads.
  • Listing each value with its probability gives a probability distribution — the next big idea.

6 Main formulas

Expectation (mean) — the probability-weighted average:

$$ E[X]=\sum_x x\,p(x)\ \text{(discrete)}\qquad E[X]=\int x\,f(x)\,dx\ \text{(continuous)} $$

Expectation of a function — plug $g(x)$ into the same sum/integral:

$$ E[g(X)]=\sum_x g(x)\,p(x)\qquad E[g(X)]=\int g(x)\,f(x)\,dx $$

Variance — spread about the mean; the right-hand form is the easy one to compute:

$$ \operatorname{Var}(X)=E[(X-\mu)^2]=E[X^2]-\big(E[X]\big)^2 $$

Linear transformation — shifting and scaling $X$:

$$ E[aX+b]=aE[X]+b,\qquad \operatorname{Var}(aX+b)=a^2\operatorname{Var}(X) $$
7 Step-by-step: mean & variance (discrete)
  1. List the values $x$ and their probabilities $p(x)$.
  2. $E[X]=\sum x\,p(x)$.
  3. $E[X^2]=\sum x^2\,p(x)$.
  4. $\operatorname{Var}(X)=E[X^2]-(E[X])^2$; std $=\sqrt{\operatorname{Var}}$.
8 Worked example
✦ A fair die

$E[X]=\tfrac{1}{6}(1+2+3+4+5+6)=3.5$.

$E[X^2]=\tfrac{1}{6}(1+4+9+16+25+36)=\tfrac{91}{6}\approx15.17$.

$\operatorname{Var}(X)=15.17-3.5^2=15.17-12.25=2.92$.

9 GATE cheat sheet
⚡ High-value facts
  • Computing formula: $\operatorname{Var}(X)=E[X^2]-(E[X])^2$ (faster than the definition).
  • $E[aX+b]=aE[X]+b$; $\operatorname{Var}(aX+b)=a^2\operatorname{Var}(X)$ (the $+b$ shift doesn't change variance).
  • $E[X+Y]=E[X]+E[Y]$ always; $\operatorname{Var}(X+Y)=\operatorname{Var}(X)+\operatorname{Var}(Y)$ only if independent.
  • Variance is never negative; std has the same units as $X$.
10 Common traps
✕ Watch out

Trap: "$\operatorname{Var}(aX)=a\operatorname{Var}(X)$." Reality: it's $a^2\operatorname{Var}(X)$.

Trap: "$\operatorname{Var}(X+Y)=\operatorname{Var}(X)+\operatorname{Var}(Y)$ always." Reality: only when independent (else add $2\operatorname{Cov}$).

Trap: "$E[X^2]=(E[X])^2$." Reality: equal only if variance is 0; in general $E[X^2]\ge(E[X])^2$.

11 Connections with other topics
  • PMF/PDF: the distribution that feeds the expectation/variance sums.
  • Joint & covariance: $\operatorname{Var}(X+Y)$ needs covariance when dependent.
  • CLT: sums of random variables become normal; mean and variance set its shape.
  • Estimation: sample mean estimates $E[X]$; sample variance estimates $\operatorname{Var}(X)$.
12 Practice problems

Q1. $X$: $P(X=0)=0.5,\ P(X=2)=0.5$. Find $E[X]$.

$0(0.5)+2(0.5)=1$.

Q2. Same $X$. Find $\operatorname{Var}(X)$.

$E[X^2]=0+4(0.5)=2$; $\operatorname{Var}=2-1^2=1$.

Q3. If $\operatorname{Var}(X)=4$, find $\operatorname{Var}(3X+5)$.

$3^2\cdot4=36$ (the $+5$ doesn't matter).

Quick MCQ

$\operatorname{Var}(2X-3)=?$ given $\operatorname{Var}(X)=5$.
The $-3$ shift doesn't change variance; scale by $a^2$.
That's $2\cdot5$; variance scales by $a^2=4$.
✅ Correct — $\operatorname{Var}(2X-3)=2^2\cdot5=20$.
Don't subtract anything — the constant shift drops out.
